# Reading 6: Simulation Methods

> **핵심 질문:** 자산 가격은 왜 로그정규분포를 따르는가? 그리고 복잡한 금융상품의 가치를 어떻게 시뮬레이션으로 추정하는가?

## 왜 시뮬레이션이 필요한가?

금융 세계에서는 수식으로 정확히 풀 수 없는 문제들이 많습니다:
- 복잡한 파생상품 가치 평가
- 연금 기금의 장기 자산/부채 시뮬레이션
- 비정규 분포 포트폴리오의 리스크 측정

이런 상황에서 **반복 시뮬레이션**으로 해답을 근사합니다.

## LOS 6.a: 정규분포와 로그정규분포의 관계

### 로그정규분포 (Lognormal Distribution)

**정의:** 확률변수 X가 정규분포를 따를 때, $e^X$는 로그정규분포를 따릅니다.

$$\text{만약 } X \sim N(\mu, \sigma^2) \text{ 이면, } Y = e^X \text{ 는 로그정규분포}$$

반대로: $\ln(Y) = X$이므로, **로그정규분포 변수의 로그 값은 정규분포**를 따릅니다. 이름의 유래입니다.

---

### 정규분포 vs 로그정규분포 비교

| 특성 | 정규분포 | 로그정규분포 |
|------|----------|-------------|
| 범위 | $-\infty$ ~ $+\infty$ | **0 ~ $+\infty$** (음수 불가) |
| 형태 | 좌우 대칭 | **오른쪽으로 치우친 비대칭** |
| 평균 = 중앙값? | 예 (대칭) | 아니오 (평균 > 중앙값) |
| 음의 값 확률 | 있음 | **0** |

```
정규분포:                    로그정규분포:
    ▲                           ▲
    │   ╭──────╮                │  ╭╮
    │  ╭╯      ╰╮               │ ╭╯ ╰──╮
    │ ╭╯        ╰╮              │╭╯     ╰──────
    ├─────────────→            ├──────────────→
   -∞     0     +∞             0       +∞
```

---

### 왜 자산 가격 모델에 로그정규분포를 쓰는가?

자산 가격의 두 가지 현실적 조건:
1. **가격은 음수가 될 수 없다** — 주가, 채권 가격 등은 0 이상
2. **연속복리 수익률은 덧셈이 가능하다**

Reading 1에서 배운 연속복리 가격 공식:
$$P_T = P_0 \cdot e^{r_{0,T}}$$

여기서:
- $P_T$ = T시점 자산 가격
- $P_0$ = 현재(0시점) 자산 가격
- $r_{0,T}$ = 0에서 T까지의 연속복리 수익률

**핵심 논리:**

시간 0~T를 짧은 구간들로 나누면, $r_{0,T}$는 각 구간 수익률의 합:
$$r_{0,T} = r_1 + r_2 + \cdots + r_n$$

각 $r_i$가 정규분포를 따른다면(또는 따르지 않더라도 **중심극한정리**에 의해) $r_{0,T}$는 근사적으로 **정규분포**를 따릅니다.

따라서 $P_T = P_0 \cdot e^{r_{0,T}}$는 **로그정규분포**를 따릅니다.

→ 가격이 항상 양수이며, 실제 자산 가격의 움직임을 잘 표현합니다.

---

### 독립 동일 분포 (i.i.d.) 가정

많은 CFA 가격 모형에서 수익률이 **독립 동일 분포(independently and identically distributed, i.i.d.)** 를 따른다고 가정합니다:

- **독립(independently distributed)**: 과거 수익률이 미래 수익률 예측에 유용하지 않음 — 이전 기간의 성과가 다음 기간에 영향 없음
- **동일(identically distributed)**: 평균과 분산이 시간에 따라 변하지 않음 — **정상성(stationarity)** 이라고 불리며 시계열 분석의 핵심 개념

## LOS 6.b: 몬테카를로 시뮬레이션 (Monte Carlo Simulation)

### 몬테카를로 시뮬레이션이란?

**정의:** 하나 이상의 위험 요소(risk factor)에 대해 무작위 값을 반복 생성하여, 증권 가치의 분포를 추정하는 기법.

이름의 유래: 모나코의 카지노 도시 몬테카를로 — 무작위성(randomness)을 활용하는 방식에서 유래.

---

### 몬테카를로 시뮬레이션 절차

**주식 옵션 가치 평가 예시 (행사일이 고정된 유러피언 옵션):**

```
1단계: 위험 요소 분포 설정
   - 주가: 로그정규분포 (평균 μ, 분산 σ²)
   - 이자율: 정규분포 (평균 r̄, 분산 σ_r²)

2단계: 무작위 값 생성
   - 컴퓨터로 주가, 이자율 무작위 생성

3단계: 옵션 가치 계산
   - 각 (주가, 이자율) 쌍에 대해 옵션 가치 산출

4단계: 반복 (수백 ~ 수만 회)
   - 매번 새로운 무작위 값으로 반복

5단계: 분포 분석
   - 모든 옵션 가치의 평균 = 기대 가치
   - 분산 = 불확실성 크기
```

---

### 몬테카를로의 활용 분야

| 활용 분야 | 설명 |
|-----------|------|
| 복잡한 증권 가치 평가 | 수식으로 풀기 어려운 옵션, 구조화 상품 |
| 트레이딩 전략 수익/손실 시뮬레이션 | 전략의 과거 성과를 미래 시뮬레이션 |
| **VaR(Value at Risk) 계산** | 포트폴리오의 최대 손실 추정 |
| 연금 자산/부채 시뮬레이션 | 장기 재정 건전성 분석 |
| 비정규 분포 포트폴리오 평가 | 정규분포 가정이 불가능한 경우 |

---

### 몬테카를로의 장점과 한계

**장점:**
- 입력 데이터가 **역사적 데이터 범위에 제한되지 않음** — "만약 금리가 갑자기 10% 급등한다면?" 같은 가상 시나리오 테스트 가능
- 복잡한 경로 의존 파생상품에도 적용 가능

**한계:**
- **복잡성**: 구현이 상당히 복잡함
- **Garbage In, Garbage Out**: 분포 가정이 잘못되면 결과도 잘못됨 — 결과의 품질은 가정의 품질을 넘지 못함
- **비분석적(non-analytic)**: 통계적 근사이므로, 수식으로 정확히 계산하는 분석적 방법이 제공하는 **직관적 이해**를 얻을 수 없음

## LOS 6.c: 부트스트랩 리샘플링 (Bootstrap Resampling)

### 리샘플링이란?

**문제:** 우리는 모집단 전체 데이터를 가질 수 없고, 표본(sample)만 가지고 있습니다.

예: 어떤 투자의 역사적 수익률은 모집단(가능한 모든 수익률)에서 추출된 표본입니다.

**리샘플링의 목적:** 보유한 표본에서 반복적으로 서브샘플(subsample)을 뽑아 모집단의 특성(평균, 분산 등)을 추론합니다.

---

### 부트스트랩 리샘플링 절차

**핵심 개념: 복원 추출(sampling with replacement)**

```
원본 표본: [2%, 5%, -1%, 8%, 3%, -2%, 7%]  (n=7)

1회 부트스트랩 샘플: [5%, 2%, 8%, 5%, -1%, 7%, 3%]  (복원, n=7)
2회 부트스트랩 샘플: [-2%, 8%, 3%, 2%, 8%, -2%, 5%]  (복원, n=7)
3회 부트스트랩 샘플: [7%, -1%, 2%, 3%, 5%, 8%, 2%]   (복원, n=7)
... (수백~수천 회 반복)
```

각 서브샘플의 평균을 계산하고, **이 샘플 평균들의 표준편차**가 표본 평균의 **표준오차(standard error)** 추정치입니다.

---

### 몬테카를로 vs 부트스트랩 비교

| 구분 | 몬테카를로 | 부트스트랩 |
|------|-----------|-----------|
| 데이터 소스 | 가정된 확률분포에서 무작위 생성 | 실제 관측 데이터에서 복원 추출 |
| 입력 범위 | 역사적 데이터 범위 넘어설 수 있음 | **역사적 데이터 범위에 제한됨** |
| 장점 | 미래 가상 시나리오 테스트 가능 | 실제 데이터 통계 특성 잘 반영 |
| 한계 | 분포 가정 의존 | 과거에 없던 극단적 사건 포착 불가 |

**공통점:** 두 방법 모두 가정의 품질만큼만 좋은 결과를 냅니다.

---

## 핵심 개념 정리 (Key Concepts)

### LOS 6.a 요약

- $X \sim N(\mu, \sigma^2)$이면 $e^X$는 로그정규분포
- 로그정규분포: 값이 항상 양수, 오른쪽 꼬리가 두꺼운 비대칭
- 자산 가격 모델: $P_T = P_0 e^{r_{0,T}}$ — 연속복리 수익률이 정규분포 → 가격은 로그정규분포
- **독립분포**: 과거 수익률이 미래 예측에 무용
- **동일분포(정상성)**: 평균과 분산이 시간 불변

### LOS 6.b 요약

- 몬테카를로: 위험 요소의 무작위 값 반복 생성 → 증권 가치 분포 추정
- 장점: 역사적 범위 초과 시나리오 테스트 ("what if" 분석)
- 한계: 복잡, 결과 품질 = 가정의 품질, 비분석적 (직관 부족)

### LOS 6.c 요약

- 부트스트랩: 실제 표본에서 복원 추출로 반복 서브샘플링
- 표본 평균들의 표준편차 = 표준오차 추정
- 장점: 모집단 통계적 특성을 잘 표현
- 한계: 입력이 실제 역사적 데이터 범위에 제한

## Module Quiz 6.1

**Q1.** 로그정규분포에서:

A. 평균과 중앙값이 같다  B. 음의 결과 확률이 0이다  C. 양의 결과 확률이 50%이다

**풀이:**
- A. 로그정규분포는 비대칭 — 평균 ≠ 중앙값 (오답)
- B. **정답** — 로그정규분포는 값이 항상 0 이상 (음수 불가능)
- C. 로그정규분포에서 양의 값 확률은 100% (0이 될 확률도 이론적으로 0)

**정답: B**

**Q2.** 다음 중 몬테카를로 분석의 한계가 **가장 아닌** 것은?

A. 통계적 방법이며 분석적 방법이 아니다
B. 분석 결과가 가정의 품질을 넘지 못한다
C. "만약" 질문에 답할 수 없다

**풀이:**
- A. 맞는 한계 (비분석적)
- B. 맞는 한계 (Garbage In, Garbage Out)
- C. **오답** — "만약" 질문(역사적 범위 밖 시나리오)에 답할 수 있는 것은 몬테카를로의 **장점**

**정답: C** (한계가 가장 아닌 것 = 장점인 것)

**Q3.** 다음 중 부트스트랩의 강점에 해당하는 것은?

A. 모집단의 통계적 특성을 잘 표현한다
B. 통계적 추정치만 제공하고 정확한 결과를 제공하지 않는다
C. 입력이 실제 결과의 분포에 의해 제한될 수 있다

- A. **정답** — 부트스트랩은 실제 데이터 기반이므로 모집단의 통계적 특성을 잘 반영
