# Reading 18: Capital Flows and the FX Market

## LOS 18.a: 외환 시장, 명목·실질 환율, 환율 변화 계산

### 왜 이걸 알아야 하는가?

해외 주식에 투자해서 10% 수익을 냈는데, 그 나라 통화가 15% 떨어졌다면? 원화 기준으로 **오히려 손해**다. 글로벌 투자에서 환율을 모르면 수익의 절반을 운에 맡기는 것이다.

### 환율 표기법 — 이것만 확실히 하면 실수의 80%가 사라진다

**USD/EUR = 1.25**

"/" 를 **"당(per)"**으로 읽는다: "1.25달러 per 유로" = **유로 1개 살 때 1.25달러 필요**

- **기준통화(분모):** EUR → 1단위 기준
- **가격통화(분자):** USD → 가격을 표시

USD/EUR이 1.25 → 1.30으로 오르면? 유로가 더 비싸졌다 = **유로 절상, 달러 절하**.

### 직접 해보자 — 절상·절하율 계산

USD/EUR이 1.42 → 1.39로 변했다.

유로 절하율 (기준통화): $(1.39 / 1.42) - 1 = -2.11\%$

달러 절상율: 먼저 EUR/USD로 변환 → $1/1.42 = 0.7042$ → $1/1.39 = 0.7194$

$(0.7194 / 0.7042) - 1 = +2.16\%$

**함정:** 2.11% ≠ 2.16%. 왜? 퍼센트 변화는 기준값에 의존하므로 분모가 다르면 답도 다르다.

### 실질 환율 — 왜 필요한가?

명목 환율이 안 변해도, 한 나라의 물가가 더 빠르게 오르면 그 나라 통화의 **실질 구매력**은 떨어진다.

$$\text{실질 환율}_{P/B} = \text{명목 환율}_{P/B} \times \frac{CPI_{\text{기준통화국}}}{CPI_{\text{가격통화국}}}$$

**숫자를 넣어보자:**

| | 기준 시점 | 3년 후 |
|--|---------|-------|
| 명목 환율 (USD/GBP) | 1.70 | 1.60 |
| 미국 CPI | 100 | 110 |
| 영국 CPI | 100 | 112 |

$$\text{실질 환율} = 1.60 \times \frac{112}{110} = 1.629$$

명목: 1.70 → 1.60 (달러 절상 5.9%). 영국 물가가 더 올라서(112 vs 110) 실질: 1.70 → 1.629 (실질 달러 절상은 4.2%로 축소).

직관: 명목상 달러가 많이 강해졌지만, 영국 물가가 더 올라서 영국 물건의 실질 가격은 생각보다 안 떨어진 것.

### 이자율 평가 (IRP) — 선물환율은 왜 현물과 다른가?

A나라 금리 5%, B나라 금리 3%. 만약 선물환율이 금리 차를 반영 안 하면, 누군가 저금리로 빌려서 고금리에 투자하고 선물환으로 환위험 제거 → 무위험 이익. 이건 불가능하므로:

$$\frac{\text{Forward}_{P/B}}{\text{Spot}_{P/B}} = \frac{1 + r_{\text{가격통화}}}{1 + r_{\text{기준통화}}}$$

**핵심 원칙: 고금리 통화는 선물에서 절하된다.** 고금리 투자의 이점을 선물환이 상쇄해야 차익거래가 불가능하다.

### 외환 시장 참가자

| 구분 | 참가자 | 역할 |
|-----|-------|------|
| **매도측** | 대형 다국적 은행 | 딜러 (시장 조성) |
| **매수측** | 기업, 펀드, 정부, 소매 | 다양한 목적으로 매수 |

일일 거래량 7조 달러 중 대부분은 투자·투기·헤징 목적. **단기 환율은 무역보다 자본 흐름이 결정.**

## LOS 18.b: 환율 제도와 무역·자본 흐름

### 불가능한 삼위일체 — 왜 환율 제도가 이렇게 다양한가?

세 가지를 **동시에** 가질 수 없다:
1. 자유로운 자본 이동
2. 고정 환율
3. 독립적 통화정책

나라마다 셋 중 하나를 포기하는 선택이 다르기 때문에 환율 제도가 다양하다.

| 선택 | 포기하는 것 | 예시 |
|-----|----------|------|
| 고정 환율 + 자본 이동 자유 | 독립적 통화정책 | 홍콩 (통화위원회) |
| 변동 환율 + 자본 이동 자유 | 환율 안정성 | 미국, 일본 |
| 고정 환율 + 독립적 통화정책 | 자본 이동 자유 | 중국 (자본 규제) |

### 환율 제도 스펙트럼 (통화정책 자율성: 낮음 → 높음)

달러화(자국통화 없음) → 통화연합 → 통화위원회 → 고정페그 → 밴드페그 → 크롤링페그/밴드 → 관리변동 → 독립변동

### 무역수지 ↔ 자본 계정 항등식

왜 미국은 매년 무역 적자인데 망하지 않는가?

GDP의 두 표현을 같다고 놓으면:

$$Y = C + I + G + (X - M) = C + S + T$$

정리하면:

$$(X - M) \equiv (S - I) + (T - G)$$

이것은 항등식(≡)이다. 예외 없이 **항상** 성립한다.

숫자를 넣어보자:
- 미국: 무역 적자 (X-M = -5,000억 달러)
- 그러면 우변도 -5,000억 → (S-I) + (T-G) = -5,000억
- 저축 < 투자, 또는 재정 적자 → 외국에서 돈을 빌려와야 한다
- → 자본 계정 흑자 5,000억 (중국이 미국 국채를 산다)

**무역 적자 = 외국 자본 유입. 무역 흑자 = 해외에 자본 투출.** 반드시 이렇게 맞아떨어진다.

## LOS 18.c: 자본 규제의 목적

| 목적 | 내용 |
|-----|------|
| 자산 가격 변동성 감소 | 위기 시 자본 유출 → 자산 급락 방지 |
| 고정 환율 유지 | 자본 흐름 제한 → 환율 목표 달성 용이 |
| 국내 금리 낮게 유지 | 투자자의 해외 고금리 추구 차단 |
| 전략 산업 보호 | 통신·방산 외국인 투자 금지 |

트레이드오프: 단기 안정 vs 장기 성장 잠재력 감소 (국제 자본시장에서 배제).

## Key Concepts

**LOS 18.a:**
- 환율 P/B: "/"를 "per"로. 기준통화 절상 → 환율↑
- 실질 환율 = 명목 × (기준통화국 CPI / 가격통화국 CPI)
- IRP: Forward/Spot = (1+r_가격)/(1+r_기준). 고금리 통화 = 선물에서 절하
- 절하율 ≠ 절상율 (기준값 다름)

**LOS 18.b:**
- 불가능한 삼위일체: 자유 자본이동 + 고정환율 + 독립 통화정책 동시 불가
- (X-M) ≡ (S-I) + (T-G) — 항등식
- 단기: 자본 흐름이 환율 결정. 장기: 무역 흐름

**LOS 18.c:**
- 자본 규제 4가지 목적. 단기 안정 vs 장기 성장의 트레이드오프

## Quiz

**Q1.** 실질 USD/EUR 환율이 3% 상승했다면, 이것이 의미하는 바는?

A. 명목 환율이 특정 수준 미만
B. 유로의 미국 재화 대비 구매력이 약 3% 증가
C. 유로존 인플레이션이 미국보다 약 3% 높음

> **풀이:** 실질 USD/EUR ↑ = EUR(기준통화) 실질 가치 상승 = 유로의 구매력 증가. 명목 환율이나 인플레이션만으로 특정할 수 없다 (여러 조합 가능).
>
> **정답: B**

---

**Q2.** 외환 시장의 매도측 참가자는?

A. 은행  B. 헤지펀드  C. 보험사

> **풀이:** 대형 다국적 은행 = 딜러 = 매도측. 헤지펀드·보험사 = 매수측.
>
> **정답: A**

---

**Q3.** 무역 적자를 감소시킬 가능성이 높은 것은?

A. 자본 계정 흑자 증가  B. 소득 대비 지출 감소 (저축 증가)  C. 국내 투자 대비 저축 감소

> **풀이:** (X-M) = (S-I) + (T-G). 저축↑ → (S-I)↑ → 무역적자↓. A는 결과이지 원인이 아님. C는 악화.
>
> **정답: B**
