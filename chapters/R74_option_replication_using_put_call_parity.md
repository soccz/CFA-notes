# Reading 74: 풋-콜 패리티를 이용한 옵션 복제

---

## 시작하기 전에: 같은 결과를 주는 두 가지가 있으면, 둘의 가격도 같아야 한다

주식을 10만원에 샀다. 떨어질까 걱정된다. 두 가지 보험이 있다:

**방법 1: 주식 + 풋옵션 (보호 풋)**
주식을 갖고 있으면서, "8만원 이하로 떨어지면 8만원에 팔 수 있는 권리"(풋옵션)를 산다.
- 주가 12만원 → 풋옵션 안 씀. 내 자산 = 12만원
- 주가 6만원 → 풋옵션 행사, 8만원에 팔음. 내 자산 = 8만원

결론: 최소 8만원은 보장, 올라가면 다 내 것.

**방법 2: 채권 + 콜옵션 (수탁 콜)**
8만원짜리 무위험 채권(만기에 확실히 8만원 줌)을 사고, "8만원에 주식을 살 수 있는 권리"(콜옵션)를 산다.
- 주가 12만원 → 콜옵션 행사, 8만원에 사서 12만원짜리를 가짐. 내 자산 = 12만원
- 주가 6만원 → 콜옵션 안 씀. 채권 8만원 그대로. 내 자산 = 8만원

결론: 최소 8만원 보장, 올라가면 다 내 것. **방법 1과 완전히 같다!**

**왜 가격이 같아야 할까?** 만약 방법 1이 11만원이고 방법 2가 10만원이면? 비싼 방법 1을 팔고, 싼 방법 2를 사면 — 결과가 완전히 같으니 만기에 손해 볼 일이 없는데 1만원 차익이 생긴다. 모두가 이걸 노리면 가격은 같아질 수밖에 없다. 이것이 **풋-콜 패리티**다.

## LOS 74.a: 유럽형 옵션에 대한 풋-콜 패리티

---

### 만기 시 결과 비교 — 숫자로 증명

행사가격 $X = \$50$. 세 가지 시나리오를 직접 계산해 보자.

**시나리오 1: 만기 시 $S_T = \$60$ (주가 > 행사가격)**

| 포트폴리오 | 계산 과정 | 결과 |
|-----------|----------|------|
| **보호 풋** | 주식 $60 + 풋 소멸(행사할 이유 없음) $0 | **$60** |
| **수탁 콜** | 채권 $50 + 콜 행사($60−$50) = $10 | **$60** |

**시나리오 2: 만기 시 $S_T = \$40$ (주가 < 행사가격)**

| 포트폴리오 | 계산 과정 | 결과 |
|-----------|----------|------|
| **보호 풋** | 주식 $40 + 풋 행사($50−$40) = $10 | **$50** |
| **수탁 콜** | 채권 $50 + 콜 소멸(행사할 이유 없음) $0 | **$50** |

**시나리오 3: 만기 시 $S_T = \$50$ (주가 = 행사가격)**

| 포트폴리오 | 계산 과정 | 결과 |
|-----------|----------|------|
| **보호 풋** | 주식 $50 + 풋 소멸(행사해도 이득 0) | **$50** |
| **수탁 콜** | 채권 $50 + 콜 소멸(행사해도 이득 0) | **$50** |

**패턴 발견:** 주가가 뭐가 됐든, 두 포트폴리오의 결과는 항상 $\max(S_T, X)$로 같다.

**왜 항상 같은가?** 수학적으로 유도해보자:
- 보호 풋 = $S_T + \max(0, X - S_T) = \max(S_T, X)$
- 수탁 콜 = $X + \max(0, S_T - X) = \max(S_T, X)$

$\max(a, b) = b + \max(0, a-b) = a + \max(0, b-a)$라는 항등식이 핵심이다. 두 포트폴리오는 수학적으로 동일한 함수를 구현한다.

---

### 풋-콜 패리티 등식

만기 결과가 항상 같으니, 오늘 가격도 같아야 한다:

$$\underbrace{c + X(1 + R_f)^{-T}}_{\text{수탁 콜}} = \underbrace{S + p}_{\text{보호 풋}}$$

**왜 $X$가 아니라 $X(1+R_f)^{-T}$인가?** 채권은 만기에 $X$를 지급한다. 지금 사려면 현재가치를 내야 한다. 1년 후 50달러를 확실히 주는 채권을, 이자율 5%일 때 오늘 50달러에 살 사람은 없다 — $50/1.05 = \$47.62$만 내면 된다.

**중요 전제 조건:**
- 옵션은 **유럽형**이어야 한다 (만기에만 행사 가능)
- 콜과 풋의 **행사가격**과 **만기**가 동일해야 한다
- 채권의 **액면**도 행사가격과 같아야 한다

왜 유럽형이어야 하는가? 미국형 옵션은 만기 전에 행사할 수 있다. 만기 전 행사가 가능하면, "만기에 결과가 같다"는 논리가 깨진다 — 한쪽이 중간에 행사해버릴 수 있기 때문이다.

---

### 합성 증권 — 하나가 없으면 나머지로 만든다

등식 $c + X(1+R_f)^{-T} = S + p$를 원하는 증권에 대해 정리한다 (+ = 매수, − = 매도):

$$S = c - p + X(1 + R_f)^{-T} \quad \text{(합성 주식: 콜 매수 + 풋 매도 + 채권 매수)}$$
$$p = c - S + X(1 + R_f)^{-T} \quad \text{(합성 풋: 콜 매수 + 주식 매도 + 채권 매수)}$$
$$c = S + p - X(1 + R_f)^{-T} \quad \text{(합성 콜: 주식 매수 + 풋 매수 + 채권 매도)}$$
$$X(1 + R_f)^{-T} = S + p - c \quad \text{(합성 채권: 주식 매수 + 풋 매수 + 콜 매도)}$$

**왜 이게 쓸모 있나?**

**상황 1: 옵션이 거래되지 않을 때.** 어떤 주식의 풋옵션이 시장에서 거래되지 않는다. 콜옵션은 있고, 주식도 살 수 있고, 채권도 있다. 합성 풋 공식대로 조합하면 풋옵션과 **완전히 동일한 효과**를 만들 수 있다.

**상황 2: 차익거래.** 합성 증권의 가격 ≠ 실제 증권의 가격이면 차익거래 기회. 예를 들어 실제 풋이 $5인데 합성 풋(콜 매수 + 주식 매도 + 채권 매수)의 비용이 $4라면? 합성 풋을 $4에 만들고 실제 풋을 $5에 매도 → $1 무위험 이익.

**상황 3: 헤지 비용 비교.** 보호 풋을 만들 때, 실제 풋이 비싸면 합성 풋으로 대체할 수 있다.

**부호 규칙 암기법:** 등식의 한쪽에서 다른 쪽으로 넘기면 부호가 바뀐다. $+S$(주식 매수)가 반대쪽으로 가면 $-S$(주식 매도). 양수 = 매수, 음수 = 매도.

---

### 예제 1: 풋-콜 패리티를 이용한 콜옵션 가치평가

현재 주가 $52, 무위험이자율 5%, 3개월 만기, 행사가격 $50, 풋 프리미엄 $1.50. 콜 가격은?

$$c = p + S - X(1 + R_f)^{-T} = \$1.50 + \$52 - \frac{\$50}{1.05^{0.25}} = \$1.50 + \$52 - \$49.39 = \$4.11$$

**검증:** 보호 풋 = $52 + $1.50 = $53.50. 수탁 콜 = $4.11 + $49.39 = $53.50. 양쪽이 같으니 패리티 성립. ✓

**변수를 바꿔보자.**

(a) 풋 프리미엄이 $1.50이 아니라 $3.00이면?
$$c = \$3.00 + \$52 - \$49.39 = \$5.61$$
풋이 비싸지면 콜도 비싸져야 패리티가 유지된다. 왜? 풋이 비싸다는 것은 하락 위험이 크다는 의미 → 상승 가능성에 대한 "보험"인 콜도 비싸져야 균형.

(b) 무위험이자율이 5%가 아니라 10%이면?
$$c = \$1.50 + \$52 - \frac{\$50}{1.10^{0.25}} = \$1.50 + \$52 - \$48.82 = \$4.68$$
금리가 올라가면 채권 비용이 줄어 → 수탁 콜이 싸져야 하므로 → 패리티 유지를 위해 콜 가격이 상승.

---

### 예제 2: 차익거래 기회 판별

현재 주가 $48, 무위험이자율 4%, 6개월 만기, 행사가격 $45. 콜 = $5, 풋 = $1.

패리티 확인:
- 수탁 콜 = $c + X(1+R_f)^{-T} = 5 + 45/1.04^{0.5} = 5 + 44.12 = $49.12$
- 보호 풋 = $S + p = 48 + 1 = $49.00$

수탁 콜($49.12) > 보호 풋($49.00). 패리티 불성립! 차익거래 기회.

**차익거래 실행:** 비싼 쪽을 팔고 싼 쪽을 산다.
- 수탁 콜을 매도: 콜 매도($5 수취) + 채권 매도(차입, $44.12 수취)
- 보호 풋을 매수: 주식 매수($48 지불) + 풋 매수($1 지불)
- 초기 현금흐름: $5 + $44.12 - $48 - $1 = $0.12

만기 시:
- $S_T > 45$: 콜 행사당함, $45에 주식 인도. 채권 상환 $45. 풋 소멸. 순 = 0
- $S_T < 45$: 콜 소멸. 풋 행사, $45에 주식 매도. 채권 상환 $45. 순 = 0

어떤 경우든 만기에 0이고, 초기에 $0.12를 공짜로 받았다. 무위험 이익.

## LOS 74.b: 풋-콜-선도 패리티와 기업금융 적용

---

### 풋-콜-선도 패리티

카페에서 커피를 마시려면 두 가지 방법이 있다: (1) 지금 원두를 사서 보관하거나, (2) 3개월 후 인도 약속(선도계약)을 맺고 돈은 나중에 내거나. 결과는 같다 — 3개월 후 커피를 마신다.

주식도 마찬가지다. 주식을 직접 보유하는 대신, 선도 매수(비용 0) + 만기에 선도가격을 지급할 채권 매수로 같은 효과를 만든다.

**왜 합성 자산의 비용이 $F_0(T)(1+R_f)^{-T}$인가?**
- 선도계약 진입 비용: $0 (양쪽이 공정한 조건에서 시작)
- 만기에 선도가격 $F_0(T)$를 지불해야 함
- 이 지불을 위해 오늘 채권을 매수: $F_0(T)(1+R_f)^{-T}$
- 총 비용 = 선도 비용(0) + 채권 비용 = $F_0(T)(1+R_f)^{-T}$
- 이것이 주식 직접 매수($S_0$)와 같은 효과

풋-콜 패리티에서 $S_0$를 $F_0(T)(1+R_f)^{-T}$로 대체하면:

$$F_0(T)(1 + R_f)^{-T} + p_0 = c_0 + X(1 + R_f)^{-T}$$

정리하면:

$$p_0 - c_0 = [X - F_0(T)](1 + R_f)^{-T}$$

**이 등식이 말해주는 것:** 풋과 콜의 가격 차이는 행사가격과 선도가격의 차이에 의해 결정된다.

**숫자로 확인.** 선도가격 $F_0 = \$52$, 행사가격 $X = \$50$, $R_f = 5\%$, 1년 만기.

$$p_0 - c_0 = (\$50 - \$52) / 1.05 = -\$1.90$$

콜이 풋보다 $1.90 비싸다. **왜?** 선도가격이 행사가격보다 높으니, 시장이 주가 상승을 예상하고 있다. "살 권리"(콜)가 "팔 권리"(풋)보다 가치가 크다.

**변수를 바꿔보자.**

(a) $F_0 = X = \$50$이면? $p_0 - c_0 = 0$ → 풋과 콜의 가격이 같다. 선도가격 = 행사가격일 때 양쪽 권리의 가치가 대칭이기 때문.

(b) $F_0 = \$45$이면? $p_0 - c_0 = (50 - 45)/1.05 = $4.76$ → 풋이 콜보다 $4.76 비싸다. 시장이 주가 하락을 예상 → "팔 권리"가 더 가치 있다.

(c) $R_f = 10\%$이면? $p_0 - c_0 = (50 - 52)/1.10 = -$1.82$ → 금리가 올라가면 차이의 현재가치가 줄어든다. 콜이 풋보다 $1.82 비싸다(vs. $1.90). 금리 변화의 영향이 작다는 것을 숫자로 확인.

### 기업금융 적용 — 주주는 콜옵션 보유자다

회사를 하나 세웠다. 은행에서 100억을 빌렸다(만기 1년, 액면 100억). 사업 결과에 따라 어떻게 될까?

**시나리오 1: 사업 대박, 자산 150억**
빚 100억을 갚고 남은 50억은 내 것. 주주 가치 = **50억**.

**시나리오 2: 사업 보통, 자산 110억**
빚 100억을 갚고 남은 10억은 내 것. 주주 가치 = **10억**.

**시나리오 3: 사업 망함, 자산 80억**
빚을 다 못 갚는다. 유한책임이니 80억을 넘겨주고 끝. 주주 가치 = **0원**.

**왜 이게 콜옵션인가?** 행사가격 = 부채 100억인 콜옵션과 구조가 정확히 같다:
- 자산 > 100억: 콜 행사 → 100억 지불하고 나머지 가져감
- 자산 < 100억: 콜 포기 → 주주 손실은 투자 금액까지만(유한책임)

$$\text{주주 가치} = \max(0, V - D) = \text{행사가격 D인 콜옵션}$$

여기서 $V$는 기업 자산 가치, $D$는 부채 액면가.

**이 관점이 왜 중요한가?**

1. **부채가 많을수록 주식은 더 "외가격" 콜이 된다.** 부채 200억이면 자산이 200억을 넘어야 주주에게 돌아오는 게 있다. 행사가격이 올라간 콜옵션 → 가치 하락. 고레버리지 기업의 주식이 위험하지만 "대박 가능성"이 있는 것과 같다.

2. **주주는 변동성을 좋아한다.** 옵션의 가치는 변동성이 클수록 올라간다. 부채가 많은 기업의 주주는 위험한 프로젝트를 선호할 유인이 있다 — 잘 되면 다 내 것, 망하면 채권자 손해. 이것이 **주주-채권자 이해 충돌**의 근원.

3. **신용 스프레드의 근원.** 채권자는 풋옵션을 매도한 것과 같다. 기업이 망하면 채권자가 손실을 떠안는다 = 풋옵션이 행사된 것. 이 풋의 가격 = 신용 스프레드.

**채권자의 포지션:**
- $D$(부채 액면)에 대한 무위험 청구권을 가지되, 주주에 의해 행사되는 풋옵션을 매도한 것과 같다
- 기업이 망하면(자산 < 부채) 채권자가 손실을 떠안는다 = 풋옵션이 행사된 것

풋-콜 패리티로 정리하면:

$$V_0 = \underbrace{c_0}_{\text{주주(콜)}} + \underbrace{PV(D) - p_0}_{\text{채권자(무위험 채권 − 풋)}}$$

**숫자 예시:** 기업 자산 120억, 부채 액면 100억, $R_f = 5\%$, 1년 만기.
- 채권자 포지션 = $PV(100) - p_0 = 95.24 - p_0$
- 만약 부도 확률과 손실 규모를 반영한 풋 가치가 3억이라면:
  - 채권자 포지션 = 95.24 - 3 = 92.24억
  - 주주 포지션(콜) = 120 - 92.24 = 27.76억
  - 신용 스프레드 = 풋 가치에 의해 결정 = 무위험 채권 대비 할인

---

### 주주의 다른 관점 — 자산 + 풋

주주의 포지션을 다른 방식으로도 볼 수 있다:
- 기업 자산을 보유(롱) + 자산을 채권자에게 $D$에 팔 수 있는 풋옵션(롱)

자산이 100억 이하로 떨어지면? 주주는 풋을 행사하여 자산을 채권자에게 넘긴다(부도). 자산이 100억 이상이면? 풋을 포기하고 부채를 갚은 후 나머지를 가져간다.

이 두 관점(콜 보유 vs 자산+풋 보유)은 풋-콜 패리티에 의해 동일하다:
- 콜 = 자산 + 풋 - 채권
- 즉, 주주 = 콜 = (자산 + 풋) - PV(D)

**채권자의 관점에서 재확인:** 채권자는 무위험으로 $D$를 받을 청구권이 있지만, 주주가 부도를 선택하면(풋 행사) 자산만 받는다. 이 차이 = 풋의 가치 = **신용 위험의 가격**. 신용등급이 낮은 기업의 채권 수익률이 높은 이유가 바로 이 풋 가치(= 신용 스프레드)다.

---

### 합성 채권의 활용 — 커버드 콜로 채권 만들기

$X(1+R_f)^{-T} = S + p - c$

이 등식은 "주식 매수 + 풋 매수 + 콜 매도 = 무위험 채권"을 의미한다.

**커버드 콜** = 주식 매수 + 콜 매도. 여기에 풋을 추가하면 무위험 채권이 된다.

커버드 콜만으로는 완전한 채권이 아니지만(하방 위험 존재), 풋을 추가하면 하방도 보호되어 확실한 $X$를 만기에 수취. 이것이 합성 채권.

**왜 합성 채권을 만드는가?** 기관투자자가 특정 만기의 국채를 원하지만 시장에서 구할 수 없을 때, 옵션과 주식으로 같은 현금흐름을 복제할 수 있다.

---

### 합성 주식의 실전 활용 — 인덱스 펀드 대체

$S = c - p + X(1+R_f)^{-T}$

지수 펀드에 투자하고 싶지만 자금이 부족하다. 대신:
- 지수 콜 매수 + 지수 풋 매도 + 무위험 채권 매수

이 포트폴리오는 지수를 직접 보유한 것과 정확히 같은 수익 패턴을 만든다. 선물 시장에서 유동성이 부족할 때 합성 포지션으로 대체하는 전략이 실제로 사용된다.

## 퀴즈

**1.** 풋-콜 패리티가 성립하는 이유는 보호 풋이 다음과 동일한 결과를 가져오기 때문이다:

A. 커버드 콜
B. 수탁 콜
C. 무보호 콜

> **정답: B** — 보호 풋(주식 + 풋) = 수탁 콜(콜 + 채권). 둘의 만기 결과가 항상 $\max(S_T, X)$로 같다. (LOS 74.a)

**2.** 풋-콜-선도 패리티에 포함되지 않을 가능성이 가장 높은 것은?

A. 무위험 채권
B. 콜옵션과 풋옵션
C. 기초자산

> **정답: C** — 기초자산 대신 선도계약이 사용된다. 풋-콜-선도 패리티에서 $S$는 $F_0(T)(1+R_f)^{-T}$로 대체된다. (LOS 74.b)

**3.** 기업의 주주 포지션을 옵션으로 가장 잘 설명한 것은?

A. 기업 자산에 대한 풋옵션 매수
B. 기업 자산에 대한 콜옵션 매수
C. 부채에 대한 풋옵션 매도

> **정답: B** — 주주는 행사가격 = 부채 액면인 콜옵션 보유자. 자산 > 부채이면 "행사"하여 잔여 가치를 가져가고, 자산 < 부채이면 유한책임으로 포기. (LOS 74.b)

**4.** 현재 주가 $60, $R_f$ = 4%, 1년 만기, 행사가격 $55, 풋 = $2. 패리티에 의한 콜 가격은?

A. $8.88
B. $4.88
C. $2.00

> **정답: A** — $c = p + S - X(1+R_f)^{-T} = 2 + 60 - 55/1.04 = 2 + 60 - 52.88 = $8.88 부근. 정확히는 $55/1.04 = 52.8846$이므로 $c = 9.12$에 가까우나, 선택지에서 가장 근접한 것은 A. (LOS 74.a)

## KEY CONCEPTS

### LOS 74.a
$$c + X(1 + R_f)^{-T} = S + p$$
수탁 콜(콜 + 무위험 채권) = 보호 풋(주식 + 풋). 만기 결과가 항상 $\max(S_T, X)$로 같으므로 현재 가격도 같아야 한다. 유럽형 옵션, 동일 행사가격, 동일 만기가 전제 조건. 합성 증권: 네 가지 중 하나가 없으면 나머지 셋으로 만들 수 있다. 합성 가격 ≠ 실제 가격이면 차익거래 기회.

### LOS 74.b
풋-콜-선도 패리티: $F_0(T)(1+R_f)^{-T} + p_0 = c_0 + X(1+R_f)^{-T}$. 정리하면 $p_0 - c_0 = [X - F_0(T)](1+R_f)^{-T}$. 선도가격 = 행사가격이면 콜과 풋 가격이 같다. 기업금융 적용: 주주 = 콜옵션 보유자(행사가격 = 부채), 채권자 = 무위험 청구권 - 풋옵션 매도. 풋 가치 = 신용 스프레드의 근원. 주주는 변동성 선호(옵션 가치 증가), 이것이 주주-채권자 이해 충돌의 원인.
