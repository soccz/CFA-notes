# Reading 74: 풋-콜 패리티를 이용한 옵션 복제

---

## 시작하기 전에: 같은 결과를 주는 두 가지가 있으면, 둘의 가격도 같아야 한다

주식을 10만원에 샀다. 주가가 떨어질까 걱정된다. 두 가지 보험 방법이 있다:

**방법 1: 주식 + 풋옵션 (보호 풋)**
주식을 갖고 있으면서, "8만원 이하로 떨어지면 8만원에 팔 수 있는 권리"(풋옵션)를 산다.
- 주가 12만원 → 풋옵션 안 씀. 내 자산 = 12만원
- 주가 6만원 → 풋옵션 행사, 8만원에 팔음. 내 자산 = 8만원

결론: 최소 8만원은 보장, 올라가면 다 내 것.

**방법 2: 채권 + 콜옵션 (수탁 콜)**
8만원짜리 무위험 채권(만기에 확실히 8만원 줌)을 사고, "8만원에 주식을 살 수 있는 권리"(콜옵션)를 산다.
- 주가 12만원 → 콜옵션 행사, 8만원에 사서 12만원짜리를 가짐. 내 자산 = 12만원
- 주가 6만원 → 콜옵션 안 씀. 채권 8만원 그대로. 내 자산 = 8만원

결론: 최소 8만원 보장, 올라가면 다 내 것. **방법 1과 완전히 같다!**

같은 결과를 주는 두 가지가 있으면, 둘의 가격도 같아야 한다 (아니면 차익거래). 이것이 **풋-콜 패리티**다.

---

## LOS 74.a: 유럽형 옵션에 대한 풋-콜 패리티

---

### 만기 시 결과 비교 — 숫자로 증명

행사가격 $X = \$50$

**만기 시 $S_T = \$60$ (주가 ≥ 행사가격):**

| 포트폴리오 | 결과 |
|-----------|------|
| **보호 풋** | 주식 $60 + 풋 소멸 $0 = **$60** |
| **수탁 콜** | 채권 $50 + 콜 행사 ($60-$50) = **$60** |

**만기 시 $S_T = \$40$ (주가 < 행사가격):**

| 포트폴리오 | 결과 |
|-----------|------|
| **보호 풋** | 주식 $40 + 풋 행사 ($50-$40) = **$50** |
| **수탁 콜** | 채권 $50 + 콜 소멸 $0 = **$50** |

주가가 오르든 내리든, 두 포트폴리오의 결과는 항상 $\max(S_T, X)$로 동일하다.

---

### 풋-콜 패리티 등식

$$\underbrace{c + X(1 + R_f)^{-T}}_{\text{수탁 콜}} = \underbrace{S + p}_{\text{보호 풋}}$$

$X(1+R_f)^{-T}$인 이유: 만기에 $X$를 지급하는 무위험 채권을 지금 사려면 현재가치를 지불해야 한다.

---

### 합성 증권 — 하나가 없으면 나머지로 만든다

등식을 각 증권에 대해 정리하면 (+ = 매수, - = 매도):

$$S = c - p + X(1 + R_f)^{-T} \quad \text{(합성 주식)}$$
$$p = c - S + X(1 + R_f)^{-T} \quad \text{(합성 풋)}$$
$$c = S + p - X(1 + R_f)^{-T} \quad \text{(합성 콜)}$$
$$X(1 + R_f)^{-T} = S + p - c \quad \text{(합성 채권)}$$

시장에서 특정 옵션이 거래되지 않을 때 나머지로 "만들어" 쓸 수 있다. 합성 증권의 가격 ≠ 실제 증권의 가격이면 차익거래 기회.

---

### 예제: 풋-콜 패리티를 이용한 콜옵션 가치평가

현재 주가 $52, 무위험이자율 5%, 3개월 만기, 행사가격 $50, 풋 프리미엄 $1.50. 콜 가격은?

$$c = p + S - X(1 + R_f)^{-T} = \$1.50 + \$52 - \frac{\$50}{1.05^{0.25}} = \$1.50 + \$52 - \$49.39 = \$4.11$$

**검증:** 보호 풋 = $52 + $1.50 = $53.50. 수탁 콜 = $4.11 + $49.39 = $53.50. 성립.

## LOS 74.b: 풋-콜-선도 패리티와 기업금융 적용

---

### 풋-콜-선도 패리티

주식을 직접 보유하지 않고도 주식의 수익을 복제할 수 있다: 선도 매수(비용 0) + 만기에 선도가격을 지급할 채권 매수.

풋-콜 패리티에서 $S_0$를 $F_0(T)(1+R_f)^{-T}$로 대체:

$$F_0(T)(1 + R_f)^{-T} + p_0 = c_0 + X(1 + R_f)^{-T}$$

$$p_0 - c_0 = [X - F_0(T)](1 + R_f)^{-T}$$

풋과 콜의 가격 차이는 행사가격과 선도가격의 차이의 현재가치와 같다.

---

### 기업금융 적용 — 주주는 콜옵션 보유자다

부채가 만기 $T$, 액면가 $D$인 무이표채 하나뿐인 기업을 생각하자.

**주주의 포지션:**
기업 자산이 150억이고 부채가 100억이면? 주주 가치 = 150 - 100 = **50억** (콜 행사).
기업 자산이 80억이면? 주주 가치 = **0** (콜 포기 — 채권자에게 80억만 돌아감).

- 자산 > $D$: 빚을 갚고 나머지를 가져감 → **콜옵션 행사**
- 자산 < $D$: 못 갚겠다고 선언, 주가 = 0 → **콜옵션 포기**

유한책임(limited liability) 덕분에 주주의 손실은 투자 금액까지만이다. 콜옵션의 구조와 정확히 같다.

**채권자의 포지션:**
- $D$에 대한 무위험 청구권 - 주주에 의해 행사 가능한 풋옵션 매도
- 풋옵션 = 기업의 채무불이행 위험 = 신용 스프레드의 근원

풋-콜 패리티로:

$$V_0 = \underbrace{c_0}_{\text{주주}} + \underbrace{PV(D) - p_0}_{\text{채권자}}$$

## 퀴즈

**1.** 풋-콜 패리티가 성립하는 이유는 보호 풋이 다음과 동일한 결과를 가져오기 때문이다:

A. 커버드 콜
B. 수탁 콜
C. 무보호 콜

> **정답: B** — 보호 풋(주식 + 풋) = 수탁 콜(콜 + 채권). (LOS 74.a)

**2.** 풋-콜-선도 패리티에 포함되지 않을 가능성이 가장 높은 것은?

A. 무위험 채권
B. 콜옵션과 풋옵션
C. 기초자산

> **정답: C** — 기초자산 대신 선도계약이 사용된다. (LOS 74.b)

---

## KEY CONCEPTS

### LOS 74.a
$$c + X(1 + R_f)^{-T} = S + p$$
수탁 콜 = 보호 풋. 합성 증권: 네 가지 중 하나가 없으면 나머지 셋으로 만들 수 있다.

### LOS 74.b
풋-콜-선도 패리티: $F_0(T)(1+R_f)^{-T} + p_0 = c_0 + X(1+R_f)^{-T}$. 기업금융 적용: 주주 = 콜옵션 보유자, 채권자 = 무위험 청구권 - 풋옵션 매도.
