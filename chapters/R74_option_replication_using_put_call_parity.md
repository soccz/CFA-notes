# Reading 74: 풋-콜 패리티를 이용한 옵션 복제

---

## 시작하기 전에: 같은 결과를 주는 두 가지가 있으면, 둘의 가격도 같아야 한다

주식을 10만원에 샀다. 떨어질까 걱정된다. 두 가지 보험이 있다:

**방법 1: 주식 + 풋옵션 (보호 풋)**
주식을 갖고 있으면서, "8만원 이하로 떨어지면 8만원에 팔 수 있는 권리"(풋옵션)를 산다.
- 주가 12만원 → 풋옵션 안 씀. 내 자산 = 12만원
- 주가 6만원 → 풋옵션 행사, 8만원에 팔음. 내 자산 = 8만원

결론: 최소 8만원은 보장, 올라가면 다 내 것.

**방법 2: 채권 + 콜옵션 (수탁 콜)**
8만원짜리 무위험 채권(만기에 확실히 8만원 줌)을 사고, "8만원에 주식을 살 수 있는 권리"(콜옵션)를 산다.
- 주가 12만원 → 콜옵션 행사, 8만원에 사서 12만원짜리를 가짐. 내 자산 = 12만원
- 주가 6만원 → 콜옵션 안 씀. 채권 8만원 그대로. 내 자산 = 8만원

결론: 최소 8만원 보장, 올라가면 다 내 것. **방법 1과 완전히 같다!**

**왜 가격이 같아야 할까?** 만약 방법 1이 11만원이고 방법 2가 10만원이면? 비싼 방법 1을 팔고, 싼 방법 2를 사면 — 결과가 완전히 같으니 만기에 손해 볼 일이 없는데 1만원 차익이 생긴다. 모두가 이걸 노리면 가격은 같아질 수밖에 없다. 이것이 **풋-콜 패리티**다.

---

## LOS 74.a: 유럽형 옵션에 대한 풋-콜 패리티

---

### 만기 시 결과 비교 — 숫자로 증명

행사가격 $X = \$50$. 두 시나리오를 직접 계산해 보자.

**시나리오 1: 만기 시 $S_T = \$60$ (주가 > 행사가격)**

| 포트폴리오 | 계산 과정 | 결과 |
|-----------|----------|------|
| **보호 풋** | 주식 $60 + 풋 소멸(행사할 이유 없음) $0 | **$60** |
| **수탁 콜** | 채권 $50 + 콜 행사($60−$50) = $10 | **$60** |

**시나리오 2: 만기 시 $S_T = \$40$ (주가 < 행사가격)**

| 포트폴리오 | 계산 과정 | 결과 |
|-----------|----------|------|
| **보호 풋** | 주식 $40 + 풋 행사($50−$40) = $10 | **$50** |
| **수탁 콜** | 채권 $50 + 콜 소멸(행사할 이유 없음) $0 | **$50** |

**변수를 바꿔보자.** $S_T = \$50$(정확히 행사가격)이면? 보호 풋 = $50+$0 = $50, 수탁 콜 = $50+$0 = $50. 역시 동일하다. 주가가 뭐가 됐든, 두 포트폴리오의 결과는 항상 $\max(S_T, X)$로 같다.

---

### 풋-콜 패리티 등식

만기 결과가 항상 같으니, 오늘 가격도 같아야 한다:

$$\underbrace{c + X(1 + R_f)^{-T}}_{\text{수탁 콜}} = \underbrace{S + p}_{\text{보호 풋}}$$

**왜 $X$가 아니라 $X(1+R_f)^{-T}$인가?** 채권은 만기에 $X$를 지급한다. 지금 사려면 현재가치를 내야 한다. 1년 후 50달러를 확실히 주는 채권을, 이자율 5%일 때 오늘 50달러에 살 사람은 없다 — $50/1.05 = \$47.62$만 내면 된다.

---

### 합성 증권 — 하나가 없으면 나머지로 만든다

등식 $c + X(1+R_f)^{-T} = S + p$를 원하는 증권에 대해 정리한다 (+ = 매수, − = 매도):

$$S = c - p + X(1 + R_f)^{-T} \quad \text{(합성 주식: 콜 매수 + 풋 매도 + 채권 매수)}$$
$$p = c - S + X(1 + R_f)^{-T} \quad \text{(합성 풋: 콜 매수 + 주식 매도 + 채권 매수)}$$
$$c = S + p - X(1 + R_f)^{-T} \quad \text{(합성 콜: 주식 매수 + 풋 매수 + 채권 매도)}$$
$$X(1 + R_f)^{-T} = S + p - c \quad \text{(합성 채권: 주식 매수 + 풋 매수 + 콜 매도)}$$

**왜 이게 쓸모 있나?** 어떤 주식의 풋옵션이 시장에서 거래되지 않는다고 하자. 콜옵션은 있고, 주식도 살 수 있고, 채권도 있다. 합성 풋 공식대로 조합하면 풋옵션과 **완전히 동일한 효과**를 만들 수 있다. 합성 증권의 가격 ≠ 실제 증권의 가격이면 차익거래 기회.

---

### 예제: 풋-콜 패리티를 이용한 콜옵션 가치평가

현재 주가 $52, 무위험이자율 5%, 3개월 만기, 행사가격 $50, 풋 프리미엄 $1.50. 콜 가격은?

$$c = p + S - X(1 + R_f)^{-T} = \$1.50 + \$52 - \frac{\$50}{1.05^{0.25}} = \$1.50 + \$52 - \$49.39 = \$4.11$$

**검증:** 보호 풋 = $52 + $1.50 = $53.50. 수탁 콜 = $4.11 + $49.39 = $53.50. 양쪽이 같으니 패리티 성립.

**변수를 바꿔보자.** 만약 풋 프리미엄이 $1.50이 아니라 $3.00이면? $c = \$3.00 + \$52 - \$49.39 = \$5.61$. 풋이 비싸지면 콜도 비싸져야 패리티가 유지된다 — 둘 다 같은 주식의 보험이니까.

## LOS 74.b: 풋-콜-선도 패리티와 기업금융 적용

---

### 풋-콜-선도 패리티

카페에서 커피를 마시려면 두 가지 방법이 있다: (1) 지금 원두를 사서 보관하거나, (2) 3개월 후 인도 약속(선도계약)을 맺고 돈은 나중에 내거나. 결과는 같다 — 3개월 후 커피를 마신다.

주식도 마찬가지다. 주식을 직접 보유하는 대신, 선도 매수(비용 0) + 만기에 선도가격을 지급할 채권 매수로 같은 효과를 만든다.

풋-콜 패리티에서 $S_0$를 $F_0(T)(1+R_f)^{-T}$로 대체하면:

$$F_0(T)(1 + R_f)^{-T} + p_0 = c_0 + X(1 + R_f)^{-T}$$

정리하면:

$$p_0 - c_0 = [X - F_0(T)](1 + R_f)^{-T}$$

**숫자로 확인.** 선도가격 $F_0 = \$52$, 행사가격 $X = \$50$, $R_f = 5\%$, 1년 만기.

$$p_0 - c_0 = (\$50 - \$52) / 1.05 = -\$1.90$$

콜이 풋보다 $1.90 비싸다. **왜?** 선도가격이 행사가격보다 높으니, "살 권리"(콜)가 "팔 권리"(풋)보다 가치가 크다.

**변수를 바꿔보자.** $F_0 = X = \$50$이면? $p_0 - c_0 = 0$ → 풋과 콜의 가격이 같다. 선도가격 = 행사가격일 때 양쪽 권리의 가치가 대칭이기 때문.

---

### 기업금융 적용 — 주주는 콜옵션 보유자다

회사를 하나 세웠다. 은행에서 100억을 빌렸다(만기 1년, 액면 100억). 사업 결과에 따라 어떻게 될까?

**시나리오 1: 사업 대박, 자산 150억**
빚 100억을 갚고 남은 50억은 내 것. 주주 가치 = **50억**.

**시나리오 2: 사업 망함, 자산 80억**
빚을 다 못 갚는다. 유한책임이니 80억을 넘겨주고 끝. 주주 가치 = **0원**.

**왜 이게 콜옵션인가?** 행사가격 = 부채 100억인 콜옵션과 구조가 정확히 같다:
- 자산 > 100억: 콜 행사 → 100억 지불하고 나머지 가져감
- 자산 < 100억: 콜 포기 → 주주 손실은 투자 금액까지만(유한책임)

**변수를 바꿔보자.** 부채가 100억이 아니라 200억이면? 자산이 200억을 넘어야 주주에게 돌아오는 게 있다. 행사가격이 올라간 콜옵션 → 가치 하락. 부채가 많을수록 주식의 가치가 떨어지는 것과 정확히 같다.

**채권자의 포지션:**
- $D$(부채 액면)에 대한 무위험 청구권을 가지되, 주주에 의해 행사되는 풋옵션을 매도한 것과 같다
- 기업이 망하면(자산 < 부채) 채권자가 손실을 떠안는다 = 풋옵션이 행사된 것
- 이 풋옵션의 가격 = 신용 스프레드의 근원

풋-콜 패리티로 정리하면:

$$V_0 = \underbrace{c_0}_{\text{주주(콜)}} + \underbrace{PV(D) - p_0}_{\text{채권자(무위험 채권 − 풋)}}$$

## 퀴즈

**1.** 풋-콜 패리티가 성립하는 이유는 보호 풋이 다음과 동일한 결과를 가져오기 때문이다:

A. 커버드 콜
B. 수탁 콜
C. 무보호 콜

> **정답: B** — 보호 풋(주식 + 풋) = 수탁 콜(콜 + 채권). (LOS 74.a)

**2.** 풋-콜-선도 패리티에 포함되지 않을 가능성이 가장 높은 것은?

A. 무위험 채권
B. 콜옵션과 풋옵션
C. 기초자산

> **정답: C** — 기초자산 대신 선도계약이 사용된다. (LOS 74.b)

---

## KEY CONCEPTS

### LOS 74.a
$$c + X(1 + R_f)^{-T} = S + p$$
수탁 콜 = 보호 풋. 합성 증권: 네 가지 중 하나가 없으면 나머지 셋으로 만들 수 있다.

### LOS 74.b
풋-콜-선도 패리티: $F_0(T)(1+R_f)^{-T} + p_0 = c_0 + X(1+R_f)^{-T}$. 기업금융 적용: 주주 = 콜옵션 보유자, 채권자 = 무위험 청구권 - 풋옵션 매도.
