# Reading 75: 1기간 이항 모형을 이용한 파생상품 가치평가


## 시작하기 전에: 실제 확률을 몰라도 옵션 가격을 구할 수 있다

내일 날씨가 "맑음" 아니면 "비"다. 맑으면 아이스크림이 300원, 비가 오면 0원의 가치가 있는 쿠폰이 있다.

직관적으로 "맑을 확률 × 300 + 비 올 확률 × 0을 할인하면 되지 않나?"라고 생각할 수 있다. 하지만 놀랍게도, **실제 확률을 몰라도** 가격을 구할 수 있다.

**왜?** 아이스크림 1통(현재 200원)과 쿠폰을 적절히 조합하면, 맑든 비가 오든 가치가 **똑같아지는** 비율을 찾을 수 있기 때문이다. 가치가 확실하면 → 무위험이자율로 할인 → 쿠폰 가격을 역산한다. 확률이 끼어들 틈이 없다. 이것이 이항 모형의 핵심이다.

이것을 다른 각도에서 보면: 옵션 가격은 "사람들이 어떻게 생각하는가"(주관적 확률)가 아니라, "시장에서 어떤 거래가 가능한가"(차익거래 관계)에 의해 결정된다.


## LOS 75.a: 1기간 이항 모형으로 옵션 가치 결정


### 이항 모형에 필요한 입력값

1기간 이항 모형을 구성하려면 다음 **네 가지**가 필요하다:

| 입력값 | 설명 |
|--------|------|
| 기초자산의 현재 가치 ($S_0$) | 주식의 현재 가격 |
| 옵션의 행사가격 ($X$) | $S_0$와 다를 수 있음 |
| 상승/하락 시 가격 ($S_u$, $S_d$) | 또는 상승/하락 배수 ($R_u$, $R_d$) |
| 무위험이자율 ($R_f$) | 1기간에 대한 이자율 |

**주목:** 실제 확률은 목록에 없다. 이것이 이항 모형의 놀라운 점이다.


### 콜옵션 예제 — 처음부터 끝까지

동네 빵집 주식을 생각하자. 현재 주가 50. 1기간 후 사업이 잘 되면 60, 안 되면 42. "이 주식을 55에 살 수 있는 권리"(콜옵션)의 가격은? 무위험이자율 3%.

```
              S_u = 60  (사업 잘 됨)
             /
S_0 = 50 --
             \
              S_d = 42  (사업 안 됨)
```

**Step 1: 만기에 콜옵션은 얼마의 가치가 있나?**

주가 60이면: 콜 행사, 55에 사서 60짜리를 가짐. 이득 = 5.
주가 42이면: 55에 살 이유 없음. 콜 가치 = 0.

$$c_u = \max(0, 60 - 55) = 5$$
$$c_d = \max(0, 42 - 55) = 0$$

**Step 2: "무조건 같은 결과"가 되는 포트폴리오를 만든다**

콜옵션 1개를 매도하고, 주식을 $h$주 매수한다. 어떤 결과든 포트폴리오 가치가 같도록 $h$를 구한다:

포트폴리오 가치:
- 상승 시: $V_u = h \times 60 - 5$
- 하락 시: $V_d = h \times 42 - 0$

$V_u = V_d$로 놓으면:

$$h \times 60 - 5 = h \times 42 - 0$$

$$h(60 - 42) = 5$$

$$h = \frac{5}{60 - 42} = \frac{5}{18} = 0.278$$

**왜 이런 비율이 나오나?** 옵션의 수익 변동 폭(5 - 0 = 5)을 주가의 변동 폭(60 - 42 = 18)으로 나눈 것이다. 주가가 1달러 움직일 때 옵션이 얼마나 움직이는지를 나타내는 **헤지 비율**:

$$h = \frac{c_u - c_d}{S_u - S_d}$$

**헤지 비율의 직관:** $h = 0.278$은 "콜옵션 1개의 위험을 상쇄하려면 주식 0.278주가 필요하다"는 뜻. 옵션이 주식보다 덜 움직이기 때문에(레버리지 효과의 반대) 1주보다 적은 양이 필요하다.

**Step 3: 진짜로 가치가 같은지 확인**

$$V_u = 0.278 \times 60 - 5 = 16.68 - 5 = 11.68$$
$$V_d = 0.278 \times 42 - 0 = 11.68 \quad \checkmark$$

상승이든 하락이든 **정확히 11.68**. 불확실성이 완전히 사라졌다.

**왜 이것이 강력한가?** 포트폴리오의 미래 가치가 확실하면, 할인율로 무위험이자율을 사용할 수 있다. 위험 프리미엄이 필요 없다. 확률도 필요 없다. 오직 무위험이자율만.

**Step 4: 확실한 수익 → 무위험이자율로 할인 → 옵션 가격 역산**

$$V_0 = \frac{11.68}{1.03} = 11.34$$

$V_0 = hS_0 - c_0$이므로:

$$c_0 = hS_0 - V_0 = 0.278 \times 50 - 11.34 = 13.89 - 11.34 = 2.56$$

**Step 5: 차익거래 검증**

시장에서 이 콜이 3에 거래된다면? 고평가다.
- 콜을 3에 매도
- 주식 0.278주를 13.89에 매수
- 순 현금 유입: 3 - 13.89 = -10.89 (투자)

만기 시:
- 상승: $0.278 \times 60 - 5 = 11.68$. 투자 10.89의 수익률 = 11.68/10.89 - 1 = 7.3%
- 하락: $0.278 \times 42 = 11.68$. 수익률 = 7.3%

무위험이자율 3%보다 높은 7.3%를 무위험으로 벌 수 있다! 차익거래가 반복되면 가격은 2.56으로 수렴.

**변수를 바꿔보자.**

(a) 행사가격이 55가 아니라 50이면?
$$c_u = \max(0, 60 - 50) = 10, \quad c_d = \max(0, 42 - 50) = 0$$
$$h = 10/18 = 0.556$$
$$V_u = 0.556 \times 60 - 10 = 23.33$$
$$V_0 = 23.33/1.03 = 22.65$$
$$c_0 = 0.556 \times 50 - 22.65 = 5.14$$

행사가격이 낮아지니 콜이 더 비싸진다(2.56 → 5.14) — 당연하다, "더 싼 가격에 살 권리"이니까.

(b) 무위험이자율이 3%가 아니라 7%이면?
$$V_0 = 11.68/1.07 = 10.92$$
$$c_0 = 13.89 - 10.92 = 2.97$$

금리가 올라가면 콜 가격도 올라간다(2.56 → 2.97) — R73에서 배운 "금리 ↑ → 콜 ↑"과 일치.

(c) 하락 시 주가가 42가 아니라 30이면? (변동성 증가)
$$c_u = 5, \quad c_d = 0$$
$$h = 5/(60-30) = 0.167$$
$$V_u = 0.167 \times 60 - 5 = 5.00$$
$$V_0 = 5.00/1.03 = 4.85$$
$$c_0 = 0.167 \times 50 - 4.85 = 3.48$$

변동성이 커지면(42→30) 콜 가격도 올라간다(2.56 → 3.48) — 변동성 ↑ → 옵션 ↑.


### 풋옵션 예제

같은 빵집 주식. 현재 50, 상승 시 60, 하락 시 42. 이번엔 "이 주식을 48에 팔 수 있는 권리"(풋옵션)의 가격은? 무위험이자율 3%.

만기 시:
- $p_u = \max(0, 48 - 60) = 0$ (주가가 48보다 높으니 팔 이유 없음)
- $p_d = \max(0, 48 - 42) = 6$ (주가가 48보다 낮으니 48에 팔면 이득)

풋은 주가가 내릴 때 돈을 벌므로, **콜과 포트폴리오 구성이 다르다.** 풋은 주식과 같은 방향으로 매수한다 (둘 다 매수, 풋+주식):

$$V_u = h \times 60 + 0 = h \times 42 + 6$$

$$h(60 - 42) = 6 \quad \Rightarrow \quad h = \frac{6}{18} = 0.333$$

**왜 콜은 매도+매수이고 풋은 매수+매수인가?** 콜은 주가 상승 시 이득이고 주식도 상승 시 이득이므로, 반대 포지션(콜 매도 + 주식 매수)으로 상쇄한다. 풋은 주가 하락 시 이득이고 주식은 상승 시 이득이므로, 같은 방향(풋 매수 + 주식 매수)으로 상쇄한다.

검증:
$$V_u = 0.333 \times 60 + 0 = 20$$
$$V_d = 0.333 \times 42 + 6 = 14 + 6 = 20 \quad \checkmark$$

$$V_0 = \frac{20}{1.03} = 19.42$$

$$p_0 = V_0 - hS_0 = 19.42 - 0.333 \times 50 = 19.42 - 16.67 = 2.75$$

**변수를 바꿔보자.** 행사가격이 48이 아니라 55이면?
$$p_u = 0, \quad p_d = \max(0, 55 - 42) = 13$$
$$h = 13/18 = 0.722$$
$$V_u = 0.722 \times 60 = 43.33$$
$$V_0 = 43.33/1.03 = 42.07$$
$$p_0 = 42.07 - 0.722 \times 50 = 42.07 - 36.11 = 5.96$$

행사가격이 높아지면 풋이 비싸진다(2.75 → 5.96) — "더 비싼 가격에 팔 권리"니까.


### 헤지 비율의 일반 공식과 의미

$$h = \frac{\text{옵션의 수익 변동폭}}{\text{기초자산의 가격 변동폭}} = \frac{c_u - c_d}{S_u - S_d} \text{ (콜)} = \frac{p_u - p_d}{S_u - S_d} \text{ (풋)}$$

| 옵션 유형 | 헤지 비율 범위 | 의미 |
|-----------|--------------|------|
| 콜 | $0 \leq h \leq 1$ | 주식 1주당 콜 $1/h$개를 매도하면 무위험 |
| 풋 | $-1 \leq h \leq 0$ (절대값 사용 시 양수) | 주식과 풋을 같은 방향으로 매수 |

깊은 내가격 콜의 $h$는 1에 가깝다 — 주가와 거의 1:1로 움직인다.
깊은 외가격 콜의 $h$는 0에 가깝다 — 주가가 움직여도 콜은 거의 반응하지 않는다.


## LOS 75.b: 위험중립 가격결정


### 위험중립 확률 — 실제 확률이 아닌 "계산용 가중치"

헤지 포트폴리오 방법은 정확하지만 매번 4단계를 밟아야 한다. 더 빠른 방법이 있다: "가상의 확률"로 기대값을 구해서 할인하는 것이다.

**일상 비유.** 복권 A: 당첨되면 100만원, 꽝이면 0원. 이 복권이 시장에서 40만원에 거래된다. 그러면 "당첨 확률이 40%인 것처럼" 가격이 매겨진 셈이다 — 실제 당첨 확률이 30%든 50%든 상관없이. 이 "40%"가 위험중립 확률이다.

**왜 "위험중립"이라 부르는가?** 이 세계에서는 모든 자산의 기대수익률 = 무위험이자율이다. 위험에 대한 추가 보상(위험 프리미엄)이 0이다. 실제 세계에서는 주식의 기대수익률이 무위험이자율보다 높지만(위험 프리미엄 때문에), 위험중립 세계에서는 모든 자산이 무위험이자율로만 성장한다.


### 위험중립 확률 공식 유도

**설정:** 현재 주가 30, 상승 배수 $R_u = 1.15$ (→ 34.50), 하락 배수 $R_d = 0.87$ (→ 26.10), 무위험이자율 7%.

```
              30 × 1.15 = 34.50
             /
30 --------
             \
              30 × 0.87 = 26.10
```

위험중립 세계에서 주식의 기대수익률 = 무위험이자율:

$$\pi_u \cdot R_u + (1 - \pi_u) \cdot R_d = 1 + R_f$$

**왜 이 등식이 성립해야 하는가?** 위험중립 세계에서는 투자자가 위험에 무관심하다. 주식이든 채권이든 기대수익률이 같다. 만약 주식의 기대수익률이 무위험이자율보다 높으면, 위험중립 투자자는 모두 주식을 사려 할 것이고, 이는 균형이 아니다.

$\pi_u$에 대해 풀면:

$$\pi_u = \frac{1 + R_f - R_d}{R_u - R_d} = \frac{1 + 0.07 - 0.87}{1.15 - 0.87} = \frac{0.20}{0.28} = 0.715$$

$$\pi_d = 1 - 0.715 = 0.285$$

**검증:** $0.715 \times 1.15 + 0.285 \times 0.87 = 0.822 + 0.248 = 1.070 = 1 + 0.07$ ✓


### 콜옵션 가치 (행사가격 30)

$$c_u = \max(0, 34.50 - 30) = 4.50$$
$$c_d = \max(0, 26.10 - 30) = 0$$

위험중립 확률로 가중 평균:

$$\text{기대 수익} = 4.50 \times 0.715 + 0 \times 0.285 = 3.22$$

무위험이자율로 할인:

$$c_0 = \frac{3.22}{1.07} = 3.01$$

**이 방법이 왜 헤지 포트폴리오와 같은 답을 주는가?** 위험중립 확률은 헤지 포트폴리오에서 도출된 것이기 때문이다. 두 방법은 같은 원리(무차익)를 다른 형태로 적용한 것일 뿐이다.

**변수를 바꿔보자.**

(a) 무위험이자율이 7%가 아니라 3%이면?
$$\pi_u = (1.03 - 0.87)/(1.15 - 0.87) = 0.16/0.28 = 0.571$$
$$\text{기대 수익} = 4.50 \times 0.571 + 0 \times 0.429 = 2.57$$
$$c_0 = 2.57/1.03 = 2.50$$

무위험이자율이 낮아지면:
1. 위험중립 상승확률이 줄어듦 (0.715 → 0.571)
2. 기대 수익이 줄어듦 (3.22 → 2.57)
3. 할인율은 더 낮아지지만(1.07 → 1.03), 기대 수익 감소 효과가 더 큼
4. 콜 가격 하락 (3.01 → 2.50)

(b) 행사가격이 30이 아니라 28이면?
$$c_u = \max(0, 34.50 - 28) = 6.50, \quad c_d = \max(0, 26.10 - 28) = 0$$
$$\text{기대 수익} = 6.50 \times 0.715 = 4.65$$
$$c_0 = 4.65/1.07 = 4.34$$

행사가격 하락 → 콜 가격 상승 (3.01 → 4.34). R73에서 배운 것과 일치.

(c) 변동성이 커지면? $R_u = 1.30$, $R_d = 0.77$로 변경.
$$S_u = 39, \quad S_d = 23.10$$
$$\pi_u = (1.07 - 0.77)/(1.30 - 0.77) = 0.30/0.53 = 0.566$$
$$c_u = \max(0, 39 - 30) = 9, \quad c_d = 0$$
$$c_0 = (9 \times 0.566)/1.07 = 5.09/1.07 = 4.76$$

변동성 증가 → 콜 가격 상승 (3.01 → 4.76). 역시 R73과 일치.


### 풋옵션 가치 (행사가격 30)

$$p_u = \max(0, 30 - 34.50) = 0$$
$$p_d = \max(0, 30 - 26.10) = 3.90$$

$$\text{기대 수익} = 0 \times 0.715 + 3.90 \times 0.285 = 1.11$$

$$p_0 = \frac{1.11}{1.07} = 1.04$$


### 풋-콜 패리티 검증 — 모든 것이 연결된다

R74의 풋-콜 패리티: $c + X/(1+R_f) = S + p$

$$3.01 + \frac{30}{1.07} = 30 + 1.04$$
$$3.01 + 28.04 = 31.04$$
$$31.05 \approx 31.04 \quad \checkmark \text{ (반올림 차이)}$$

이항 모형으로 구한 콜과 풋의 가격이 풋-콜 패리티를 정확히 만족한다. 이것은 우연이 아니다 — 두 가지 다른 방법(이항 모형, 풋-콜 패리티)이 모두 **같은 무차익 원리**에 기반하기 때문이다.


### 왜 실제 확률이 아닌 위험중립 확률을 쓰는가?

내일 맑을 확률이 60%이든 80%이든, 아이스크림 쿠폰의 "공정 가격"은 헤지 포트폴리오로 결정된다. 실제 확률이 바뀌면 사람들의 기대는 바뀌지만, **무차익 가격은 변하지 않는다**.

**왜?** 무차익 가격은 "이 쿠폰과 아이스크림을 적절히 섞으면 확실한 결과를 만들 수 있다"는 **물리적 사실**에만 의존하기 때문이다. 확률은 사람마다 다르게 생각할 수 있지만, 상승 시 가격과 하락 시 가격은 모두에게 같다.

**비유:** 자동차 보험료를 정할 때, 보험회사는 "사고 확률 × 보상금"으로 계산한다고 생각할 수 있다. 하지만 실제로는 헤지(재보험) 비용에 기반한다. 재보험 시장에서 "사고 시 100만원 보상 계약"의 가격이 정해져 있으면, 그것이 보험료의 기준이다 — 사고 확률에 대한 의견이 달라도.

**위험중립 확률의 핵심 속성:**
- 실제 확률이 아니다 (pseudo-probability)
- 모든 자산의 기대수익률을 무위험이자율로 만드는 가중치
- 무차익 관계에서 자동으로 도출된다
- 어떤 옵션이든(콜, 풋, 행사가격 관계없이) 같은 위험중립 확률을 사용


### 두 방법의 비교 — 어떤 것을 쓸까?

| | 헤지 포트폴리오 방법 | 위험중립 가격결정 |
|---|---|---|
| **원리** | 무위험 포트폴리오 → 무위험이자율로 할인 | 위험중립 기대값 → 무위험이자율로 할인 |
| **장점** | 직관적, 차익거래를 직접 보여줌 | 계산이 빠름, 다기간 확장 용이 |
| **단점** | 매번 헤지 비율 계산 필요 | "가상 확률"이 직관에 반함 |
| **결과** | 동일 | 동일 |

시험에서는 두 방법 모두 나올 수 있다. 위험중립 방법이 더 빠르지만, 헤지 포트폴리오가 "왜 이 가격이 맞는가"를 더 명확히 보여준다.


### 이항 모형의 한계와 확장

**한계:**
- 1기간 모형은 가격이 두 가지로만 움직인다고 가정 — 비현실적
- 실제 주가는 연속적으로 움직임

**확장:**
- 기간 수를 늘려 현실에 가까워짐 (2기간 → 3기간 → ... → n기간)
- n이 무한대로 가면 블랙-숄즈 모형에 수렴

**2기간 모형의 간략한 구조:**
```
              S_uu = 72
             /
      S_u = 60
     /       \
S_0 = 50     S_ud = 50.40
     \       /
      S_d = 42
             \
              S_dd = 35.28
```

2기간에서는 만기에 3가지 가능한 가격이 생긴다. 각 노드에서 위험중립 확률로 기대값을 구하고, 한 기간씩 뒤로 할인해 나간다. 원리는 1기간과 완전히 동일.


### 실전 적용: 다기간 모형으로의 확장

실제로는 1기간이 아니라 수십, 수백 기간의 이항 트리를 사용한다. 기간을 늘릴수록:
- 각 기간의 가격 변동이 작아짐
- 가능한 만기 가격의 수가 늘어남 (2기간이면 3개, 3기간이면 4개...)
- 극한에서 **블랙-숄즈 모형**에 수렴

CFA Level I에서는 1기간 모형만 요구하지만, 원리는 동일하다: 매 기간마다 위험중립 확률로 기대값을 구하고 무위험이자율로 할인.


## 퀴즈

**1.** 1기간 이항 모형을 구성할 때 기초자산 가격의 상승/하락 확률이 필요한가?

A. 필요하지 않다.
B. 필요하지만, 상승/하락 수익률로부터 계산할 수 있다.
C. 필요하며, 실제 확률의 추정치가 있어야 한다.

> **정답: A** — 복제와 무차익 가격결정에 기반하므로 실제 확률 불필요. 헤지 포트폴리오는 어떤 결과든 가치가 같으므로 확률이 개입하지 않는다. (LOS 75.a)

**2.** 위험중립성에 기반한 1기간 이항 모형에서, 옵션의 가치는?

A. 두 가지 가능한 결과의 확률 가중 평균의 현재가치
B. 선택된 수의 가능한 결과의 확률 가중 평균
C. 하나의 결과에 기반한 값

> **정답: A** — 위험중립 확률로 가중 평균한 기대 수익을 무위험이자율로 할인. 1기간이므로 두 가지 가능한 결과(상승, 하락). (LOS 75.b)

**3.** 위험중립 확률을 사용하는 이유는?

A. 모형이 무차익 관계에 기반하기 때문이다.
B. 실제 확률의 불편 추정량이기 때문이다.
C. 매수자가 외가격 옵션을 행사하지 않기 때문이다.

> **정답: A** — 위험중립 확률은 무차익 조건에서 도출된다. 헤지 포트폴리오가 확실한 결과를 만들기 때문에, 이 "가상 확률"로 기대값을 구해 무위험이자율로 할인할 수 있다. B는 틀림 — 위험중립 확률은 실제 확률의 추정치가 아니다. (LOS 75.b)


## KEY CONCEPTS

### LOS 75.a
1기간 이항 모형: 기초자산 현재 가치, 행사가격, 상승/하락 시 가격, 무위험이자율이 필요. **실제 확률은 불필요.** 기초자산과 옵션을 결합하여 확실한 포트폴리오를 구성하고, 무위험이자율로 할인하여 옵션 가격을 역산. 헤지 비율: $h = (c_u - c_d)/(S_u - S_d)$. 시장 가격이 모형 가격과 다르면 차익거래 기회.

### LOS 75.b
위험중립 확률: $\pi_u = (1 + R_f - R_d)/(R_u - R_d)$. 모든 자산의 기대수익률을 무위험이자율로 만드는 가상의 확률. 옵션 가치 = 위험중립 확률로 가중 평균한 기대 수익을 무위험이자율로 할인. 실제 확률은 무차익 가격에 영향을 주지 않는다. 헤지 포트폴리오 방법과 위험중립 가격결정은 같은 원리(무차익)의 두 가지 표현이며, 항상 같은 답을 준다. 이항 모형으로 구한 옵션 가격은 풋-콜 패리티를 만족한다.


## 추가 연습문제

**4.** 현재 주가 40, 상승 시 52, 하락 시 32, 무위험이자율 5%, 행사가격 45인 콜옵션의 가치는?

$$c_u = \max(0, 52-45) = 7, \quad c_d = 0$$
$$h = 7/(52-32) = 0.35$$
$$V_u = 0.35 \times 52 - 7 = 11.20$$
$$V_0 = 11.20/1.05 = 10.67$$
$$c_0 = 0.35 \times 40 - 10.67 = 14.00 - 10.67 = 3.33$$

A. 3.33
B. 2.56
C. 7.00

> **정답: A** (LOS 75.a)
