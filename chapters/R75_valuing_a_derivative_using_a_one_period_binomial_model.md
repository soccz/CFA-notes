# Reading 75: 1기간 이항 모형을 이용한 파생상품 가치평가

---

## 시작하기 전에: 실제 확률을 몰라도 옵션 가격을 구할 수 있다

내일 날씨가 "맑음" 아니면 "비"다. 맑으면 아이스크림이 300원, 비가 오면 0원의 가치가 있는 쿠폰이 있다.

직관적으로 "맑을 확률 × 300 + 비 올 확률 × 0을 할인하면 되지 않나?"라고 생각할 수 있다. 하지만 놀랍게도, **실제 확률을 몰라도** 가격을 구할 수 있다.

**왜?** 아이스크림 1통(현재 200원)과 쿠폰을 적절히 조합하면, 맑든 비가 오든 가치가 **똑같아지는** 비율을 찾을 수 있기 때문이다. 가치가 확실하면 → 무위험이자율로 할인 → 쿠폰 가격을 역산한다. 확률이 끼어들 틈이 없다. 이것이 이항 모형의 핵심이다.

---

## LOS 75.a: 1기간 이항 모형으로 옵션 가치 결정

---

### 콜옵션 예제 — 처음부터 끝까지

동네 빵집 주식을 생각하자. 현재 주가 $50. 1기간 후 사업이 잘 되면 $60, 안 되면 $42. "이 주식을 $55에 살 수 있는 권리"(콜옵션)의 가격은? 무위험이자율 3%.

```
              S_u = $60  (사업 잘 됨)
             /
S_0 = $50 --
             \
              S_d = $42  (사업 안 됨)
```

**Step 1: 만기에 콜옵션은 얼마의 가치가 있나?**

주가 $60이면: 콜 행사, $55에 사서 $60짜리를 가짐. 이득 = $5.
주가 $42이면: $55에 살 이유 없음. 콜 가치 = $0.

$$c_u = \max(0, \$60 - \$55) = \$5$$
$$c_d = \max(0, \$42 - \$55) = \$0$$

**Step 2: "무조건 같은 결과"가 되는 포트폴리오를 만든다**

콜옵션 1개를 매도하고, 주식을 $h$주 매수한다. 어떤 날씨든 포트폴리오 가치가 같도록 $h$를 구한다:

$$h \times \$60 - \$5 = h \times \$42 - \$0$$

$$h = \frac{\$5}{\$60 - \$42} = \frac{\$5}{\$18} = 0.278$$

**왜 이런 비율이 나오나?** 옵션의 수익 변동 폭($5)을 주가의 변동 폭($18)으로 나눈 것이다. 주가가 1달러 움직일 때 옵션이 얼마나 움직이는지를 나타내는 **헤지 비율**:

$$h = \frac{c_u - c_d}{S_u - S_d}$$

**Step 3: 진짜로 가치가 같은지 확인**

$$V_u = 0.278 \times \$60 - \$5 = \$11.68$$
$$V_d = 0.278 \times \$42 - \$0 = \$11.68 \quad \checkmark$$

상승이든 하락이든 **정확히 $11.68**. 불확실성이 완전히 사라졌다.

**Step 4: 확실한 수익 → 무위험이자율로 할인 → 옵션 가격 역산**

$$V_0 = \frac{\$11.68}{1.03} = \$11.34$$

$V_0 = hS_0 - c_0$이므로:

$$c_0 = 0.278 \times \$50 - \$11.34 = \$13.89 - \$11.34 = \$2.56$$

**변수를 바꿔보자.** 시장에서 이 콜이 $3에 거래된다면? 고평가다. 콜을 $3에 매도하고, 0.278주를 매수하면 $0.44의 무위험 이익. 차익거래가 반복되면 가격은 $2.56으로 수렴한다.

행사가격이 $55가 아니라 $50이면? $c_u = \$10$, $c_d = \$0$, $h = 10/18 = 0.556$. 행사가격이 낮아지니 콜이 더 비싸진다 — 당연하다, "더 싼 가격에 살 권리"이니까.

---

### 풋옵션 예제

같은 빵집 주식. 현재 $50, 상승 시 $60, 하락 시 $42. 이번엔 "이 주식을 $48에 팔 수 있는 권리"(풋옵션)의 가격은? 무위험이자율 3%.

만기 시:
- $p_u = \max(0, \$48 - \$60) = \$0$ (주가가 $48보다 높으니 팔 이유 없음)
- $p_d = \max(0, \$48 - \$42) = \$6$ (주가가 $48보다 낮으니 $48에 팔면 이득)

풋은 주가가 내릴 때 돈을 벌므로, 주식 **매수**와 결합하여 헤지한다:

$$h \times \$60 + \$0 = h \times \$42 + \$6 \quad \Rightarrow \quad h = \frac{\$6}{\$18} = 0.333$$

$$V_u = 0.333 \times \$60 + \$0 = \$20$$
$$V_d = 0.333 \times \$42 + \$6 = \$20 \quad \checkmark$$

$$V_0 = \frac{\$20}{1.03} = \$19.42$$

$$p_0 = V_0 - hS_0 = \$19.42 - 0.333 \times \$50 = \$19.42 - \$16.67 = \$2.75$$

**변수를 바꿔보자.** 행사가격이 $48이 아니라 $55이면? $p_d = \$13$, $h = 13/18 = 0.722$. 행사가격이 높아지면 풋이 비싸진다 — "더 비싼 가격에 팔 권리"니까.

## LOS 75.b: 위험중립 가격결정

---

### 위험중립 확률 — 실제 확률이 아닌 "계산용 가중치"

헤지 포트폴리오 방법은 정확하지만 매번 4단계를 밟아야 한다. 더 빠른 방법이 있다: "가상의 확률"로 기대값을 구해서 할인하는 것이다.

**일상 비유.** 복권 A: 당첨되면 100만원, 꽝이면 0원. 이 복권이 시장에서 40만원에 거래된다. 그러면 "당첨 확률이 40%인 것처럼" 가격이 매겨진 셈이다 — 실제 당첨 확률이 30%든 50%든 상관없이. 이 "40%"가 위험중립 확률이다.

**설정:** 현재 주가 $30, 상승 배수 $R_u = 1.15$ (→ $34.50), 하락 배수 $R_d = 0.87$ (→ $26.10), 무위험이자율 7%.

```
              $30 × 1.15 = $34.50
             /
$30 --------
             \
              $30 × 0.87 = $26.10
```

**위험중립 확률 공식 유도:**

위험중립 세계에서는 모든 자산의 기대수익률 = 무위험이자율이다. 왜? 위험에 대한 보상이 없다고 "가정"하는 세계이기 때문이다.

$$\pi_u \cdot R_u + (1 - \pi_u) \cdot R_d = 1 + R_f$$

$\pi_u$에 대해 풀면:

$$\pi_u = \frac{1 + R_f - R_d}{R_u - R_d} = \frac{1 + 0.07 - 0.87}{1.15 - 0.87} = \frac{0.20}{0.28} = 0.715$$

$$\pi_d = 1 - 0.715 = 0.285$$

---

### 콜옵션 가치 (행사가격 $30)

$$c_u = \max(0, \$34.50 - \$30) = \$4.50$$
$$c_d = \max(0, \$26.10 - \$30) = \$0$$

위험중립 확률로 가중 평균:

$$\text{기대 수익} = \$4.50 \times 0.715 + \$0 \times 0.285 = \$3.22$$

무위험이자율로 할인:

$$c_0 = \frac{\$3.22}{1.07} = \$3.01$$

**변수를 바꿔보자.** 무위험이자율이 7%가 아니라 3%이면? $\pi_u = (1.03 - 0.87)/(0.28) = 0.571$. 기대 수익 = $4.50 × 0.571 = $2.57. $c_0 = $2.57/1.03 = $2.50$. 무위험이자율이 낮아지면 위험중립 상승확률이 줄어들고, 콜 가격도 내려간다.

---

### 풋옵션 가치 (행사가격 $30)

$$p_u = \max(0, \$30 - \$34.50) = \$0$$
$$p_d = \max(0, \$30 - \$26.10) = \$3.90$$

$$\text{기대 수익} = \$0 \times 0.715 + \$3.90 \times 0.285 = \$1.11$$

$$p_0 = \frac{\$1.11}{1.07} = \$1.04$$

---

### 풋-콜 패리티 검증 — 모든 것이 연결된다

R74의 풋-콜 패리티: $c + X/(1+R_f) = S + p$

$$\$3.01 + \frac{\$30}{1.07} = \$30 + \$1.04$$
$$\$3.01 + \$28.04 = \$31.04$$
$$\$31.05 \approx \$31.04 \quad \checkmark \text{ (반올림 차이)}$$

이항 모형으로 구한 콜과 풋의 가격이 풋-콜 패리티를 정확히 만족한다. 두 가지 다른 방법으로 같은 답에 도달한 것이다.

---

### 왜 실제 확률이 아닌 위험중립 확률을 쓰는가?

내일 맑을 확률이 60%이든 80%이든, 아이스크림 쿠폰의 "공정 가격"은 헤지 포트폴리오로 결정된다. 실제 확률이 바뀌면 사람들의 기대는 바뀌지만, **무차익 가격은 변하지 않는다**.

**왜?** 무차익 가격은 "이 쿠폰과 아이스크림을 적절히 섞으면 확실한 결과를 만들 수 있다"는 **물리적 사실**에만 의존하기 때문이다. 확률은 사람마다 다르게 생각할 수 있지만, 상승 시 가격과 하락 시 가격은 모두에게 같다.

## 퀴즈

**1.** 1기간 이항 모형을 구성할 때 기초자산 가격의 상승/하락 확률이 필요한가?

A. 필요하지 않다.
B. 필요하지만, 상승/하락 수익률로부터 계산할 수 있다.
C. 필요하며, 실제 확률의 추정치가 있어야 한다.

> **정답: A** — 복제와 무차익 가격결정에 기반하므로 실제 확률 불필요. (LOS 75.a)

**2.** 위험중립성에 기반한 1기간 이항 모형에서, 옵션의 가치는?

A. 두 가지 가능한 결과의 확률 가중 평균의 현재가치
B. 선택된 수의 가능한 결과의 확률 가중 평균
C. 하나의 결과에 기반한 값

> **정답: A** — 위험중립 확률로 가중 평균한 기대 수익을 무위험이자율로 할인. (LOS 75.b)

**3.** 위험중립 확률을 사용하는 이유는?

A. 모형이 무차익 관계에 기반하기 때문이다.
B. 실제 확률의 불편 추정량이기 때문이다.
C. 매수자가 외가격 옵션을 행사하지 않기 때문이다.

> **정답: A** (LOS 75.b)

## KEY CONCEPTS

### LOS 75.a
1기간 이항 모형: 기초자산 현재 가치, 행사가격, 상승/하락 시 가격, 무위험이자율이 필요. 기초자산과 옵션을 결합하여 확실한 포트폴리오를 구성하고, 무위험이자율로 할인하여 옵션 가격을 역산. 헤지 비율: $h = (c_u - c_d)/(S_u - S_d)$.

### LOS 75.b
위험중립 확률: $\pi_u = (1 + R_f - R_d)/(R_u - R_d)$. 옵션 가치 = 위험중립 확률로 가중 평균한 기대 수익을 무위험이자율로 할인. 실제 확률은 무차익 가격에 영향을 주지 않는다.
