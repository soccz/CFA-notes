# Reading 55: 이자율의 기간 구조 (The Term Structure of Interest Rates)

> **핵심 질문:** 현물이자율, 액면가 수익률, 선도이자율은 무엇이고, 어떻게 연결되는가?

---

## 왜 "기간별로 다른 이자율"이 존재하는가?

은행에서 정기예금 금리를 보면, 1년짜리와 5년짜리가 다르다. 보통 5년짜리가 더 높다. 왜? 더 오래 기다려야 하니까 더 많은 보상을 요구한다.

채권도 마찬가지다. R52에서 YTM으로 채권 가격을 구할 때, 우리는 모든 현금흐름에 **같은 할인율**을 적용했다. 하지만 현실에서는 1년 후에 돌려받을 돈과 10년 후에 돌려받을 돈의 "시간 가치"가 다르다. 기간마다 이자율이 다른 것 — 이것이 **이자율의 기간 구조**다.

이 Reading에서는 이 "기간별로 다른 이자율"을 세 가지 도구로 정확히 표현하는 방법을 배운다: **현물이자율(spot rate)**, **선도이자율(forward rate)**, **액면가 수익률(par yield)**.

---

## MODULE 55.1: 현물 곡선, 액면가 곡선, 선도 곡선

### LOS 55.a: 현물이자율과 현물 곡선

#### 왜 현물이자율이 필요한가?

YTM은 "모든 현금흐름에 동일한 할인율을 적용한 평균적 수익률"이다. 하지만 실제로 1년 후 쿠폰, 3년 후 원금은 각각 다른 시간 가치를 갖는다. 각 시점에 맞는 정확한 할인율이 **현물이자율**이다.

**현물이자율** = "지금 돈을 빌려주고, n년 후에 딱 한 번 돌려받을 때의 연간 이자율." 무이표채(zero-coupon bond)의 수익률과 같다.

#### 현물이자율로 채권 가격 계산

$$\text{채권 가격} = \frac{CPN}{(1+S_1)^1} + \frac{CPN}{(1+S_2)^2} + \cdots + \frac{CPN + FV}{(1+S_n)^n}$$

YTM처럼 하나의 할인율을 쓰는 대신, 각 기간의 실제 시장 할인율을 적용한다. 이렇게 구한 가격이 **무차익 가격(no-arbitrage price)**이다.

#### EXAMPLE: 현물이자율로 채권 가치 평가

현물이자율: 1년 3%, 2년 4%, 3년 5%.
3년 만기, 5% 연간 쿠폰 채권의 가치:

$$\frac{5}{(1.03)^1} + \frac{5}{(1.04)^2} + \frac{105}{(1.05)^3} = 100.180$$

각 쿠폰을 해당 기간의 현물이자율로 할인한다. 이 가격(100.180)이 무차익 가격. YTM으로 역산하면:

N = 3, PMT = 5, FV = 100, PV = -100.180 → I/Y = **4.93%**

YTM(4.93%)은 세 현물이자율(3%, 4%, 5%)의 가중평균이다. 현금흐름의 대부분이 3년차(원금 105)에 집중되어 있으므로, YTM이 5%에 가깝되 약간 낮다.

#### 현물 곡선

만기별 현물이자율을 그래프로 나타낸 것. **제로 곡선(zero curve)** 또는 **스트립 곡선(strip curve)**이라고도 한다.

### LOS 55.b: 액면가 수익률과 선도이자율

#### 액면가 수익률 (Par Yield)

"만기 n년짜리 채권을 **액면가에 발행**하려면 쿠폰율을 몇 %로 설정해야 하는가?" 에 답하는 것이 액면가 수익률이다.

현물이자율 $S_1 = 1\%$, $S_2 = 2\%$, $S_3 = 3\%$일 때 3년 액면가 수익률:

$$\frac{PMT}{(1.01)} + \frac{PMT}{(1.02)^2} + \frac{PMT + 100}{(1.03)^3} = 100$$

풀면 $PMT = 2.96$이므로 **3년 액면가 수익률 = 2.96%**.

#### 선도이자율 (Forward Rate)

"미래 특정 시점에 대출을 받으면 금리가 얼마일까?"를 현재 시점에서 알 수 있는 이자율이다.

표기법: **1y1y** = 지금부터 1년 후에 시작하는 1년 만기 대출 금리

#### 핵심 무차익 관계

3년 동안 빌리는 비용은, 3년 현물이자율로 한 번에 빌리든, 1년씩 세 번 연속 빌리든 **같아야** 한다 (아니면 차익거래 가능).

$$\boxed{(1+S_3)^3 = (1+S_1)(1+1y1y)(1+2y1y)}$$

현물이자율은 선도이자율의 **기하평균**:

$$S_3 = \left[(1+S_1)(1+1y1y)(1+2y1y)\right]^{1/3} - 1$$

#### EXAMPLE: 선도이자율 → 현물이자율

$S_1 = 2\%$, 1y1y = 3%, 2y1y = 4%.

$$S_3 = \left[(1.02)(1.03)(1.04)\right]^{1/3} - 1 = 2.997\%$$

단순평균 근사: (2% + 3% + 4%) / 3 = 3%. 정확한 값 2.997%와 거의 같다.

#### EXAMPLE: 현물이자율 → 선도이자율

$S_1 = 4\%$, $S_2 = 8\%$. 1y1y는?

$$(1.08)^2 = (1.04)(1+1y1y)$$

$$1y1y = \frac{(1.08)^2}{(1.04)} - 1 = 12.154\%$$

투자자가 1년 채권 4%를 수용하는 이유: 1년 후에 12.154%짜리 1년 채권에 투자할 수 있다고 기대하기 때문이다.

**근사법:** $2 \times 8\% - 1 \times 4\% = 12\%$ (정확한 값 12.154%와 가까움).

#### 다기간 선도이자율

$S_2 = 6\%$, $S_4 = 8\%$일 때 2y2y:

$$2y2y = \left[\frac{(1.08)^4}{(1.06)^2}\right]^{1/2} - 1 = 10.04\%$$

근사: $(4 \times 8 - 2 \times 6) / 2 = 10\%$.

#### 선도이자율로 채권 가치 평가

$S_1 = 4\%$, 1y1y = 5%, 2y1y = 6%. 3년, 5% 쿠폰, $1,000 채권:

$$\frac{50}{(1.04)} + \frac{50}{(1.04)(1.05)} + \frac{1{,}050}{(1.04)(1.05)(1.06)}$$

각 현금흐름의 할인계수는 해당 기간까지의 선도이자율을 모두 곱한 것이다. 현물이자율로 할인하는 것과 동일한 결과.

### LOS 55.c: 현물 곡선, 액면가 곡선, 선도 곡선의 비교

선도이자율이 상승하면 → 기하평균인 현물이자율도 상승하되 더 완만 → 현물이자율의 가중평균인 액면가 수익률은 현물보다 약간 낮음.

| 수익률 곡선 환경 | 관계 |
|----------------|------|
| **우상향 (정상)** | 선도 > 현물 > 액면가 수익률 |
| **우하향 (역전)** | 선도 < 현물 < 액면가 수익률 |
| **수평 (flat)** | 선도 = 현물 = 액면가 수익률 |

단순 예시: $S_1 = 1\%$, 1y1y = 3%. 선도가 상승(1%→3%) → $S_2 \approx 2\%$ (더 완만) → 액면가 수익률은 2%보다 약간 낮음.

## KEY CONCEPTS

### LOS 55.a
- **현물이자율:** 미래 단일 지급액의 할인율. 무차익 가격 = 각 현금흐름을 해당 현물이자율로 할인하여 합산

### LOS 55.b
- **액면가 수익률:** 현물 곡선에서 도출, 채권이 액면가에 거래되기 위한 쿠폰율
- **선도이자율:** 미래 기간의 대출/투자 이자율. 현물이자율은 선도이자율의 기하평균
- 두 현물이자율에서 선도이자율을 역산 가능

### LOS 55.c
- 우상향: 선도 > 현물 > 액면가
- 우하향: 선도 < 현물 < 액면가
- 수평: 세 곡선 일치
