# Reading 8: Hypothesis Testing


## LOS 8.a: 가설 검정의 구성 요소

### 왜 가설 검정이 필요한가?

동전이 공정한지 확인하고 싶습니다. 10번 던져서 7번 앞면이 나왔다면, 조작된 건가요? "7/10 = 70%이니까 조작이다"라고 단정하기엔 이릅니다. 공정한 동전도 10번 중 7번 앞면이 나올 수 있으니까요. **"이 결과가 우연히 발생할 확률은 얼마인가?"** — 이 질문에 체계적으로 답하는 것이 가설 검정입니다.

펀드의 평균 수익률이 +0.1%인데, 이게 진짜 양의 수익률인지 우연인지 어떻게 알 수 있을까요? 가설 검정은 **표본 데이터로 모집단 특성에 대한 주장을 검증**하는 과학적 절차입니다.

#### 일상 속 가설 검정

**시나리오 1 — 신약 임상 시험:**
제약사가 신약을 개발했습니다. "이 약이 위약(placebo)보다 효과가 있다"고 주장하려면, 환자 그룹에게 신약을 투여하고 위약 그룹과 비교합니다. 신약 그룹의 회복률이 60%, 위약 그룹이 45%입니다. 이 15%p 차이가 "우연"일 수 있을까요? 환자 수가 20명이면 우연일 수 있지만, 10,000명이면 거의 확실히 진짜 효과입니다. 가설 검정은 이 판단을 수학적으로 내려줍니다.

**시나리오 2 — 마케팅 A/B 테스트:**
이커머스 회사가 버튼 색상을 빨간색(A)에서 파란색(B)으로 바꿨더니 전환율이 2.1%에서 2.4%로 올랐습니다. 이 0.3%p 차이가 진짜일까요?
- 방문자 100명: 우연일 가능성 높음 (p-값 ≈ 0.45)
- 방문자 100,000명: 거의 확실히 진짜 차이 (p-값 ≈ 0.001)

**왜 이런 결과가 나올까?** 표본이 작으면 변동이 크기 때문입니다. 동전을 4번 던져서 3번 앞면이 나올 확률은 25%나 되지만, 400번 던져서 300번 앞면이 나올 확률은 사실상 0입니다. 표본 크기가 클수록 "우연"의 범위가 좁아집니다.

**시나리오 3 — 펀드 매니저 실력 검증:**
펀드 매니저가 3년간 S&P 500을 연 평균 2% 초과 달성했습니다. 이게 실력일까요?
- H₀: 초과 수익 = 0 (실력 없음, 운이 좋았을 뿐)
- Hₐ: 초과 수익 > 0 (진짜 실력)
- 36개월 데이터: 월 초과 수익 = 0.167%, 표준편차 = 1.2%
- 표준오차 = 1.2% / √36 = 0.2%
- t = 0.167% / 0.2% = 0.835

t = 0.835 < 1.69 (단측 5% 임계값, df=35). **기각 실패.** 3년치 데이터로는 이 매니저의 실력을 통계적으로 입증할 수 없습니다.

**만약 10년(120개월) 데이터라면?**
- 표준오차 = 1.2% / √120 = 0.1095%
- t = 0.167% / 0.1095% = 1.525

여전히 기각하기 어렵습니다. **변수를 바꾸면?** 월 초과 수익이 0.5%였다면:
- t(3년) = 0.5% / 0.2% = 2.5 → 기각! 실력 인정
- t(10년) = 0.5% / 0.1095% = 4.57 → 훨씬 강력한 증거

교훈: 작은 효과를 입증하려면 훨씬 많은 데이터가 필요합니다.

### 가설 검정 7단계 — 각 단계가 왜 필요한가

1. **가설 설정 (H₀와 Hₐ)** — "무엇을 증명하고 싶은가?" 명확히 해야 검정 방향이 결정됩니다
2. **검정 통계량 선택** — 데이터 특성(정규분포? 표본 크기?)에 따라 t, z, χ², F 중 선택
3. **유의수준 설정** — "얼마나 확실해야 믿을 건가?" α = 5%가 관례지만, 의학은 1%, 탐색적 연구는 10%도 사용
4. **결정 규칙** — "검정 통계량이 얼마 이상이면 기각" 기준을 사전에 설정. 데이터를 본 후 기준을 바꾸면 과학이 아닌 조작
5. **표본 수집·계산** — 실제 데이터를 모아 검정 통계량 계산
6. **통계적 결정** — 검정 통계량 vs 임계값 비교
7. **해석** — 통계적 유의성이 실질적 중요성을 의미하는지 판단

**왜 7단계가 필요한가?** 3단계(유의수준)를 6단계(결정) 후에 바꾸면, 원하는 결과가 나올 때까지 기준을 조작할 수 있습니다. 과학적 검정은 **데이터를 보기 전에** 기준을 정해야 합니다. 이것이 "사전 등록(pre-registration)"의 핵심입니다.

### 귀무가설과 대립가설

**귀무가설 (H₀):** 기각하고 싶은 가설. 항상 "등호(=)"를 포함. "효과 없음"이 기본 가정.

**대립가설 (Hₐ):** 실제로 보여주고 싶은 것. H₀가 기각되면 결론이 됨.

"귀무가설을 채택한다"고 말하지 않습니다 — 증명할 수 없고, 단지 기각하지 못할 뿐입니다.

**왜 "채택"이 아닌 "기각 실패"인가?** 법정에서 "무죄"는 "결백이 입증되었다"가 아니라 "유죄를 입증하지 못했다"입니다. 증거 불충분일 뿐 진짜 결백한지는 모릅니다. 마찬가지로 H₀를 기각하지 못한 것은 H₀가 참이라는 증거가 아닙니다.

#### 단측 vs 양측 검정 — 언제 어떤 걸 쓰는가?

**양측 검정:** "차이가 있는가?" (방향 불문)
- H₀: μ = μ₀ vs Hₐ: μ ≠ μ₀
- 임계값 양쪽에 α/2씩 배분
- 예: "이 약이 혈압에 영향을 주는가?" (올릴 수도, 내릴 수도)

**단측 검정:** "특정 방향으로 차이가 있는가?"
- H₀: μ ≤ μ₀ vs Hₐ: μ > μ₀ (오른꼬리)
- 또는 H₀: μ ≥ μ₀ vs Hₐ: μ < μ₀ (왼꼬리)
- 예: "이 약이 혈압을 낮추는가?" (방향이 정해져 있음)

**수치 시나리오:**
- 같은 검정 통계량 t = 1.72에 대해:
  - 양측 5% 검정: 임계값 ±1.96 → 기각 실패
  - 단측 5% 검정: 임계값 1.65 → **기각**

같은 데이터인데 검정 유형에 따라 결론이 달라집니다. 단측 검정은 검정력이 높지만, 방향을 사전에 정당화해야 합니다.

### 검정 통계량

$$\text{검정 통계량} = \frac{\text{표본 통계량} - \text{귀무가설 값}}{\text{표준오차}}$$

표본 통계량이 귀무가설에서 가정한 값으로부터 **몇 표준오차 떨어져 있는지**를 나타냅니다. 3 표준오차 떨어져 있으면 우연일 확률이 매우 낮습니다(±3σ 밖 ≈ 0.3%).

#### 수식 유도 — 왜 이 형태인가?

표본 평균 $\bar{X}$의 분포를 생각해봅시다. 중심극한정리에 의해:

$$\bar{X} \sim N\left(\mu, \frac{\sigma^2}{n}\right)$$

이를 표준화하면:

$$z = \frac{\bar{X} - \mu}{\sigma / \sqrt{n}}$$

H₀: μ = μ₀ 하에서:

$$z = \frac{\bar{X} - \mu_0}{\sigma / \sqrt{n}}$$

σ를 모르면 s로 대체하고, 분포가 z에서 t로 바뀝니다:

$$t = \frac{\bar{X} - \mu_0}{s / \sqrt{n}}, \quad df = n - 1$$

**왜 표준오차로 나누는가?** 표본 평균이 귀무가설 값에서 2% 떨어져 있다고 합시다. 표준오차가 1%이면 2배 거리(유의할 수 있음). 표준오차가 5%이면 0.4배 거리(충분히 우연). 같은 편차도 변동성 대비 얼마나 큰지가 중요합니다.

**표본 크기 n이 매우 크면** 표준오차(= s/√n)가 작아져 아주 작은 차이도 "통계적으로 유의"해집니다. 이것이 **통계적 유의성 ≠ 실질적 중요성**인 이유입니다.

**숫자로 느끼기:**
- n = 25: 표준오차 = s/5
- n = 10,000: 표준오차 = s/100 (20배 작음)

0.01%의 차이도 n = 10,000이면 유의할 수 있지만, 투자 의사결정에 0.01% 차이가 의미가 있을까요? 이것이 "경제적 유의성(economic significance)"을 별도로 따져야 하는 이유입니다.

### 주요 z-분포 임계값

| 임계값 | 사용 상황 | 기각역 확률 |
|--------|----------|------------|
| **1.65** | 양측 10%, 단측 5% | 꼬리 한쪽 5% |
| **1.96** | 양측 5% | 꼬리 한쪽 2.5% |
| **2.33** | 단측 1% | 꼬리 한쪽 1% |
| **2.58** | 양측 1% | 꼬리 한쪽 0.5% |

**암기 팁:** 1.65 → 2.33은 단측, 1.96 → 2.58은 양측. 양측은 항상 "절반의 확률"이므로 더 큰 임계값이 필요합니다.

**왜 1.96인가?** 표준정규분포에서 P(Z > 1.96) = 0.025이므로, 양쪽 합이 0.05 = 5%입니다. 이 숫자 자체를 외워야 합니다.

### 1종 오류와 2종 오류

| | H₀ 참 | H₀ 거짓 |
|--|-------|---------|
| H₀ 기각 안 함 | 올바른 결정 | 2종 오류 (β) |
| H₀ 기각 | 1종 오류 (α) | 올바른 결정 (검정력=1−β) |

- **1종 오류:** 무고한 사람을 유죄 판결. 확률 = α (유의수준)
- **2종 오류:** 유죄인 사람을 무죄 판결. 확률 = β

#### 왜 두 오류가 중요한가?

**시나리오 — 금융 사기 탐지:**
은행이 거래를 "사기" 또는 "정상"으로 분류합니다.
- **1종 오류:** 정상 거래를 사기로 판정 → 고객 계좌 동결, 불편 초래
- **2종 오류:** 사기 거래를 정상으로 판정 → 실제 사기 피해 발생

어떤 오류가 더 나쁜가? 상황에 따라 다릅니다. 은행은 사기 피해(2종 오류)가 더 치명적이라 α를 높여(1종 오류를 늘려) 의심스러운 거래를 많이 차단합니다.

**시나리오 — 제약사 신약 승인:**
- **1종 오류:** 효과 없는 약을 승인 → 환자에게 위험
- **2종 오류:** 효과 있는 약을 승인 거부 → 환자가 치료 기회 상실

FDA는 1종 오류를 매우 엄격히 통제합니다(α = 0.01 또는 더 낮게).

#### α와 β의 트레이드오프 — 숫자로 이해하기

같은 표본 크기 n = 50에서:
- α = 0.10 → β ≈ 0.15 → 검정력 85%
- α = 0.05 → β ≈ 0.25 → 검정력 75%
- α = 0.01 → β ≈ 0.50 → 검정력 50%

α를 낮추면(1종 오류를 줄이면) β가 증가(2종 오류가 늘어남)합니다. 두 오류를 동시에 줄이려면 **표본 크기를 늘리는 것**이 유일한 방법.

**만약 n을 50에서 200으로 늘리면?**
- α = 0.05 → β ≈ 0.05 → 검정력 95%

표본이 4배 늘면 검정력이 75%에서 95%로 뛰어오릅니다.

### 검정력 (Power of a Test)

검정력 = 1 − β = H₀가 거짓일 때 올바르게 기각할 확률

검정력을 높이는 3가지 방법:
1. **표본 크기 증가** (가장 직접적)
2. **유의수준 α 증가** (1종 오류 위험 감수)
3. **효과 크기가 클 때** (큰 차이는 발견하기 쉬움)

### p-값

귀무가설이 참일 때 현재 검정 통계량만큼 극단적인 값을 얻을 확률 = 기각할 수 있는 최소 유의수준.

p-값 < α → 기각. p-값 ≥ α → 기각 실패.

#### p-값을 직관적으로 이해하기

**시나리오:** 동전을 100번 던져 62번 앞면. 공정한 동전(H₀)이라면?
- z = (62 − 50) / √(100 × 0.5 × 0.5) = 12/5 = 2.4
- p-값(양측) = 2 × P(Z > 2.4) = 2 × 0.0082 = 0.0164

"공정한 동전인데 우연히 이 정도 극단적 결과가 나올 확률이 1.64%"

- α = 5%: p = 0.0164 < 0.05 → **기각** (조작된 동전이라 결론)
- α = 1%: p = 0.0164 > 0.01 → **기각 실패** (아직 확실치 않음)

**같은 데이터, 다른 결론.** 유의수준 선택이 결론을 바꿀 수 있습니다. 이래서 α를 데이터 보기 **전에** 정해야 합니다.

**변수를 바꾸면?** 동전을 1,000번 던져 620번 앞면:
- z = (620 − 500) / √(1000 × 0.25) = 120/15.81 = 7.59
- p-값 ≈ 0 (사실상 0에 가까움)

같은 비율(62%)이지만 표본이 10배 → 증거가 압도적으로 강해집니다.

## LOS 8.b: 가설 검정 유형별 통계량

| 검정 대상 | 통계량 | 자유도 |
|-----------|--------|--------|
| 1개 모평균 | t | n-1 |
| 2개 모평균 차이 (독립) | t (pooled) | n₁+n₂-2 |
| 2개 모평균 차이 (대응) | t | n-1 |
| 1개 모분산 | χ² | n-1 |
| 2개 모분산 비교 | F | n₁-1, n₂-1 |

**왜 이렇게 많은 검정이 필요한가?** 각 상황의 데이터 구조가 다르기 때문입니다. 하나의 평균을 검정하는 것과, 두 그룹의 평균 차이를 검정하는 것, 분산을 검정하는 것은 모두 다른 수학적 구조입니다.

### 단일 모평균 검정 (t-test)

> **예제:** 콜옵션 포트폴리오 250일, 평균 0.1%, 표준편차 0.25%

**단계별 풀이:**

1. 가설: H₀: μ = 0 vs Hₐ: μ ≠ 0
2. 표준오차 계산: $s_{\bar{X}} = \frac{s}{\sqrt{n}} = \frac{0.0025}{\sqrt{250}} = 0.000158$
3. 검정 통계량: $t = \frac{\bar{X} - \mu_0}{s_{\bar{X}}} = \frac{0.001}{0.000158} = 6.33$
4. 판단: 6.33 > 1.96 → **H₀ 기각.** 평균 수익률은 0이 아님.

**왜 이런 결과가 나올까?** 250일이라는 큰 표본이 표준오차를 매우 작게 만들었습니다. 0.1%라는 작은 수익률이라도 250일 동안 일관되게 양수이면 "우연"이라고 보기 어렵습니다.

**만약 표본이 25일이었다면?**
- $s_{\bar{X}} = 0.0025/\sqrt{25} = 0.0005$
- $t = 0.001/0.0005 = 2.0$
- 2.0 > 1.96 → 여전히 기각이지만 **아슬아슬합니다.** 표본이 작으면 결론의 확신도 줄어듭니다.

**만약 표준편차가 0.5%(2배)였다면?** (n = 250)
- $s_{\bar{X}} = 0.005/\sqrt{250} = 0.000316$
- $t = 0.001/0.000316 = 3.16$
- 여전히 기각이지만, 변동성이 크면 같은 평균을 입증하기 더 어려워집니다.

### 독립 표본 t-검정

$$t = \frac{(\bar{X}_1 - \bar{X}_2)}{s_p\sqrt{1/n_1 + 1/n_2}}, \quad s_p^2 = \frac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1+n_2-2}$$

#### 수식 유도 — 왜 합동분산(pooled variance)인가?

두 모집단의 분산이 같다고 가정하면, 두 표본의 분산 정보를 합쳐서 더 정확한 분산 추정치를 얻을 수 있습니다. 합동분산은 두 표본 분산의 **가중 평균**입니다 (가중치 = 자유도).

**예제 — 수평적 합병 vs 수직적 합병의 초과수익률:**
- 수평적 합병: $\bar{X}_1$ = 1.0%, $s_1$ = 1.0%, $n_1$ = 61
- 수직적 합병: $\bar{X}_2$ = 2.5%, $s_2$ = 2.0%, $n_2$ = 61
- $s_p^2 = \frac{60(0.01)^2 + 60(0.02)^2}{120} = \frac{0.006 + 0.024}{120} = 0.00025$
- $t = \frac{0.01 - 0.025}{\sqrt{0.00025}\sqrt{1/61 + 1/61}} = \frac{-0.015}{0.01581 \times 0.1811} = \frac{-0.015}{0.002863} = -5.24$
- |t| = 5.24 > 1.98 (양측 5%, df=120) → **기각.** 두 유형의 초과수익률은 다릅니다.

### 대응 표본 검정 (Paired Comparisons)

동일 기업의 규제 전/후처럼 표본이 독립적이지 않을 때. 각 쌍의 차이를 먼저 계산하고 그 평균이 0인지 검정합니다.

**왜 독립 표본 검정을 쓰면 안 되는가?** 같은 기업의 전/후 데이터는 시장 위험, 산업 위험 등 공통 요인의 영향을 받습니다. 독립 표본으로 취급하면 이 공통 변동을 무시하고 표준오차를 과대 추정하여 검정력이 떨어집니다.

**예제 — 규제 해제 전후 베타 변화:**
- 통신 39개사의 규제 전/후 베타를 비교
- 각 기업의 차이 $d_i$ = (규제 후 베타) − (규제 전 베타)
- $\bar{d}$ = 평균 차이, $s_d$ = 차이의 표준편차
- $t = \frac{\bar{d}}{s_d / \sqrt{n}}$, df = n − 1 = 38

t = 10.26 > 2.024 (양측 5%, df=38) → **기각.** 규제 해제 후 베타가 유의미하게 변했습니다.

**왜 t가 이렇게 큰가?** 각 기업 내에서 전후 비교를 하면 기업 간 차이라는 "잡음"이 제거되어 신호가 더 선명해집니다.

### 카이제곱 검정 (분산)

$$\chi^2_{n-1} = \frac{(n-1)s^2}{\sigma_0^2}$$

**왜 이 형태인가?** $(n-1)s^2/\sigma^2$는 자유도 n-1인 카이제곱 분포를 따릅니다. H₀: σ² = σ₀² 하에서 이 통계량은 카이제곱(n-1)이 됩니다.

**예제 — 펀드 변동성 검증:**
High-Return 펀드의 과거 월 표준편차가 4%로 광고. 최근 24개월 표본에서 s = 3.8%.
- $\chi^2 = \frac{23 \times (0.038)^2}{(0.04)^2} = \frac{23 \times 0.001444}{0.0016} = 20.76$
- 임계값: χ²(23, 0.025) = 38.076, χ²(23, 0.975) = 11.689
- 11.689 < 20.76 < 38.076 → **기각 실패.** 변동성이 4%와 유의미하게 다르다고 볼 수 없음.

**카이제곱 분포의 특징:**
- 항상 양수 (분산은 음수가 될 수 없으므로)
- 오른쪽 꼬리 (비대칭)
- df가 커질수록 정규분포에 접근

### F-검정 (두 분산 비교)

$$F = \frac{s_1^2}{s_2^2}$$

큰 분산을 분자에 놓아 F ≥ 1이 되게 합니다. 이렇게 하면 오른쪽 임계값만 확인하면 됩니다.

**예제 — 섬유업 vs 제지업 수익 분산 비교:**
- 섬유: $s_1$ = 4.30, $n_1$ = 31
- 제지: $s_2$ = 3.80, $n_2$ = 41
- $F = (4.30)^2 / (3.80)^2 = 18.49 / 14.44 = 1.2805$
- 임계값: F(30, 40, 0.025) = 1.94
- 1.2805 < 1.94 → **기각 실패.** 두 산업의 수익 분산은 유의미하게 다르지 않음.

**만약 F = 2.5였다면?** 2.5 > 1.94 → 기각. 분산이 다르다는 결론.

**F-분포의 특징:**
- 항상 양수
- 두 개의 자유도(분자 df, 분모 df)로 정의
- F = 1이면 두 분산이 같음


## LOS 8.c: 모수 검정과 비모수 검정

| | 모수 검정 | 비모수 검정 |
|--|----------|-----------|
| 분포 가정 | 있음 (정규 등) | 없거나 적음 |
| 데이터 유형 | 수치, 정규 | 순위, 비정규 |
| 검정력 | 가정이 맞으면 더 높음 | 일반적으로 낮음 |
| 사용 예 | t, F, χ² | 런 검정, 순위 검정 |

### 왜 비모수 검정이 필요한가?

**시나리오 1 — 비정규 데이터:**
소형주 수익률은 극단적 값이 자주 나타나 정규분포 가정이 무리입니다. 표본 크기도 작다면(n < 30) 중심극한정리에 의존할 수도 없습니다. 이때 비모수 검정이 더 신뢰할 수 있습니다.

**시나리오 2 — 순위 데이터:**
펀드 성과를 1위~10위로 순위를 매겼을 때, "순위 간 상관관계"를 검정하려면 순위 데이터용 비모수 검정(스피어만 순위 상관)이 필요합니다.

**시나리오 3 — 무작위성 검정:**
주가가 "+, +, -, +, -, -, +"으로 변동합니다. 이 패턴이 무작위인지 아니면 추세가 있는지 검정하려면 **런 검정(runs test)**을 사용합니다. 이건 분포 모수와 관련 없는 가설입니다.

**비모수 검정의 단점:** 가정이 적은 대신, 같은 표본 크기에서 모수 검정보다 검정력이 낮습니다. 가정을 충족하면 모수 검정이 더 효율적입니다.

## 핵심 개념 정리 (Key Concepts)

### LOS 8.a
- H₀: 등호 포함, 기각하려는 가설. "채택"이 아닌 "기각 실패"
- 검정 통계량 = (표본 통계량 − 귀무가설 값) / 표준오차
- 1종 오류(α): 참인 H₀ 기각. 2종 오류(β): 거짓인 H₀ 기각 실패
- 검정력 = 1 − β. 표본 크기 증가 → 주어진 α에서 검정력 증가
- p-값 = 기각 가능한 최소 유의수준. p < α → 기각
- 통계적 유의성 ≠ 실질적(경제적) 유의성

### LOS 8.b
| 검정 | 통계량 | df |
|------|--------|-----|
| 1개 평균 | t | n-1 |
| 2개 평균(독립) | t (합동분산) | n₁+n₂-2 |
| 2개 평균(대응) | t (차이의 평균) | n-1 |
| 1개 분산 | χ² | n-1 |
| 2개 분산 | F (큰 분산이 분자) | n₁-1, n₂-1 |

- 독립 표본 → 차이 검정. 종속 표본 → 대응 비교 검정
- χ² 분포: 항상 양수, 비대칭, df 증가 시 정규 접근
- F 분포: 두 자유도로 정의, 항상 양수

### LOS 8.c
모수 검정: 분포 가정 필요 (t, F, χ²). 가정 충족 시 검정력 높음.
비모수 검정: 분포 가정 최소화. 순위 데이터, 비정규, 무작위성 검정 시 사용. 검정력은 상대적으로 낮음.
