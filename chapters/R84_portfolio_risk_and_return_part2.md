# Reading 84: 포트폴리오 위험과 수익률 Part II

---

## 왜 이 챕터가 존재하는가?

R83에서 "자산을 섞으면 위험이 줄어든다"는 것을 수식으로 증명했다. 이제 한 발 더 나간다.

만약 **모든 투자자가 같은 정보를 갖고 같은 기대**를 한다면? 모두가 같은 "최적 위험자산 포트폴리오"를 보유할 것이다. 그리고 그것은 **시장 전체** — 즉 **시장 포트폴리오**여야 한다.

그렇다면 개별 주식의 기대 수익률은 어떻게 결정될까? "그 주식이 시장 포트폴리오의 위험에 얼마나 기여하느냐"에 달려 있다. 이것이 CAPM이다.

---

## Module 84.1: 체계적 위험과 베타

### LOS 84.a: 무위험 자산과 위험자산 포트폴리오의 결합

R83 복습. 위험자산 포트폴리오(P)에 비중 $w_A$, 무위험 자산에 $1 - w_A$를 투자하면:

$$E(R_p) = w_A E(R_A) + (1 - w_A) R_f$$
$$\sigma_p = w_A \sigma_A$$

무위험 자산의 표준편차 = 0, 상관계수 = 0이므로 이렇게 단순해진다. 위험과 수익의 관계가 **직선**.

---

### LOS 84.b: 자본 배분선(CAL)과 자본 시장선(CML)

**동질적 기대:** 모든 투자자가 기대 수익률, 표준편차, 상관관계에 대해 **동일한 추정**을 갖는다고 가정.

이 가정 하에서 모든 투자자가 같은 효율적 프론티어를 직면하고, 같은 최적 위험 포트폴리오를 보유한다. 모든 사람이 같은 위험 포트폴리오를 보유하면, 그것은 **시장 포트폴리오**여야 한다.

이 특수한 CAL이 **자본 시장선(CML)**:

$$E(R_p) = R_f + \frac{E(R_M) - R_f}{\sigma_M} \cdot \sigma_p$$

기울기 $\frac{E(R_M) - R_f}{\sigma_M}$는 시장 위험 한 단위당 추가 기대 수익률이다.

숫자로 느껴보자. $R_f = 3\%$, $E(R_M) = 10\%$, $\sigma_M = 15\%$.

- 위험 0 → 수익 3% (무위험)
- 위험 7.5% (시장의 절반) → 수익 = 3% + (7/15) x 7.5% = 3% + 3.5% = **6.5%**
- 위험 15% (시장과 동일) → 수익 = **10%**
- 위험 30% (시장의 2배, 차입 투자) → 수익 = 3% + (7/15) x 30% = 3% + 14% = **17%**

CML 위에서 오른쪽으로 갈수록 위험과 수익이 비례하여 증가한다.

---

### LOS 84.c: 체계적 위험과 비체계적 위험

주식 10개를 포트폴리오에 넣었다. 20개로 늘렸다. 30개로 늘렸다.

- 10개: 표준편차 25%
- 20개: 표준편차 20%
- 30개: 표준편차 18%
- 50개: 표준편차 17.5%
- 100개: 표준편차 17%

어? 30개를 넘으면 위험이 거의 안 줄어든다. 왜?

제거할 수 있는 부분(**비체계적 위험**: 기업 고유 위험)은 이미 충분히 제거되었고, 남은 17%는 **체계적 위험**(시장 전체에 영향을 미치는 GDP 성장률, 금리 변동 등)이기 때문이다.

$$\text{총 위험} = \text{체계적 위험} + \text{비체계적 위험}$$

**핵심 결론:** 분산 투자는 (거의) 무비용이므로, 투자자는 무비용으로 제거할 수 있는 비체계적 위험에 대해 **보상받지 않는다**. 기대 수익률은 오직 **체계적 위험에만** 의존한다.

바이오텍 주식은 총 위험이 높지만 대부분 기업 고유(비체계적) 위험이다 → 체계적 위험은 작을 수 있다 → 기대 수익률도 낮을 수 있다. 반면 공작기계 제조업체는 총 위험은 낮지만 GDP에 민감(체계적 위험 높음) → 기대 수익률이 더 높을 수 있다.

---

### LOS 84.d: 수익률 생성 모형

**다요인 모형:**
$$E(R_i) - R_f = \beta_{i1} \times E(F_1) + \beta_{i2} \times E(F_2) + \cdots$$

Fama-French 3요인: 시장, 기업 규모, 장부가/시가. Carhart 4요인: + 가격 모멘텀.

**시장 모형 (단일 인덱스):**
$$R_i = \alpha_i + \beta_i R_m + e_i$$

---

### LOS 84.e: 베타의 계산과 해석

$$\beta_i = \frac{\text{Cov}(R_i, R_m)}{\sigma_m^2} = \rho_{im} \frac{\sigma_i}{\sigma_m}$$

공분산은 절대 크기라 해석이 어렵다. 시장 분산으로 나누면 "시장 1% 변할 때 이 자산이 몇 % 변하나"를 직접 알 수 있다.

- $\beta = 1$: 시장과 같은 민감도
- $\beta > 1$: 시장보다 공격적
- $\beta < 1$: 시장보다 방어적

### EXAMPLE: 베타 계산

시장 표준편차 20%.

**1.** 자산 A: $\sigma_A = 30\%$, $\rho = 0.8$
$$\beta_A = 0.80 \times \frac{0.30}{0.20} = 1.2$$

**2.** $\text{Cov}(A, M) = 0.048$
$$\beta_A = \frac{0.048}{0.04} = 1.2$$

## Module 84.2: CAPM과 SML

### LOS 84.f, 84.g: 자본 자산 가격 결정 모형 (CAPM)

$$\boxed{E(R_i) = R_f + \beta_i [E(R_M) - R_f]}$$

유도의 직관:

1. 비체계적 위험에 보상 없다 (분산으로 제거 가능)
2. 기대 수익률은 체계적 위험만의 함수
3. 체계적 위험의 측도 = $\beta$
4. $\beta = 0$ → $E(R) = R_f$. $\beta = 1$ → $E(R) = E(R_M)$. 선형이므로:

$$E(R_i) = R_f + \beta_i[E(R_M) - R_f]$$

### EXAMPLE: CAPM

$E(R_M) = 8\%$, $R_f = 2\%$, $\beta_A = 1.2$.

$$E(R_A) = 2\% + 1.2 \times (8\% - 2\%) = 2\% + 7.2\% = 9.2\%$$

---

### CML vs SML

| | CML | SML |
|---|---|---|
| x축 | 총 위험 ($\sigma$) | 체계적 위험 ($\beta$) |
| 위에 놓이는 것 | 효율적 포트폴리오만 | **모든** 적정 가격 증권과 포트폴리오 |

---

### LOS 84.h: CAPM의 응용 — 고평가/저평가 판단

무위험 7%, 시장 15%.

| 주식 | 현재가 | 1년 후 기대가 | 배당 | 베타 | 예측 수익률 | 요구 수익률 | 판단 |
|---|---|---|---|---|---|---|---|
| A | $25 | $27 | $1 | 1.0 | 12.0% | 15.0% | 고평가 (숏) |
| B | $40 | $45 | $2 | 0.8 | 17.5% | 13.4% | **저평가 (매수)** |
| C | $15 | $17 | $0.5 | 1.2 | 16.6% | 16.6% | 적정 |

예측 수익률 > 요구 수익률 → SML **위** → **저평가** → 매수
예측 수익률 < 요구 수익률 → SML **아래** → **고평가** → 숏

---

### LOS 84.i: 성과 측도

왜 네 가지가 필요한가? 포트폴리오 수익 15%가 좋은지 나쁜지는 **얼마나 위험을 감수했느냐**에 달렸다.

| 측도 | 공식 | 위험 기준 | 언제 사용? |
|---|---|---|---|
| **Sharpe** | $(R_p - R_f) / \sigma_p$ | 총 위험 | 단일 매니저 |
| **M²** | $R_f + SR \times \sigma_M$ | 총 위험 | Sharpe와 같은 순위, % 단위 |
| **Treynor** | $(R_p - R_f) / \beta_p$ | 체계적 위험 | 분산된 전체 포트폴리오 |
| **Jensen's alpha** | $R_p - [R_f + \beta_p(R_M - R_f)]$ | 체계적 위험 | 분산된 전체 포트폴리오 |

### EXAMPLE: M²

포트폴리오 P: 수익 10%, $\sigma = 20\%$. $R_f = 5\%$, $R_M = 11\%$, $\sigma_M = 30\%$.

- Sharpe = $(10 - 5)/20 = 0.25$
- $M^2 = 0.25 \times 30\% + 5\% = 12.5\%$
- M² alpha = $12.5\% - 11\% = 1.5\%$ → 시장 대비 1.5%p 우수

**1.** SML 위의 증권은 가치가 없다? → **틀림.** SML 위 = 균형 기대 수익률. (LOS 84.f)

**2.** $\beta = 1.2$, $R_f = 6\%$, $R_M = 12\%$. 기대 수익률? → $6 + 1.2 \times 6 = 13.2\%$. (LOS 84.g)

**3.** $\beta = 0.7$, $R_f = 7\%$, $R_M = 14\%$. 요구 수익률? → $7 + 0.7 \times 7 = 11.9\%$. (LOS 84.g)

**5.** $\beta = 0.7$, $R_f = 8\%$, $R_M = 15\%$, 현재 $50, 연말 $55 + $1. → 요구 12.9%, 예측 12% → **고평가**. (LOS 84.h)

**6.** 체계적 위험 한 단위당 초과 수익률은? → **Treynor 측도**. (LOS 84.i)

## Key Concepts

### LOS 84.a
무위험 자산과 위험자산을 결합하면 우월한 위험-수익 조합을 구성 가능.

### LOS 84.b
CAL: 위험자산 + 무위험의 직선. CML: 시장 포트폴리오를 위험자산으로 사용한 특수 CAL.

### LOS 84.c
총 위험 = 체계적(시장) + 비체계적(기업 고유). 비체계적 위험은 분산으로 제거 가능. 보상은 체계적 위험에만.

### LOS 84.d
수익률 생성 모형: 거시경제적, 기본적, 통계적 요인. 시장 모형: $R_i = \alpha + \beta R_m + e$.

### LOS 84.e
$\beta_i = \text{Cov}(R_i, R_m) / \sigma_m^2 = \rho_{im} \sigma_i / \sigma_m$. 시장 평균 베타 = 1.

### LOS 84.f, 84.g
CAPM: $E(R_i) = R_f + \beta_i[E(R_M) - R_f]$. SML은 기대 수익률 대 베타의 직선.

### LOS 84.h
예측 수익률 vs SML 요구 수익률 비교 → 고평가/저평가/적정 판단.

### LOS 84.i
Sharpe/M²: 총 위험 기반. Treynor/Jensen: 체계적 위험 기반.
