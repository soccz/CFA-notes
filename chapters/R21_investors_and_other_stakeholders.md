# Reading 21: Investors and Other Stakeholders

## LOS 21.a: 대출자와 주주의 청구권과 동기

### 왜 이걸 알아야 하는가?

회사에 1억을 투자할 때, 주식으로 살 것인가 채권으로 살 것인가? 이 선택에 따라 회사가 잘될 때와 망할 때 결과가 완전히 다르다.

### 당신이 친구와 치킨집을 같이 열었다고 합시다

- **친구 A:** 5,000만 원을 빌려줬다. 매달 이자 50만 원 받기로 했다 (채권자)
- **친구 B:** 5,000만 원을 출자했다. 이익이 나면 나눠 갖기로 했다 (주주)

**1년 후 치킨집이 대박이다. 순이익 3,000만 원.**
- A: 이자 600만 원 받음. 끝. 3,000만 원 중 남은 2,400만 원은 관심 없음
- B: 2,400만 원 전부 차지. "더 벌어라!"

**1년 후 치킨집이 망했다. 자산 매각하면 4,000만 원.**
- A: 5,000만 원 중 4,000만 원을 **먼저** 가져감 (1,000만 원 손해)
- B: A가 4,000만 원 가져간 뒤 남은 0원. 5,000만 원 전액 손실

이것이 핵심 차이다:

| | 채권자 (A) | 주주 (B) |
|---|---|---|
| 우선순위 | **높음** (먼저 받음) | 낮음 (나중에 받음) |
| 상승 잠재력 | **제한** (약속된 이자만) | **무한** (이익 전부) |
| 하방 위험 | 디폴트까지 안전 | 투자금 전액 위험 |

### 레버리지의 양면 — 숫자로 느껴보자

자산 1,000만 원짜리 사업. 수익 200만 원, 이자율 10%.

**시나리오 1: 100% 자기자본**
- 자기자본 1,000만, 이자 0, 순이익 200만
- ROE = 200/1,000 = **20%**

**시나리오 2: 50% 부채**
- 자기자본 500만, 부채 500만, 이자 50만, 순이익 150만
- ROE = 150/500 = **30%**

레버리지가 ROE를 올렸다. 그런데 수익이 100만으로 떨어지면?

**시나리오 1:** ROE = 100/1,000 = 10%
**시나리오 2:** 순이익 = 100 - 50 = 50만. ROE = 50/500 = **10%**

수익이 더 떨어져서 30만이면?
**시나리오 1:** ROE = 30/1,000 = 3%
**시나리오 2:** 순이익 = 30 - 50 = **-20만**. ROE = **-4%**

레버리지는 좋을 때는 더 좋게, 나쁠 때는 더 나쁘게 만든다.

### 왜 채권자와 주주는 근본적으로 충돌하는가?

수익 구조의 비대칭:
- **주주:** 위험한 프로젝트 선호. 성공하면 이익 독식, 실패해도 유한책임
- **채권자:** 안전한 프로젝트 선호. 성공해도 이자만 받고, 실패하면 원금 날림

주주가 채권자를 해치는 구체적 행동:
- 추가 부채 발행 → 기존 채권자의 파이 축소
- 과도한 배당 → 회사 현금 유출 → 채권자 담보 가치↓

채권자의 자기 보호: **약정(Covenants)** — "부채비율 3배 이상 올리지 마라", "배당을 순이익의 50% 이상 하지 마라" 등 사전 조건.

## LOS 21.b: 이해관계자 집단

### 주주 이론 vs 이해관계자 이론

| | 주주 이론 | 이해관계자 이론 |
|---|---|---|
| 초점 | 주주 이익 극대화 | 모든 이해관계자 균형 |
| 목표 | 주가 최대화 | 다양한 이해관계 조화 |

### 주요 이해관계자

| 이해관계자 | 주요 관심사 |
|---------|----------|
| 주주 | 수익, 성장, 주가 |
| 채권자 | 이자·원금 상환, 디폴트 방지 |
| 이사회 | 주주 이익 보호, 경영진 감시 |
| 경영진 | 고용, 보상, 기업 성과 |
| 직원 | 급여, 근무 환경, 안정성 |
| 고객 | 품질, 가격, 서비스 |
| 정부 | 세수, 법규 준수, 고용 |

### 이사회 구성

- **독립 이사:** 기업과 관계 없음 → 주주 이익 보호에 유리
- **내부 이사:** 임원, 창업자 → 이해충돌 가능

선출 방식:
- **전체 선출:** 매년 전원 재선 → 주주 영향력 높음
- **시차 이사회:** 매년 일부만 → 단기 전면 교체 불가 → 경영 연속성 vs 주주 영향력 약화

## LOS 21.c: ESG 요소

### 왜 ESG가 투자에 중요한가?

탄소 발전소에 투자했다고 하자. 지금은 돈을 잘 벌지만, 5년 뒤 탄소세가 도입되면? 발전소가 **좌초 자산(Stranded Asset)**이 된다 — 새 규제 때문에 경제적으로 쓸모없어진 자산.

### 3가지 ESG 요소

**환경 (E):**
- 기후 변화, 오염, 에너지 효율
- 물리적 위험: 기상이변으로 자산 피해
- 전환 위험: 저탄소 전환 요구로 사업 모델 변화

**사회 (S):**
- 개인정보 보호, 고객 만족
- 직원 다양성·포용성, 노사 관계

**거버넌스 (G):**
- 이사회 구성, 임원 보상
- 부패 방지, 정치 로비

ESG 위험 노출: 주주가 가장 크게 부담 (잔여 청구권자). 채권자는 디폴트를 유발할 정도가 아니면 제한적.

## Key Concepts

**LOS 21.a:**
- 채권자: 높은 우선순위, 상승 제한, 위험 회피
- 주주: 낮은 우선순위, 무한 상승, 위험 선호
- 레버리지: 좋을 때 더 좋게, 나쁠 때 더 나쁘게
- 채권자-주주 충돌 → 약정으로 해소

**LOS 21.b:**
- 이해관계자: 주주, 채권자, 이사회, 경영진, 직원, 고객, 정부
- 독립 이사 > 내부 이사 (주주 보호)

**LOS 21.c:**
- ESG: 환경(기후·좌초자산), 사회(개인정보·직원), 거버넌스(이사회·보상)
- 주주가 ESG 위험 최대 부담

## Quiz

**Q1.** 재무적으로 건전한 기업의 성장률이 높아지면 가장 혜택을 받는 집단은?

A. 주주  B. 채권자  C. 둘 다 아님

> **풀이:** 건전 = 이자 정상 지급. 채권자는 추가 이익 없음. 주주가 성장분 전부 누림.
>
> **정답: A**

---

**Q2.** 소수 주주가 선호하는 이사회 구조는?

A. 다수 독립 이사 + 시차 선출  B. 다수 독립 이사 + 전체 선출  C. 다수 내부 이사 + 시차 선출

> **풀이:** 독립 이사 → 주주 보호. 전체 선출 → 부실 이사회 즉시 교체 가능. 둘 다 소수 주주에게 유리.
>
> **정답: B**

---

**Q3.** 기업이 새 환경 규제 준수 대신 공장을 폐쇄. 이것은?

A. 거버넌스 요소  B. 사회 요소  C. 좌초 자산

> **풀이:** 환경 기준 변화로 자산이 경제적으로 쓸모없어짐 = 좌초 자산.
>
> **정답: C**
