# Reading 21: Investors and Other Stakeholders

---

## LOS 21.a: 대출자와 주주의 청구권과 동기

### 상황 속으로

당신이 친구와 치킨집을 같이 열었다고 합시다.

- **친구 A:** 5,000만 원을 빌려줬다. 매달 이자 50만 원 받기로 했다 (채권자)
- **친구 B:** 5,000만 원을 출자했다. 이익이 나면 나눠 갖기로 했다 (주주)

### 숫자를 직접 넣어보자 — 대박 vs 망했을 때

**시나리오 1: 치킨집 대박. 순이익 3,000만 원.**
- A (채권자): 이자 600만 원 받음. 끝. 3,000만 원 중 남은 2,400만 원은 관심 없음.
- B (주주): 2,400만 원 전부 차지. "더 벌어라!"

**시나리오 2: 치킨집 망함. 자산 매각하면 4,000만 원.**
- A (채권자): 5,000만 원 중 4,000만 원을 **먼저** 가져감 (1,000만 원 손해)
- B (주주): A가 4,000만 원 가져간 뒤 남은 0원. 5,000만 원 전액 손실.

**왜 이런 결과가 나올까?** 채권자는 "먼저 받는 권리"가 있지만 "추가로 받을 권리"가 없다. 주주는 "나중에 받지만" "남는 건 전부" 가져간다. 위험과 보상이 정확히 비례한다.

| | 채권자 (A) | 주주 (B) |
|---|---|---|
| 우선순위 | **높음** (먼저 받음) | 낮음 (나중에 받음) |
| 상승 잠재력 | **제한** (약속된 이자만) | **무한** (이익 전부) |
| 하방 위험 | 디폴트까지 안전 | 투자금 전액 위험 |

### 레버리지의 양면 — 변수를 바꿔가며 느끼기

자산 1,000만 원짜리 사업. 이자율 10%.

**수익 200만 원일 때:**

| | 100% 자기자본 | 50% 부채 |
|---|---|---|
| 자기자본 | 1,000만 | 500만 |
| 이자 | 0 | 50만 |
| 순이익 | 200만 | 150만 |
| ROE | **20%** | **30%** |

레버리지가 ROE를 올렸다!

**수익이 100만 원으로 떨어지면?**

| | 100% 자기자본 | 50% 부채 |
|---|---|---|
| 순이익 | 100만 | 50만 |
| ROE | **10%** | **10%** |

같다. 이 수준이 손익분기다.

**수익이 30만 원으로 더 떨어지면?**

| | 100% 자기자본 | 50% 부채 |
|---|---|---|
| 순이익 | 30만 | **-20만** |
| ROE | **3%** | **-4%** |

**왜 이런 결과가 나올까?** 레버리지는 좋을 때는 더 좋게, 나쁠 때는 더 나쁘게 만든다. 부채의 이자(50만)는 고정이므로, 수익이 이자보다 높으면 남는 게 주주 몫이 되고, 이자보다 낮으면 주주가 손해를 본다.

**만약 이자율이 5%로 낮아지면?** 이자가 25만으로 줄어든다. 수익 30만일 때 순이익 = 5만, ROE = 1%. 여전히 양수다. 이자율이 레버리지 위험의 크기를 결정한다.

### 채권자와 주주의 근본적 충돌

수익 구조의 비대칭 때문이다:
- **주주:** 위험한 프로젝트 선호. 성공하면 이익 독식, 실패해도 유한책임
- **채권자:** 안전한 프로젝트 선호. 성공해도 이자만 받고, 실패하면 원금 날림

주주가 채권자를 해치는 구체적 행동:
- 추가 부채 발행 → 기존 채권자의 파이 축소
- 과도한 배당 → 회사 현금 유출 → 채권자 담보 가치 하락

채권자의 자기 보호: **약정(Covenants)** — "부채비율 3배 이상 올리지 마라", "배당을 순이익의 50% 이상 하지 마라" 등 사전 조건.

## LOS 21.b: 이해관계자 집단

### 상황 속으로

당신이 대기업 CEO라고 합시다. 이번 분기 실적이 나쁘다.

- **주주:** "비용 줄여! 직원 해고해!"
- **직원:** "해고하지 마! 가족이 있어!"
- **고객:** "품질 떨어뜨리지 마!"
- **정부:** "세금 제대로 내고 고용 유지해!"

**왜 이런 결과가 나올까?** 각 집단이 회사에서 원하는 것이 다르기 때문이다. 주주는 수익, 직원은 안정, 고객은 품질, 정부는 세수. 모두를 만족시키는 건 불가능하다.

| | 주주 이론 | 이해관계자 이론 |
|---|---|---|
| 초점 | 주주 이익 극대화 | 모든 이해관계자 균형 |
| 목표 | 주가 최대화 | 다양한 이해관계 조화 |

### 주요 이해관계자

| 이해관계자 | 주요 관심사 |
|---------|----------|
| 주주 | 수익, 성장, 주가 |
| 채권자 | 이자·원금 상환, 디폴트 방지 |
| 이사회 | 주주 이익 보호, 경영진 감시 |
| 경영진 | 고용, 보상, 기업 성과 |
| 직원 | 급여, 근무 환경, 안정성 |
| 고객 | 품질, 가격, 서비스 |
| 정부 | 세수, 법규 준수, 고용 |

### 이사회 구성 — 숫자로 비교

두 회사를 비교하자:

**회사 A:** 이사 10명 중 독립 이사 8명, 내부 이사 2명. 전체 선출 방식.
**회사 B:** 이사 10명 중 독립 이사 3명, 내부 이사 7명. 시차 이사회(매년 3명만 교체).

당신이 소수 주주로서 이사회가 마음에 안 들어 교체하고 싶다면?

- **회사 A:** 올해 주총에서 독립 이사 위주로 전원 교체 가능. 주주 영향력 높음.
- **회사 B:** 올해 3명만 바꿀 수 있고, 내부 이사가 다수라 경영진 감시 약함. 전면 교체에 최소 4년.

**왜 이런 결과가 나올까?**

- **독립 이사:** 기업과 재무적 관계 없음 → 주주 이익 보호에 유리
- **내부 이사:** 임원, 창업자 → 이해충돌 가능
- **전체 선출:** 매년 전원 재선 → 주주 영향력 높음
- **시차 이사회:** 매년 일부만 → 경영 연속성 vs 주주 영향력 약화

## LOS 21.c: ESG 요소

### 상황 속으로

당신이 석탄 발전소에 투자했다고 합시다. 지금은 연 15% 수익을 올린다. 훌륭해 보인다.

하지만 두 시나리오를 비교하자:

**5년 후 시나리오 1: 탄소세 없음**
- 수익률 15% 유지. 투자금 1억 → 5년 후 약 2억.

**5년 후 시나리오 2: 탄소세 도입 (톤당 5만 원)**
- 운영비 급증 → 수익률 -5%. 발전소를 팔려 해도 아무도 안 산다.
- 투자금 1억 → **좌초 자산(Stranded Asset)**. 새 규제 때문에 경제적으로 쓸모없어진 자산.

**왜 이런 결과가 나올까?** 재무제표에 보이지 않는 위험이 있다. ESG 분석은 이런 숨은 위험을 사전에 식별하는 도구다.

### 3가지 ESG 요소

**환경 (E):**
- 기후 변화, 오염, 에너지 효율
- 물리적 위험: 기상이변으로 자산 피해
- 전환 위험: 저탄소 전환 요구로 사업 모델 변화

**사회 (S):**
- 개인정보 보호, 고객 만족
- 직원 다양성·포용성, 노사 관계

**거버넌스 (G):**
- 이사회 구성, 임원 보상
- 부패 방지, 정치 로비

**만약 ESG 위험이 현실화되면 누가 가장 손해인가?** 주주다. 채권자는 디폴트를 유발할 정도가 아니면 이자를 받는다. 주주는 잔여 청구권자이므로 가치 하락을 전부 떠안는다.

## Key Concepts

**LOS 21.a:**
- 채권자: 높은 우선순위, 상승 제한, 위험 회피
- 주주: 낮은 우선순위, 무한 상승, 위험 선호
- 레버리지: 좋을 때 더 좋게, 나쁠 때 더 나쁘게
- 채권자-주주 충돌 → 약정으로 해소

**LOS 21.b:**
- 이해관계자: 주주, 채권자, 이사회, 경영진, 직원, 고객, 정부
- 독립 이사 > 내부 이사 (주주 보호)

**LOS 21.c:**
- ESG: 환경(기후·좌초자산), 사회(개인정보·직원), 거버넌스(이사회·보상)
- 주주가 ESG 위험 최대 부담

## Quiz

**Q1.** 재무적으로 건전한 기업의 성장률이 높아지면 가장 혜택을 받는 집단은?

A. 주주  B. 채권자  C. 둘 다 아님

> **풀이:** 건전 = 이자 정상 지급. 채권자는 추가 이익 없음. 주주가 성장분 전부 누림.
>
> **정답: A**

---

**Q2.** 소수 주주가 선호하는 이사회 구조는?

A. 다수 독립 이사 + 시차 선출  B. 다수 독립 이사 + 전체 선출  C. 다수 내부 이사 + 시차 선출

> **풀이:** 독립 이사 → 주주 보호. 전체 선출 → 부실 이사회 즉시 교체 가능. 둘 다 소수 주주에게 유리.
>
> **정답: B**

---

**Q3.** 기업이 새 환경 규제 준수 대신 공장을 폐쇄. 이것은?

A. 거버넌스 요소  B. 사회 요소  C. 좌초 자산

> **풀이:** 환경 기준 변화로 자산이 경제적으로 쓸모없어짐 = 좌초 자산.
>
> **정답: C**
