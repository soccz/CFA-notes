## LOS 5.a: 포트폴리오 기대 수익률과 분산

### 왜 포트폴리오 수학이 필요한가?

"달걀을 한 바구니에 담지 마라"는 격언이 있습니다. 한 바구니가 떨어져도 나머지가 무사하니까요. 하지만 모든 바구니를 같은 트럭에 실으면 — 모든 자산이 같이 움직이면 — 분산 효과가 없습니다. 포트폴리오 수학은 "얼마나 다르게 움직이는 자산을 조합하면 얼마나 안전해지는가?"를 정확히 계산합니다.

### 포트폴리오 기대 수익률

$$E(R_p) = \sum_{i=1}^{n} w_i E(R_i)$$

단순 가중 평균입니다. 기댓값 연산자는 선형이므로 공분산 항이 필요 없습니다.

### 2자산 포트폴리오 분산: 왜 공분산이 들어가는가?

두 자산 A, B의 포트폴리오 수익률은 $R_p = w_A R_A + w_B R_B$입니다. 분산의 정의를 전개하면:

$(a+b)^2 = a^2 + b^2 + 2ab$ 패턴으로:

$$\sigma_p^2 = w_A^2 \sigma_A^2 + w_B^2 \sigma_B^2 + 2w_A w_B \text{Cov}_{AB}$$

상관계수로 대체하면($\text{Cov}_{AB} = \rho_{AB} \sigma_A \sigma_B$):

$$\sigma_p^2 = w_A^2 \sigma_A^2 + w_B^2 \sigma_B^2 + 2w_A w_B \rho_{AB} \sigma_A \sigma_B$$

**공분산 항이 핵심입니다.** 이 항이 작을수록(음수일수록) 포트폴리오 분산이 줄어듭니다.

| 상관계수 | 포트폴리오 표준편차 | 분산투자 효과 |
|---------|-------------------|-------------|
| $\rho = +1$ | $w_A\sigma_A + w_B\sigma_B$ | **없음** |
| $-1 < \rho < +1$ | 가중 합보다 작음 | **있음** |
| $\rho = -1$ | $|w_A\sigma_A - w_B\sigma_B|$ | **완전 헤지 가능** |

**$\rho = -1$이면 분산을 0으로 만들 수 있습니다.** 조건: $w_A\sigma_A = w_B\sigma_B$. 현실에서 $\rho = -1$인 두 자산은 거의 존재하지 않습니다.

### 3자산 포트폴리오 분산

$$\sigma_p^2 = w_A^2 \sigma_A^2 + w_B^2 \sigma_B^2 + w_C^2 \sigma_C^2 + 2w_Aw_B\text{Cov}_{AB} + 2w_Aw_C\text{Cov}_{AC} + 2w_Bw_C\text{Cov}_{BC}$$

n자산이면 분산 항 n개 + 공분산 항 $n(n-1)/2$개. 자산이 많아질수록 **공분산 항이 지배적**이 됩니다. 포트폴리오 위험의 대부분은 자산 간 공분산에 의해 결정됩니다.

**예제:** 국내 주식(60%), 국내 채권(30%), 해외 주식(10%), 공분산 행렬:

| | 국내 주식 | 국내 채권 | 해외 주식 |
|--|-----------|-----------|-----------|
| **국내 주식** | 400 | 44 | 180 |
| **국내 채권** | 44 | 70 | 35 |
| **해외 주식** | 180 | 35 | 450 |

$\sigma_p^2 = 144 + 6.3 + 4.5 + 15.84 + 21.6 + 2.1 = 194.34$, $\sigma_p = 13.94\%$

국내 주식만이면 $\sigma = 20\%$이지만, 분산투자로 13.94%로 줄었습니다.

## LOS 5.b: 결합 확률 함수로 공분산 계산

$$\text{Cov}(R_A, R_B) = \sum P(s) \cdot [R_A(s) - E(R_A)] \cdot [R_B(s) - E(R_B)]$$

**예제:**

| 확률 | $R_A$ | $R_B$ |
|------|--------|--------|
| 0.3 | 20% | 30% |
| 0.5 | 12% | 10% |
| 0.2 | 5% | 0% |

$E(R_A) = 13\%$, $E(R_B) = 14\%$
$\text{Cov} = 0.00336 + 0.00020 + 0.00224 = 0.0058$ (양수 → 같은 방향)

## LOS 5.c: 손실 부족 위험과 안전 제일 기준

### 왜 이 개념이 필요한가?

다리를 건설할 때 "평균 하중"만 보면 안 됩니다. "최악의 경우에도 무너지지 않는가?"가 중요합니다. 연금 기금은 "평균 수익이 좋다"보다 "최소 3% 수익은 반드시 확보해야 수혜자에게 지급 가능하다"가 더 중요합니다.

**Roy의 안전 제일 기준:** 수익률이 최소 허용 수준 $R_L$ 아래로 떨어질 확률을 최소화

$$\text{SFR} = \frac{E(R_p) - R_L}{\sigma_p}$$

SFR은 기대 수익률이 목표 수익률보다 **몇 표준편차 위에 있는지**를 나타냅니다. SFR이 클수록 좋습니다.

**만약 $R_L = R_f$(무위험이자율)라면** SFR은 정확히 **샤프 비율**이 됩니다.

**예제:** 목표 $R_L = 3\%$

| | 포트폴리오 A | 포트폴리오 B | 포트폴리오 C |
|--|-------------|-------------|-------------|
| $E(R_p)$ | 9% | 11% | 6.6% |
| $\sigma_p$ | 12% | 20% | 8.2% |
| **SFR** | **0.50** | 0.40 | 0.44 |

**최선: 포트폴리오 A** (SFR 0.50). B는 기대 수익률이 가장 높지만(11%) 변동성이 너무 높아(20%) 목표 미달 위험이 큽니다. 기대 수익률만 보면 안 됩니다.

손실 부족 확률: $P(R_A < 3\%) = \Phi(-0.50) = 30.85\%$

## Module Quiz 5.1

**Q1.** 상관계수 0.50, 공분산 0.0043, B의 표준편차 26%. A의 분산은?

$$\sigma_A = \frac{0.0043}{0.26 \times 0.50} = 0.03308, \quad \text{Var} = 0.001094 \approx 0.0011$$

**정답: A. 0.0011**

**Q2.** 결합 확률 표에서 P와 Q의 공분산은?

| | Q=7% | Q=4% | Q=0% |
|--|------|------|------|
| P=15% | 0.2 | 0 | 0 |
| P=12% | 0 | 0.2 | 0 |
| P=0% | 0 | 0 | 0.6 |

$E(P)=5.4\%$, $E(Q)=2.2\%$
$\text{Cov} = 0.2(9.6)(4.8) + 0.2(6.6)(1.8) + 0.6(-5.4)(-2.2) = 9.216+2.376+7.128 = 18.72$

**정답: B. 18.7**

**Q3.** 목표 4%에서 Roy의 안전 제일 기준 최적 포트폴리오는?

$\text{SFR}_A=1/8=0.125$, $\text{SFR}_B=7/21=0.333$, $\text{SFR}_C=14/40=0.350$

**정답: C. Portfolio C**

## 핵심 개념 정리 (Key Concepts)

### LOS 5.a
$$E(R_p) = \sum w_i E(R_i)$$
$$\sigma_p^2 = w_A^2\sigma_A^2 + w_B^2\sigma_B^2 + 2w_Aw_B\rho_{AB}\sigma_A\sigma_B$$
공분산(상관계수)이 낮을수록 포트폴리오 분산 감소

### LOS 5.b
$$\text{Cov}(R_A, R_B) = \sum P(s)[R_A(s)-E(R_A)][R_B(s)-E(R_B)]$$

### LOS 5.c
$$\text{SFR} = \frac{E(R_p) - R_L}{\sigma_p}$$
SFR 최대 포트폴리오 선택. $R_L = R_f$이면 SFR = 샤프 비율
