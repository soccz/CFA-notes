# Reading 5: Portfolio Mathematics

> **핵심 질문:** 여러 자산을 섞어 포트폴리오를 만들면 수익률과 위험은 어떻게 계산되는가? 그리고 가장 안전한 포트폴리오를 어떻게 선택하는가?

---

## 왜 포트폴리오 수학이 필요한가?

> **[Level 1 — 비유]** "달걀을 한 바구니에 담지 마라"는 격언이 있습니다. 왜 여러 바구니에 나누면 안전할까요? 한 바구니가 떨어져도 나머지가 무사하기 때문입니다. 하지만 모든 바구니를 같은 트럭에 실으면(= 모든 자산이 같이 움직이면) 분산 효과가 없습니다. 포트폴리오 수학은 "얼마나 다르게 움직이는 바구니를 조합하면 얼마나 안전해지는가?"를 정확히 계산합니다.

> **[Level 2 — 개념]** 투자자는 보통 한 가지 자산만 보유하지 않습니다. 주식, 채권, 해외 자산을 조합한 **포트폴리오**를 구성합니다. 이때 두 가지 핵심 질문이 생깁니다:
>
> 1. **기대 수익률**: 포트폴리오 전체의 평균 수익률은? → 단순 가중 평균
> 2. **위험(분산)**: 포트폴리오 전체의 변동성은? → **공분산 때문에 단순 합이 아님**
>
> 두 번째 질문이 훨씬 복잡하고 중요합니다.

> **[개념 연결 ← R03, R04]** R03에서 공분산과 상관계수를 배웠고, R04에서 확률 가중 기댓값과 분산을 배웠습니다. 이 챕터에서는 그 두 가지를 결합하여 **포트폴리오** 수준의 수익-위험을 계산합니다.

## LOS 5.a: 포트폴리오 기대 수익률과 분산

### 포트폴리오 기대 수익률

n개 자산으로 구성된 포트폴리오의 기대 수익률:

$$E(R_p) = \sum_{i=1}^{n} w_i E(R_i) = w_1 E(R_1) + w_2 E(R_2) + \cdots + w_n E(R_n)$$

**포트폴리오 비중(weight) 정의:**

$$w_i = \frac{\text{자산 } i \text{에 투자된 시장가치}}{\text{포트폴리오 전체 시장가치}}$$

단, $\sum w_i = 1$ (모든 비중의 합 = 1)

**이것은 단순 가중 평균입니다.** 분산과 달리 공분산 항이 필요 없습니다. 왜냐하면 기댓값 연산자 $E(\cdot)$는 선형이기 때문입니다: $E(aX + bY) = aE(X) + bE(Y)$.

---

### 공분산 (Covariance)

**왜 공분산이 포트폴리오에서 핵심인가?**

> **[Level 2 — 개념]** 기대 수익률은 각 자산의 가중 평균으로 단순하게 결정됩니다. 하지만 **분산(위험)은 자산들이 어떻게 함께 움직이는지**에 따라 달라집니다. 이 "함께 움직임"을 측정하는 것이 공분산입니다.

$$\text{Cov}(R_i, R_j) = E\{[R_i - E(R_i)][R_j - E(R_j)]\}$$

**직관적 해석:**
- **양(+)의 공분산**: 한 자산이 평균 위에 있을 때, 다른 자산도 평균 위에 있는 경향
- **음(-)의 공분산**: 한 자산이 평균 위에 있을 때, 다른 자산은 평균 아래에 있는 경향
- **0**: 두 자산의 움직임이 무관

**중요한 성질:**
- $\text{Cov}(R_A, R_A) = \text{Var}(R_A)$ — 자기 자신과의 공분산 = 분산
- $\text{Cov}(R_A, R_B) = \text{Cov}(R_B, R_A)$ — 순서 무관 (대칭)

---

### 표본 공분산 (Sample Covariance)

과거 수익률 데이터로 공분산을 추정할 때:

$$s_{1,2} = \frac{\sum_{t=1}^{n}(R_{1t} - \bar{R}_1)(R_{2t} - \bar{R}_2)}{n-1}$$

(분모 n-1: R03에서 배운 자유도 보정과 같은 이유)

---

### 공분산 행렬 (Covariance Matrix)

3개 자산 A, B, C의 공분산 행렬:

|      | A | B | C |
|------|---|---|---|
| **A** | Cov(R_A, R_A) | Cov(R_A, R_B) | Cov(R_A, R_C) |
| **B** | Cov(R_B, R_A) | Cov(R_B, R_B) | Cov(R_B, R_C) |
| **C** | Cov(R_C, R_A) | Cov(R_C, R_B) | Cov(R_C, R_C) |

**핵심 포인트:**
- **대각선**: 각 자산의 분산 (Cov(R_i, R_i) = Var(R_i))
- **비대각선**: 자산 간 공분산 (대칭이므로 절반만 독립)
- n개 자산의 경우: n개의 분산 + n(n-1)/2개의 고유한 공분산

> **[Level 3 — "만약~라면?"]** n이 커지면 고유한 공분산의 수가 급증합니다. 10개 자산이면 45개, 100개 자산이면 4,950개의 공분산을 추정해야 합니다. 이것이 실무에서 팩터 모형(factor model)이 필요한 이유입니다 — 수천 개의 공분산을 직접 추정하는 대신, 소수의 공통 요인으로 공분산 구조를 근사합니다.

---

### 2자산 포트폴리오 분산

**[Level 3 — 수식 유도: 왜 공분산이 포트폴리오 분산에 들어가는가?]**

> 두 자산 A, B로 구성된 포트폴리오의 수익률:
> $$R_p = w_A R_A + w_B R_B$$
>
> 분산의 정의를 전개합니다:
> $$\text{Var}(R_p) = E[(R_p - E(R_p))^2]$$
>
> $R_p - E(R_p) = w_A(R_A - E(R_A)) + w_B(R_B - E(R_B))$를 대입하면:
>
> $$\text{Var}(R_p) = E[w_A(R_A - E(R_A)) + w_B(R_B - E(R_B))]^2$$
>
> $(a+b)^2 = a^2 + b^2 + 2ab$ 전개:
>
> $$= w_A^2 E[(R_A - E(R_A))^2] + w_B^2 E[(R_B - E(R_B))^2] + 2w_A w_B E[(R_A - E(R_A))(R_B - E(R_B))]$$
>
> $$= w_A^2 \text{Var}(R_A) + w_B^2 \text{Var}(R_B) + 2w_A w_B \text{Cov}(R_A, R_B)$$

$$\boxed{\sigma_p^2 = w_A^2 \sigma_A^2 + w_B^2 \sigma_B^2 + 2w_A w_B \text{Cov}_{AB}}$$

상관계수로 대체하면 ($\text{Cov}_{AB} = \rho_{AB} \sigma_A \sigma_B$):
$$\sigma_p^2 = w_A^2 \sigma_A^2 + w_B^2 \sigma_B^2 + 2w_A w_B \rho_{AB} \sigma_A \sigma_B$$

**핵심 인사이트: 분산 감소 효과 (Diversification)**

> **[Level 2 — 개념]** 공분산 항($2w_Aw_B\text{Cov}_{AB}$)이 핵심입니다. 이 항이 작을수록(심지어 음수일수록) 포트폴리오 분산이 줄어듭니다. 이것이 **분산투자의 수학적 근거**입니다.

| 상관계수 | 결과 | 분산투자 효과 |
|---------|------|-------------|
| $\rho_{AB} = +1$ | $\sigma_p = w_A\sigma_A + w_B\sigma_B$ | **없음** (단순 가중 평균) |
| $-1 < \rho_{AB} < +1$ | $\sigma_p < w_A\sigma_A + w_B\sigma_B$ | **있음** |
| $\rho_{AB} = 0$ | $\sigma_p = \sqrt{w_A^2\sigma_A^2 + w_B^2\sigma_B^2}$ | **부분 분산** |
| $\rho_{AB} = -1$ | $\sigma_p = |w_A\sigma_A - w_B\sigma_B|$ | **완전 헤지 가능** |

> **[Level 3 — "만약~라면?"]** $\rho = -1$이면 분산을 0으로 만들 수 있습니다. 조건: $w_A\sigma_A = w_B\sigma_B$, 즉 $w_A/w_B = \sigma_B/\sigma_A$. 변동성이 큰 자산에 적은 비중을 두고, 변동성이 작은 자산에 많은 비중을 두면 됩니다. 하지만 현실에서 $\rho = -1$인 두 자산은 거의 존재하지 않습니다.

---

### 3자산 포트폴리오 분산

$$\sigma_p^2 = w_A^2 \sigma_A^2 + w_B^2 \sigma_B^2 + w_C^2 \sigma_C^2 + 2w_A w_B \text{Cov}_{AB} + 2w_A w_C \text{Cov}_{AC} + 2w_B w_C \text{Cov}_{BC}$$

> **패턴:** n자산이면 분산 항 n개 + 공분산 항 $n(n-1)/2$개. 자산 수가 많아질수록 **공분산 항이 분산 항보다 훨씬 많아집니다.** 결국 포트폴리오 위험의 대부분은 개별 자산의 분산이 아니라 자산 간 공분산에 의해 결정됩니다.

---

### 예제: 3자산 포트폴리오 분산 계산

포트폴리오 구성: 국내 주식(60%), 국내 채권(30%), 해외 주식(10%)

공분산 행렬 (단위: %²):

| | 국내 주식 | 국내 채권 | 해외 주식 |
|--|-----------|-----------|-----------|
| **국내 주식** | 400 | 44 | 180 |
| **국내 채권** | 44 | 70 | 35 |
| **해외 주식** | 180 | 35 | 450 |

**포트폴리오 분산 계산:**

$$\sigma_p^2 = (0.6)^2(400) + (0.3)^2(70) + (0.1)^2(450)$$
$$+ 2(0.6)(0.3)(44) + 2(0.6)(0.1)(180) + 2(0.3)(0.1)(35)$$
$$= 144 + 6.3 + 4.5 + 15.84 + 21.6 + 2.1 = 194.34 \text{ %}^2$$

**포트폴리오 표준편차:**
$$\sigma_p = \sqrt{194.34} = 13.94\%$$

> **해석:** 국내 주식만 보유하면 $\sigma = \sqrt{400} = 20\%$이지만, 채권과 해외 주식을 추가하여 13.94%로 위험이 줄었습니다. 이것이 분산투자 효과입니다.

---

### 상관계수 행렬 (Correlation Matrix)

공분산 대신 상관계수를 이용할 수도 있습니다 ($\text{Cov}_{AB} = \rho_{AB} \sigma_A \sigma_B$):

| | 국내 주식 | 국내 채권 | 해외 주식 |
|--|-----------|-----------|-----------|
| **국내 주식** | 1.000 | 0.263 | 0.424 |
| **국내 채권** | 0.263 | 1.000 | 0.197 |
| **해외 주식** | 0.424 | 0.197 | 1.000 |

**대각선은 항상 1** — 자산과 자신의 상관계수는 완전 정(+)의 상관.

> **[개념 연결 → R03]** 이 상관계수 행렬에서 0.263, 0.424, 0.197은 모두 1보다 훨씬 작습니다. R03에서 배운 "상관계수가 1보다 작으면 분산투자 효과가 있다"는 원리가 여기서 적용됩니다.

## LOS 5.b: 결합 확률 함수로 공분산 계산

**결합 확률 함수(Joint Probability Function)**: 두 자산이 특정 수익률을 동시에 달성할 확률을 나타냅니다.

> **[개념 연결 ← R04]** R04에서 확률 가중 기댓값과 분산을 배웠습니다. 공분산도 동일한 방법으로 확률 가중하여 계산합니다.

### 예제: 결합 확률 함수에서 공분산 계산

경제 상황별 자산 A, B의 수익률:

| | $R_B = 30\%$ | $R_B = 10\%$ | $R_B = 0\%$ |
|--|---|---|---|
| $R_A = 20\%$ | 0.30 | 0 | 0 |
| $R_A = 12\%$ | 0 | 0.50 | 0 |
| $R_A = 5\%$ | 0 | 0 | 0.20 |

> **해석:** 확률이 대각선에만 있으므로, A와 B는 항상 같은 시나리오에서 움직입니다 → 강한 양의 공분산 예상

**1단계: 기댓값 계산**
$$E(R_A) = (0.3)(0.20) + (0.5)(0.12) + (0.2)(0.05) = 0.13 = 13\%$$
$$E(R_B) = (0.3)(0.30) + (0.5)(0.10) + (0.2)(0.00) = 0.14 = 14\%$$

**2단계: 공분산 계산**

$$\text{Cov}(R_A, R_B) = \sum P(s) \cdot [R_A(s) - E(R_A)] \cdot [R_B(s) - E(R_B)]$$

| 확률 | $R_A$ | $R_B$ | 계산 |
|------|--------|--------|------|
| 0.3 | 0.20 | 0.30 | $(0.3)(0.20-0.13)(0.30-0.14) = 0.00336$ |
| 0.5 | 0.12 | 0.10 | $(0.5)(0.12-0.13)(0.10-0.14) = 0.00020$ |
| 0.2 | 0.05 | 0.00 | $(0.2)(0.05-0.13)(0.00-0.14) = 0.00224$ |

$$\text{Cov}(R_A, R_B) = 0.00336 + 0.00020 + 0.00224 = 0.0058$$

> 공분산이 양수 → A와 B는 같은 방향으로 움직이는 경향. 이 두 자산으로 포트폴리오를 구성하면 분산투자 효과가 제한적입니다.

## LOS 5.c: 손실 부족 위험과 안전 제일 기준

### 손실 부족 위험 (Shortfall Risk)

**왜 이 개념이 필요한가?**

> **[Level 1 — 비유]** 다리를 건설할 때 "평균 하중"만 보면 안 됩니다. "최악의 경우에도 무너지지 않는가?"가 중요합니다. 투자에서도 마찬가지입니다. 연금 기금은 "평균적으로 수익이 좋다"보다 "최소 3% 수익은 반드시 확보해야 수혜자에게 지급할 수 있다"가 더 중요합니다.

**정의:** 포트폴리오 가치 또는 수익률이 특정 목표 수준 아래로 떨어질 확률.

---

### Roy의 안전 제일 기준 (Roy's Safety-First Criterion)

**목표:** 수익률이 최소 허용 수준 $R_L$ 아래로 떨어질 확률 최소화

$$\text{최소화}: P(R_p < R_L)$$

**[Level 3 — 수식 유도: SFR의 유도]**

> **수익률이 정규분포를 따를 때**, $P(R_p < R_L)$을 표준정규분포로 변환합니다:
>
> $$P(R_p < R_L) = P\left(\frac{R_p - E(R_p)}{\sigma_p} < \frac{R_L - E(R_p)}{\sigma_p}\right) = \Phi\left(\frac{R_L - E(R_p)}{\sigma_p}\right)$$
>
> $\frac{R_L - E(R_p)}{\sigma_p}$는 음수(목표가 기댓값보다 낮으면)이므로, 이를 $-\text{SFR}$로 정의합니다:
>
> $$\text{SFR} = \frac{E(R_p) - R_L}{\sigma_p}$$
>
> $$P(R_p < R_L) = \Phi(-\text{SFR})$$
>
> $\Phi$는 단조 증가 함수이므로, SFR이 클수록 $\Phi(-\text{SFR})$가 작아집니다.

$$\boxed{\text{최대화}: \text{SFR} = \frac{E(R_p) - R_L}{\sigma_p}}$$

**직관:** SFR은 기대 수익률이 목표 수익률보다 **몇 표준편차 위에 있는지**를 나타냅니다. 이 거리가 클수록 목표 미달 가능성이 낮습니다.

> **[개념 연결 → 샤프 비율]** SFR의 형태를 보면: $(E(R_p) - R_L) / \sigma_p$. 만약 $R_L = R_f$(무위험이자율)라면 이것은 정확히 **샤프 비율(Sharpe Ratio)** 입니다! 안전 제일 기준은 샤프 비율의 일반화된 버전으로 볼 수 있습니다.

---

### 예제: Roy의 안전 제일 기준

**상황:** $120백만 대학 기금이 연말 최소 $123.6백만을 목표로 합니다.

목표 수익률: $R_L = \frac{123.6 - 120}{120} = 0.030 = 3\%$

세 포트폴리오 비교:

| | 포트폴리오 A | 포트폴리오 B | 포트폴리오 C |
|--|-------------|-------------|-------------|
| $E(R_p)$ | 9% | 11% | 6.6% |
| $\sigma_p$ | 12% | 20% | 8.2% |

안전 제일 비율 계산:

$$\text{SFR}_A = \frac{9 - 3}{12} = \frac{6}{12} = 0.50$$
$$\text{SFR}_B = \frac{11 - 3}{20} = \frac{8}{20} = 0.40$$
$$\text{SFR}_C = \frac{6.6 - 3}{8.2} = \frac{3.6}{8.2} = 0.44$$

**최선 선택: 포트폴리오 A** (SFR이 가장 큰 0.50)

**포트폴리오 A의 손실 부족 확률:**
- $P(R_A < 3\%) = \Phi(-0.50)$
- z-표에서 $\Phi(-0.50) = 0.3085 = 30.85\%$

> **[Level 3 — "만약~라면?"]** 포트폴리오 B는 기대 수익률이 가장 높지만(11%), SFR은 가장 낮습니다(0.40). 이유: 변동성이 너무 높아(20%) 목표를 밑돌 위험이 크기 때문입니다. 이것은 "기대 수익률만 보면 안 된다"는 교훈입니다.

## 핵심 개념 정리 (Key Concepts)

### LOS 5.a 요약

**포트폴리오 기대 수익률:**
$$E(R_p) = \sum_{i=1}^{n} w_i E(R_i)$$

**2자산 포트폴리오 분산:**
$$\sigma_p^2 = w_A^2 \sigma_A^2 + w_B^2 \sigma_B^2 + 2w_A w_B \text{Cov}_{AB}$$

상관계수로 표현:
$$\sigma_p^2 = w_A^2 \sigma_A^2 + w_B^2 \sigma_B^2 + 2w_A w_B \rho_{AB} \sigma_A \sigma_B$$

> 공분산(상관계수)이 낮을수록 포트폴리오 분산 감소 → 분산투자의 수학적 근거

### LOS 5.b 요약

**결합 확률 함수에서 공분산:**
$$\text{Cov}(R_A, R_B) = \sum_{i=1}^{n} P(A_i B_i) [A_i - E(A)][B_i - E(B)]$$

### LOS 5.c 요약

**안전 제일 비율 (Safety-First Ratio):**
$$\text{SFR} = \frac{E(R_p) - R_L}{\sigma_p}$$

- **손실 부족 위험**: 포트폴리오 수익률이 목표 $R_L$ 미만이 될 확률
- **Roy의 안전 제일 기준**: SFR 최대 포트폴리오 선택
- SFR이 클수록 목표 미달 확률이 작음
- $R_L = R_f$이면 SFR = 샤프 비율

## Module Quiz 5.1

**Q1.** 주식 A와 B의 상관계수는 0.50, 공분산은 0.0043, 주식 B의 표준편차는 26%입니다. 주식 A의 분산은?

A. 0.0011  B. 0.0331  C. 0.2656

**풀이:**

상관계수 공식에서 역으로 분산 계산:
$$\rho_{AB} = \frac{\text{Cov}(R_A, R_B)}{\sigma_A \sigma_B} \quad \Rightarrow \quad \sigma_A = \frac{\text{Cov}(R_A, R_B)}{\sigma_B \cdot \rho_{AB}}$$

$$\sigma_A = \frac{0.0043}{0.26 \times 0.50} = \frac{0.0043}{0.13} = 0.03308$$

$$\text{Var}(R_A) = \sigma_A^2 = (0.03308)^2 = 0.001094 \approx 0.0011$$

**정답: A. 0.0011**

**Q2.** 다음 결합 확률 표에서 P와 Q의 공분산은?

| | Q=7% | Q=4% | Q=0% |
|--|------|------|------|
| **P=15%** | 0.2 | 0 | 0 |
| **P=12%** | 0 | 0.2 | 0 |
| **P=0%** | 0 | 0 | 0.6 |

A. 18.0  B. 18.7  C. 19.3

**풀이:**

1단계 - 기댓값:
$$E(P) = 0.2(15) + 0.2(12) + 0.6(0) = 3 + 2.4 + 0 = 5.4\%$$
$$E(Q) = 0.2(7) + 0.2(4) + 0.6(0) = 1.4 + 0.8 + 0 = 2.2\%$$

2단계 - 공분산:
$$\text{Cov}(P, Q) = 0.2(15-5.4)(7-2.2) + 0.2(12-5.4)(4-2.2) + 0.6(0-5.4)(0-2.2)$$
$$= 0.2(9.6)(4.8) + 0.2(6.6)(1.8) + 0.6(-5.4)(-2.2)$$
$$= 0.2(46.08) + 0.2(11.88) + 0.6(11.88)$$
$$= 9.216 + 2.376 + 7.128 = 18.72$$

**정답: B. 18.7** (≈18.72)

**Q3.** 목표 수익률 4%에서 Roy의 안전 제일 기준으로 최적 포트폴리오는?

| | 포트폴리오 A | 포트폴리오 B | 포트폴리오 C |
|--|-------------|-------------|-------------|
| $E(R_p)$ | 5% | 11% | 18% |
| $\sigma_p$ | 8% | 21% | 40% |

A. A  B. B  C. C

**풀이:**

$$\text{SFR}_A = \frac{5-4}{8} = \frac{1}{8} = 0.125$$
$$\text{SFR}_B = \frac{11-4}{21} = \frac{7}{21} = 0.333$$
$$\text{SFR}_C = \frac{18-4}{40} = \frac{14}{40} = 0.350$$

포트폴리오 C가 SFR 0.350으로 가장 높습니다.

**정답: C. Portfolio C**
