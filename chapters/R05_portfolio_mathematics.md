# Reading 5: Portfolio Mathematics

> **핵심 질문:** 여러 자산을 섞어 포트폴리오를 만들면 수익률과 위험은 어떻게 계산되는가? 그리고 가장 안전한 포트폴리오를 어떻게 선택하는가?

---

## 왜 포트폴리오 수학이 필요한가?

투자자는 보통 한 가지 자산만 보유하지 않습니다. 주식, 채권, 해외 자산을 조합한 **포트폴리오**를 구성합니다. 이때 두 가지 핵심 질문이 생깁니다:

1. **기대 수익률**: 포트폴리오 전체의 평균 수익률은?
2. **위험(분산)**: 포트폴리오 전체의 변동성은?

두 번째 질문이 훨씬 복잡합니다. 왜냐하면 자산들 사이의 **공동 움직임(공분산)**이 포트폴리오 위험에 영향을 미치기 때문입니다.

---

## LOS 5.a: 포트폴리오 기대 수익률과 분산

### 포트폴리오 기대 수익률

n개 자산으로 구성된 포트폴리오의 기대 수익률:

$$E(R_p) = \sum_{i=1}^{n} w_i E(R_i) = w_1 E(R_1) + w_2 E(R_2) + \cdots + w_n E(R_n)$$

**포트폴리오 비중(weight) 정의:**

$$w_i = \frac{\text{자산 } i \text{에 투자된 시장가치}}{\text{포트폴리오 전체 시장가치}}$$

단, $\sum w_i = 1$ (모든 비중의 합 = 1)

**이것은 단순 가중 평균입니다.** 기대 수익률은 각 자산의 기대 수익률을 비중으로 가중 평균한 값과 동일합니다. 분산과 달리 공분산 항이 필요 없습니다.

---

### 공분산 (Covariance)

**공분산이란?** 두 자산의 수익률이 함께 움직이는 정도를 측정합니다.

$$\text{Cov}(R_i, R_j) = E\{[R_i - E(R_i)][R_j - E(R_j)]\}$$

**직관적 해석:**
- **양(+)의 공분산**: 한 자산이 평균 위에 있을 때, 다른 자산도 평균 위에 있는 경향
- **음(-)의 공분산**: 한 자산이 평균 위에 있을 때, 다른 자산은 평균 아래에 있는 경향
- **0**: 두 자산의 움직임이 무관

**공분산의 범위**: $-\infty$ ~ $+\infty$

**중요한 성질:**
- $\text{Cov}(R_A, R_A) = \text{Var}(R_A)$ — 자기 자신과의 공분산 = 분산
- $\text{Cov}(R_A, R_B) = \text{Cov}(R_B, R_A)$ — 순서 무관 (대칭)

---

### 표본 공분산 (Sample Covariance)

과거 수익률 데이터로 공분산을 추정할 때:

$$s_{1,2} = \frac{\sum_{t=1}^{n}(R_{1t} - \bar{R}_1)(R_{2t} - \bar{R}_2)}{n-1}$$

(분모 n-1: 표본 평균을 사용하기 때문에 자유도 1 손실)

---

### 공분산 행렬 (Covariance Matrix)

3개 자산 A, B, C의 공분산 행렬:

|      | A | B | C |
|------|---|---|---|
| **A** | Cov(R_A, R_A) | Cov(R_A, R_B) | Cov(R_A, R_C) |
| **B** | Cov(R_B, R_A) | Cov(R_B, R_B) | Cov(R_B, R_C) |
| **C** | Cov(R_C, R_A) | Cov(R_C, R_B) | Cov(R_C, R_C) |

**핵심 포인트:**
- **대각선**: 각 자산의 분산 (Cov(R_i, R_i) = Var(R_i))
- **비대각선**: 자산 간 공분산 (대칭이므로 절반만 독립)
- n개 자산의 경우: n개의 분산 + n(n-1)/2개의 고유한 공분산

---

### 2자산 포트폴리오 분산

**왜 공분산이 포트폴리오 분산에 들어가는가?**

두 자산 A, B로 구성된 포트폴리오의 수익률:
$$R_p = w_A R_A + w_B R_B$$

분산의 정의를 전개하면:
$$\text{Var}(R_p) = E[(R_p - E(R_p))^2]$$

$R_p - E(R_p) = w_A(R_A - E(R_A)) + w_B(R_B - E(R_B))$ 이므로:

$$\text{Var}(R_p) = w_A^2 \text{Var}(R_A) + w_B^2 \text{Var}(R_B) + 2w_A w_B \text{Cov}(R_A, R_B)$$

더 간결하게:
$$\sigma_p^2 = w_A^2 \sigma_A^2 + w_B^2 \sigma_B^2 + 2w_A w_B \text{Cov}_{AB}$$

상관계수로 대체하면 ($\text{Cov}_{AB} = \rho_{AB} \sigma_A \sigma_B$):
$$\sigma_p^2 = w_A^2 \sigma_A^2 + w_B^2 \sigma_B^2 + 2w_A w_B \rho_{AB} \sigma_A \sigma_B$$

**핵심 인사이트: 분산 감소 효과 (Diversification)**
- $\rho_{AB} = 1$일 때: 포트폴리오 분산 최대 (분산 효과 없음)
- $\rho_{AB} < 1$일 때: 포트폴리오 분산이 가중 평균 분산보다 작음
- $\rho_{AB} = -1$일 때: 포트폴리오 분산 최소 (완전 분산 가능)

공분산(또는 상관계수)이 낮을수록 포트폴리오 분산이 낮아집니다!

---

### 3자산 포트폴리오 분산

$$\sigma_p^2 = w_A^2 \sigma_A^2 + w_B^2 \sigma_B^2 + w_C^2 \sigma_C^2 + 2w_A w_B \text{Cov}_{AB} + 2w_A w_C \text{Cov}_{AC} + 2w_B w_C \text{Cov}_{BC}$$

---

### 예제: 3자산 포트폴리오 분산 계산

포트폴리오 구성: 국내 주식(60%), 국내 채권(30%), 해외 주식(10%)

공분산 행렬 (단위: %²):

| | 국내 주식 | 국내 채권 | 해외 주식 |
|--|-----------|-----------|-----------|
| **국내 주식** | 400 | 44 | 180 |
| **국내 채권** | 44 | 70 | 35 |
| **해외 주식** | 180 | 35 | 450 |

**포트폴리오 분산 계산:**

$$\sigma_p^2 = (0.6)^2(400) + (0.3)^2(70) + (0.1)^2(450)$$
$$+ 2(0.6)(0.3)(44) + 2(0.6)(0.1)(180) + 2(0.3)(0.1)(35)$$
$$= 144 + 6.3 + 4.5 + 15.84 + 21.6 + 2.1 = 194.34 \text{ %}^2$$

**포트폴리오 표준편차:**
$$\sigma_p = \sqrt{194.34} = 13.94\%$$

---

### 상관계수 행렬 (Correlation Matrix)

공분산 대신 상관계수를 이용할 수도 있습니다 ($\text{Cov}_{AB} = \rho_{AB} \sigma_A \sigma_B$):

| | 국내 주식 | 국내 채권 | 해외 주식 |
|--|-----------|-----------|-----------|
| **국내 주식** | 1.000 | 0.263 | 0.424 |
| **국내 채권** | 0.263 | 1.000 | 0.197 |
| **해외 주식** | 0.424 | 0.197 | 1.000 |

**대각선은 항상 1** — 자산과 자신의 상관계수는 완전 정(+)의 상관.

## LOS 5.b: 결합 확률 함수로 공분산 계산

**결합 확률 함수(Joint Probability Function)**: 두 자산이 특정 수익률을 동시에 달성할 확률을 나타냅니다.

### 예제: 결합 확률 함수에서 공분산 계산

경제 상황별 자산 A, B의 수익률:

| | $R_B = 30\%$ | $R_B = 10\%$ | $R_B = 0\%$ |
|--|---|---|---|
| $R_A = 20\%$ | 0.30 | 0 | 0 |
| $R_A = 12\%$ | 0 | 0.50 | 0 |
| $R_A = 5\%$ | 0 | 0 | 0.20 |

**1단계: 기댓값 계산**
$$E(R_A) = (0.3)(0.20) + (0.5)(0.12) + (0.2)(0.05) = 0.13 = 13\%$$
$$E(R_B) = (0.3)(0.30) + (0.5)(0.10) + (0.2)(0.00) = 0.14 = 14\%$$

**2단계: 공분산 계산**

$$\text{Cov}(R_A, R_B) = \sum P(s) \cdot [R_A(s) - E(R_A)] \cdot [R_B(s) - E(R_B)]$$

| 확률 | $R_A$ | $R_B$ | 계산 |
|------|--------|--------|------|
| 0.3 | 0.20 | 0.30 | $(0.3)(0.20-0.13)(0.30-0.14) = 0.00336$ |
| 0.5 | 0.12 | 0.10 | $(0.5)(0.12-0.13)(0.10-0.14) = 0.00020$ |
| 0.2 | 0.05 | 0.00 | $(0.2)(0.05-0.13)(0.00-0.14) = 0.00224$ |

$$\text{Cov}(R_A, R_B) = 0.00336 + 0.00020 + 0.00224 = 0.0058$$

**상관계수 계산:**
$$\rho_{AB} = \frac{\text{Cov}(R_A, R_B)}{\sigma_A \sigma_B}$$

## LOS 5.c: 손실 부족 위험과 안전 제일 기준

### 손실 부족 위험 (Shortfall Risk)

**정의:** 포트폴리오 가치 또는 수익률이 특정 목표 수준 아래로 떨어질 확률.

투자자에게 절대적으로 피해야 할 손실 수준이 있을 때 사용합니다.

예:
- 연금 기금: 수혜자 지급을 위해 최소 3% 수익률 필요
- 대학 기금: 연간 운영비 충당을 위한 최소 포트폴리오 가치 유지

---

### Roy의 안전 제일 기준 (Roy's Safety-First Criterion)

**목표:** 수익률이 최소 허용 수준 $R_L$ 아래로 떨어질 확률 최소화

$$\text{최소화}: P(R_p < R_L)$$

**수익률이 정규분포를 따를 때**, 이 문제는 **안전 제일 비율(Safety-First Ratio, SFR)을 최대화**하는 것으로 변환됩니다:

$$\text{최대화}: \text{SFR} = \frac{E(R_p) - R_L}{\sigma_p}$$

**왜 이 비율인가?**

정규분포에서 $P(R_p < R_L)$는 표준화 변수 $z = \frac{R_L - E(R_p)}{\sigma_p}$의 누적분포에 해당합니다.

$P(R_p < R_L) = \Phi\left(\frac{R_L - E(R_p)}{\sigma_p}\right) = \Phi(-\text{SFR})$

$\Phi$는 감소 함수이므로, SFR이 클수록 $P(R_p < R_L)$가 작아집니다.

즉, **SFR이 클수록 안전**합니다.

**직관:** SFR은 기대 수익률이 목표 수익률보다 몇 표준편차 위에 있는지를 나타냅니다. 이 거리가 클수록 목표 미달 가능성이 낮습니다.

---

### 예제: Roy의 안전 제일 기준

**상황:** $120백만 대학 기금이 연말 최소 $123.6백만을 목표로 합니다.

목표 수익률: $R_L = \frac{123.6 - 120}{120} = 0.030 = 3\%$

세 포트폴리오 비교:

| | 포트폴리오 A | 포트폴리오 B | 포트폴리오 C |
|--|-------------|-------------|-------------|
| $E(R_p)$ | 9% | 11% | 6.6% |
| $\sigma_p$ | 12% | 20% | 8.2% |

안전 제일 비율 계산:

$$\text{SFR}_A = \frac{9 - 3}{12} = \frac{6}{12} = 0.50$$
$$\text{SFR}_B = \frac{11 - 3}{20} = \frac{8}{20} = 0.40$$
$$\text{SFR}_C = \frac{6.6 - 3}{8.2} = \frac{3.6}{8.2} = 0.44$$

**최선 선택: 포트폴리오 A** (SFR이 가장 큰 0.50)

**포트폴리오 A의 손실 부족 확률:**
- $P(R_A < 3\%) = \Phi(-0.50)$
- z-표에서 $\Phi(-0.50) = 0.3085 = 30.85\%$

---

### 시각적 이해: SFR과 정규분포

```
포트폴리오 A: E(R)=12%, σ=18%          포트폴리오 B: E(R)=10%, σ=12%

    ┌──────────┐                              ┌──────────┐
    │          │                              │          │
    │  E(R)=12 │              손실 위험       │  E(R)=10 │
    │          │◄──── SFR=0.667 ────►0%       │          │◄── SFR=0.833 ──►0%
  0% 12%                                    0% 10%

SFR_A = (12-0)/18 = 0.667             SFR_B = (10-0)/12 = 0.833
P(R<0%) ≈ 25%                         P(R<0%) ≈ 20%

→ 포트폴리오 B가 더 안전 (SFR 더 큼)
```

## 핵심 개념 정리 (Key Concepts)

### LOS 5.a 요약

**포트폴리오 기대 수익률:**
$$E(R_p) = \sum_{i=1}^{n} w_i E(R_i)$$

**2자산 포트폴리오 분산:**
$$\sigma_p^2 = w_A^2 \sigma_A^2 + w_B^2 \sigma_B^2 + 2w_A w_B \text{Cov}_{AB}$$

상관계수로 표현:
$$\sigma_p^2 = w_A^2 \sigma_A^2 + w_B^2 \sigma_B^2 + 2w_A w_B \rho_{AB} \sigma_A \sigma_B$$

> 공분산(상관계수)이 낮을수록 포트폴리오 분산 감소 → 분산투자의 수학적 근거

### LOS 5.b 요약

**결합 확률 함수에서 공분산:**
$$\text{Cov}(R_A, R_B) = \sum_{i=1}^{n} P(A_i B_i) [A_i - E(A)][B_i - E(B)]$$

### LOS 5.c 요약

**안전 제일 비율 (Safety-First Ratio):**
$$\text{SFR} = \frac{E(R_p) - R_L}{\sigma_p}$$

- **손실 부족 위험**: 포트폴리오 수익률이 목표 $R_L$ 미만이 될 확률
- **Roy의 안전 제일 기준**: SFR 최대 포트폴리오 선택
- SFR이 클수록 목표 미달 확률이 작음

## Module Quiz 5.1

**Q1.** 주식 A와 B의 상관계수는 0.50, 공분산은 0.0043, 주식 B의 표준편차는 26%입니다. 주식 A의 분산은?

A. 0.0011  B. 0.0331  C. 0.2656

**풀이:**

상관계수 공식에서 역으로 분산 계산:
$$\rho_{AB} = \frac{\text{Cov}(R_A, R_B)}{\sigma_A \sigma_B} \quad \Rightarrow \quad \sigma_A = \frac{\text{Cov}(R_A, R_B)}{\sigma_B \cdot \rho_{AB}}$$

$$\sigma_A = \frac{0.0043}{0.26 \times 0.50} = \frac{0.0043}{0.13} = 0.03308$$

$$\text{Var}(R_A) = \sigma_A^2 = (0.03308)^2 = 0.001094 \approx 0.0011$$

**정답: A. 0.0011**

---

**Q2.** 다음 결합 확률 표에서 P와 Q의 공분산은?

| | Q=7% | Q=4% | Q=0% |
|--|------|------|------|
| **P=15%** | 0.2 | 0 | 0 |
| **P=12%** | 0 | 0.2 | 0 |
| **P=0%** | 0 | 0 | 0.6 |

A. 18.0  B. 18.7  C. 19.3

**풀이:**

1단계 - 기댓값:
$$E(P) = 0.2(15) + 0.2(12) + 0.6(0) = 3 + 2.4 + 0 = 5.4\%$$
$$E(Q) = 0.2(7) + 0.2(4) + 0.6(0) = 1.4 + 0.8 + 0 = 2.2\%$$

2단계 - 공분산:
$$\text{Cov}(P, Q) = 0.2(15-5.4)(7-2.2) + 0.2(12-5.4)(4-2.2) + 0.6(0-5.4)(0-2.2)$$
$$= 0.2(9.6)(4.8) + 0.2(6.6)(1.8) + 0.6(-5.4)(-2.2)$$
$$= 0.2(46.08) + 0.2(11.88) + 0.6(11.88)$$
$$= 9.216 + 2.376 + 7.128 = 18.72$$

**정답: B. 18.7** (≈18.72)

**Q3.** 목표 수익률 4%에서 Roy의 안전 제일 기준으로 최적 포트폴리오는?

| | 포트폴리오 A | 포트폴리오 B | 포트폴리오 C |
|--|-------------|-------------|-------------|
| $E(R_p)$ | 5% | 11% | 18% |
| $\sigma_p$ | 8% | 21% | 40% |

A. A  B. B  C. C

**풀이:**

$$\text{SFR}_A = \frac{5-4}{8} = \frac{1}{8} = 0.125$$
$$\text{SFR}_B = \frac{11-4}{21} = \frac{7}{21} = 0.333$$
$$\text{SFR}_C = \frac{18-4}{40} = \frac{14}{40} = 0.350$$

포트폴리오 C가 SFR 0.350으로 가장 높습니다.

**정답: C. Portfolio C**

## 실전 응용

### 분산투자의 수학적 근거

$$\sigma_p^2 = w_A^2 \sigma_A^2 + w_B^2 \sigma_B^2 + 2w_A w_B \rho_{AB} \sigma_A \sigma_B$$

- $\rho_{AB} = 1$: $\sigma_p = w_A \sigma_A + w_B \sigma_B$ (가중 평균과 동일, 분산 효과 없음)
- $\rho_{AB} = 0$: $\sigma_p = \sqrt{w_A^2 \sigma_A^2 + w_B^2 \sigma_B^2}$ (부분 분산)
- $\rho_{AB} = -1$: 완전 헤지 가능 ($w_A \sigma_A = w_B \sigma_B$이면 $\sigma_p = 0$)

### 실무에서의 안전 제일 기준 활용
- 연금 펀드: 법적 의무 지급액 대비 최소 수익률 설정
- 소버린 웰스 펀드: 국가 재정 안정을 위한 목표 수준 설정
- 개인 투자자: 생활비 마련을 위한 최소 수익률 목표
