# Reading 25: Capital Structure

> **핵심 질문:** 기업은 왜 특정 비율로 부채와 자기자본을 섞는가? 최적 자본구조는 어떻게 결정되는가?

---

## 왜 자본구조를 이해해야 하는가?

### Level 1 (비유): 집 구매의 대출 비율

집을 살 때 자기 돈 30%와 대출 70%를 쓸 수 있습니다. 대출이 많을수록 적은 돈으로 큰 집을 살 수 있지만(레버리지 효과), 이자 부담과 상환 위험이 커집니다. **얼마나 빌릴 것인가?** 이것이 자본구조 결정입니다.

### Level 2 (개념): 투자자에게 왜 중요한가

기업은 자산을 취득하기 위해 자금이 필요합니다. 부채와 자기자본 중 어떤 조합을 사용할까요? 부채는 싸지만(세금 절감) 위험하고(디폴트), 자기자본은 안전하지만 비쌉니다(높은 요구수익률). 이 균형을 잘못 잡으면 기업 가치가 낮아지거나 심지어 파산합니다.

### Level 3 (심화): R20-R24와의 총합

R20(법인 구조)에서 부채와 자기자본의 기본 차이를 배웠고, R21에서 채권자-주주 이해충돌을, R24에서 투자 의사결정의 핵심인 NPV를 배웠습니다. 이 Reading에서 배우는 **WACC가 R24의 NPV 계산에서 할인율**로 사용됩니다. 또한 WACC를 최소화하는 최적 자본구조가 기업 가치를 극대화합니다 — 이것이 기업 재무의 **중심 정리**입니다.

## LOS 25.a: 가중평균자본비용 (WACC)

### WACC란?

**정의:** 기업이 부채와 자기자본 모두로부터 자금을 조달할 때의 **가중 평균 비용**.

$$\text{WACC} = \left[W_d \times r_d \times (1 - T_c)\right] + \left[W_e \times r_e\right]$$

| 변수 | 의미 |
|-----|------|
| $W_d$ | 부채 비중 |
| $r_d$ | 세전 부채 비용 (이자율) |
| $T_c$ | 법인세율 |
| $W_e$ | 자기자본 비중 |
| $r_e$ | 자기자본 비용 |

#### 왜 부채 비용에 (1 - 세율)을 곱하는가? — 이자의 세금 절감 효과 (Tax Shield)

이자비용은 법인세 계산 시 **비용으로 인정**됩니다 → 이자를 내면 세금이 줄어듦.

**예시:** 이자 $100, 세율 30% → 이자로 인한 세금 절감 = $100 × 30% = $30
→ 실질 이자 부담 = $100 - $30 = $70 = $100 × (1-0.30)

이것이 $r_d \times (1-T_c)$의 의미입니다. **배당은 공제 불가** → 자기자본 비용에는 세금 조정 없음.

> **이것이 부채가 "싸게" 느껴지는 근본 이유입니다.** 세금 절감 효과로 부채의 실질 비용이 명목 이자율보다 낮아집니다. 그러나 부채가 과다하면 디폴트 위험 증가 → 부채 비용과 자기자본 비용 모두 상승 → WACC 증가. 최적점이 존재합니다.

#### WACC 가중치 선택

| 가중치 유형 | 설명 | 언제 사용 |
|----------|------|---------|
| **목표 가중치 (Target)** | 경영진이 지향하는 자본구조 | WACC 추정의 이상적 방법 |
| **시장가치 가중치** | 현재 시장에서 거래되는 가치 | 현재 기회비용 반영에 적합 |
| **장부가치 가중치** | 재무제표상 가치 | 경영진 내부 분석에 사용 |

#### WACC 계산 예시

ABC, Inc. 자본구조: 부채 50%, 자기자본 50%
부채 비용: 8%, 자기자본 비용: 11%, 세율: 30%

$$\text{WACC} = [0.50 \times 0.08 \times (1-0.30)] + [0.50 \times 0.11] = 0.028 + 0.055 = 8.3\%$$

## LOS 25.b: 자본구조에 영향을 미치는 요인

### WACC 최소화 원칙

기업은 **WACC를 최소화**하는 자본구조를 목표로 함. 동시에 자산의 특성(단기·장기)에 맞는 자금 조달 기간도 고려.

### 내부 요인 (기업 고유)

| 요인 | 부채 사용 증가 방향 |
|-----|-----------------|
| **매출 성장성·안정성** | 안정적 성장 → 부채 상환 능력 높음 → 부채 비중 ↑ |
| **현금 흐름 성장·예측 가능성** | 예측 가능 → 부채 비중 ↑ |
| **사업 위험** | 위험 클수록 변동성 ↑ → 부채 비중 ↓ |
| **자산 규모·유동성** | 담보 제공 가능한 유동성 높은 자산 → 부채 비중 ↑ |
| **부채 비용·가용성** | 저금리·풍부한 차입 여건 → 부채 비중 ↑ |

**원칙:** 매출·현금 흐름이 안정적이고 예측 가능할수록 더 많은 부채를 수용할 수 있음.

### 산업별 특성

| 산업 특성 | 부채 수용 능력 |
|---------|-------------|
| 비경기순환 (음식·유틸리티) | **높음** — 안정적 매출 |
| 경기순환 (자동차·항공) | **낮음** — 변동성 큰 매출 |
| 고정비 낮음 (낮은 운영 레버리지) | **높음** |
| 구독형 매출 모델 | **높음** — 예측 가능한 현금 흐름 |

### 자산 유형과 부채

- **유형 자산 (Tangible):** 담보로 제공 용이 → 부채 접근성 ↑
- **무형 자산 (Intangible):** 담보 가치 낮음 → 부채 접근성 ↓
- **대체 가능 자산 (Fungible):** 쉽게 매각 가능 → 담보 가치 ↑
- 자산을 직접 소유(vs 프랜차이즈 모델) → 담보 많음 → 부채 비용 ↓

### 기업 성장 단계별 자본구조

| 단계 | 특징 | 부채 비중 |
|-----|------|---------|
| **초창기 (Start-up)** | 매출 초기, 현금흐름 음수, 사업 위험 높음, 담보 자산 부족 | **거의 0% (자기자본만)** |
| **성장기 (Growth)** | 매출·현금흐름 증가, 위험 감소, 고정자산·채권 담보 가능 | **보수적으로 부채 사용** |
| **성숙기 (Mature)** | 매출 성장 둔화, 현금흐름 안정·예측 가능 | **부채 폭넓게 사용** |

> **초창기 예외:** 주가 급등 기업은 전환사채 발행 가능 (저금리 + 즉시 희석 방지)

### 외부 요인 (거시경제)

- **인플레이션, 실질 GDP 성장률, 통화정책, 환율** → 기준금리에 영향
- 경기 침체기: 투자자들이 회사채 스프레드 요구 ↑ → 부채 비용 ↑ (경기순환 기업에 특히 영향)
- 특정 산업 호황 (예: 유가 상승 시 석유 기업): 신용 스프레드 축소

## LOS 25.c: 모딜리아니-밀러(MM) 명제

### MM이란?

1958년 프랑코 모딜리아니와 머튼 밀러는 자본구조 이론의 기초가 되는 연구를 발표했습니다.

### MM 명제 I: 세금 없음 — 자본구조 무관련성

#### 가정

| 가정 | 내용 |
|-----|------|
| 완전 자본시장 | 거래비용·세금·파산비용 없음 |
| 동질적 기대 | 모든 투자자 동일한 현금흐름 기대 |
| 무위험 차입·대출 | 투자자가 무위험 이자율로 차입·대출 가능 |
| 대리인 비용 없음 | 경영자-주주 간 이해충돌 없음 |
| 투자 결정의 독립성 | 영업 이익은 자금 조달 방식과 무관 |

#### 결론: 자본구조는 기업 가치에 영향을 주지 않는다

**파이 비유:** 파이(기업 가치)의 크기는 어떻게 자르느냐(자본구조)와 무관.

왜냐하면:
- 순부채 없는 기업: EBIT 전부 → 주주
- 부채+자기자본 기업: EBIT → 채권자 + 주주
- 둘 다 같은 EBIT → 총 가치는 동일

### MM 명제 II: 세금 없음 — 자기자본 비용과 레버리지

**결론:** 부채 비중이 높아질수록 자기자본 비용이 **선형**으로 증가.

$$r_e = r_0 + \frac{D}{E}(r_0 - r_d)$$

| 변수 | 의미 |
|-----|------|
| $r_e$ | 자기자본 비용 |
| $r_0$ | 무부채 기업의 자기자본 비용 |
| $r_d$ | 부채 비용 |
| $D/E$ | 부채-자기자본 비율 (레버리지) |

**직관:** 부채가 많아질수록 주주의 잔여 현금흐름이 더 불확실 → 주주가 더 높은 수익률 요구.

**핵심 결론:**
- 부채 비중 ↑ → $r_e$ ↑ (자기자본 비용 증가)
- 하지만 저렴한 부채의 이득 = 비싸진 자기자본의 손해 → **WACC 불변**
- 즉: 부채 사용이 자본구조를 싸게 만들지 않음

### MM — 세금 포함: 100% 부채가 최적

#### 이자 세금 방패 (Tax Shield)

대부분 국가에서 **이자비용은 세금 공제** 가능, 배당은 불가. → 부채 사용 → 정부의 파이 조각 감소 → 주주·채권자에게 돌아갈 파이 증가.

$$\text{세금 방패 가치} = T_c \times D$$

$$V_L = V_U + (T_c \times D)$$

| 변수 | 의미 |
|-----|------|
| $V_L$ | 부채 있는 기업의 가치 |
| $V_U$ | 무부채 기업의 가치 |
| $T_c$ | 세율 |
| $D$ | 부채 금액 |

#### 세금 포함 MM 결론

- 부채 비중 ↑ → 세금 방패 ↑ → WACC ↓ → 기업 가치 ↑
- **이론적 결론: 100% 부채가 최적** (세금 방패 극대화)

#### 세금 포함 자기자본 비용 공식

$$r_e = r_0 + \frac{D}{E}(r_0 - r_d)(1 - T_c)$$

세금 방패로 인해 부채 증가 시 자기자본 비용 상승 폭이 세금 없을 때보다 낮음.

#### 현실에서 100% 부채 기업이 없는 이유

→ **파산비용 (재무적 곤경 비용)**이 존재하기 때문!

---

### 재무적 곤경 비용 (Costs of Financial Distress)

부채가 너무 많으면 이자 지급 불능 위험 → 아래 비용 발생:

| 구분 | 내용 |
|-----|------|
| **직접 비용** | 법적·행정 비용 (변호사 비용, 파산 절차 비용) |
| **간접 비용** | 투자 기회 상실, 고객·공급자·직원 신뢰 상실 |
| **부채의 대리인 비용** | 재무 위기 시 경영진(주주 대리)과 채권자 간 이해충돌 |

**재무적 곤경 확률이 높아지는 경우:**
- 금융 레버리지 ↑
- 경영진 역량 낮음
- 기업 지배구조 미흡

## LOS 25.d: 최적·목표 자본구조

### 정적 트레이드오프 이론 (Static Tradeoff Theory)

**세금 방패 이득**과 **재무적 곤경 비용**을 균형 맞추는 최적점 존재.

$$V_L = V_U + (T_c \times D) - PV(\text{재무적 곤경 비용})$$

#### 기업 가치와 레버리지 관계

| 구간 | 레버리지 증가 시 |
|-----|--------------|
| **낮은 레버리지** | 세금 방패 이득 > 재무적 곤경 비용 → 기업 가치 ↑ |
| **최적점** | 한계 세금 방패 이득 = 한계 재무적 곤경 비용 → **기업 가치 극대, WACC 극소** |
| **높은 레버리지** | 재무적 곤경 비용 > 세금 방패 이득 → 기업 가치 ↓ |

**최적 자본구조 결정 요인:** 사업 위험, 세율, 기업 지배구조, 산업 관행.

### MM 이론 3단계 요약

| 모델 | WACC | 기업 가치 |
|-----|------|---------|
| **MM (세금 없음)** | 자본구조와 무관 | 자본구조와 무관 |
| **MM (세금 있음)** | 부채 ↑ → WACC ↓ | 부채 ↑ → 가치 ↑ (최적: 100% 부채) |
| **정적 트레이드오프** | 최적점에서 최소 | 최적점에서 최대 |

### 목표 자본구조 (Target Capital Structure)

**정의:** 기업 가치를 극대화하기 위해 장기적으로 달성하고자 하는 평균 자본구조.

#### WACC 계산 시 어떤 가중치를 사용?

이상적으로는 목표 자본구조 가중치 사용 (시장가치 기반).

**외부 분석가가 목표 자본구조를 추정하는 방법:**
1. 현재 자본구조 (시장가치 기준)를 목표로 사용
2. 최근 자본구조 추세를 외삽하여 추정
3. 동종 산업 평균 자본구조 사용

**경영진이 장부가치를 선호하는 이유:**
- 단기 시장 변동이 실제 적정 부채 수준에 영향 없음
- 경영진은 자본 배분 관점 (투자자 관점과 다름)
- 신용평가사가 장부가치 지표 사용 → 경영진이 이를 의식

### 실제 자본구조의 변동

- 목표 주변에서 자본구조 **변동** 발생
- 주가 급등 시 추가 자기자본 발행 → 일시적으로 자기자본 비중 ↑
- 외부 자본은 최소 단위로 조달 → 정확한 가중치 유지 불가

---

### 정보 비대칭 비용 (Costs of Asymmetric Information)

경영진은 투자자보다 기업 전망에 대해 더 많은 정보 보유 → 신호 효과 발생.

| 자금 조달 방식 | 투자자에게 보내는 신호 |
|-------------|------------------|
| **부채 발행** | "이자 지급 능력 있다" → **긍정적** 신호 |
| **자기자본 발행** | "주가가 고평가됐다" → **부정적** 신호 |

복잡한 제품이나 불투명한 재무제표 기업 → 정보 비대칭 비용 ↑ → 자기자본·부채 요구 수익률 모두 ↑

### 자기자본의 대리인 비용 (Agency Costs of Equity)

경영진-주주 이해충돌에서 발생하는 비용:

| 구성 요소 | 내용 |
|---------|------|
| **모니터링 비용** | 경영진 감시 비용 (주주 보고, 이사회 보수) |
| **결속 비용 (Bonding)** | 경영진의 충성 보장 비용 (성과 보험료, 비경쟁 계약) |
| **잔여 손실** | 감시·결속으로도 완전히 방지 못한 손실 |

**자유 현금흐름 가설 (Free Cash Flow Hypothesis):**
- 부채 ↑ → 이자 지급 의무 ↑ → 경영진이 자유롭게 쓸 현금 ↓ → 방만 경영 억제 → 대리인 비용 ↓

### 순서 이론 (Pecking Order Theory)

정보 비대칭에 기반한 자금 조달 선호 순서:

$$\text{내부 유보 이익} \succ \text{부채} \succ \text{외부 자기자본 발행}$$

| 순위 | 방식 | 이유 |
|-----|------|------|
| 1위 | **내부 유보 이익** | 투자자에게 아무 신호 없음 |
| 2위 | **부채** | 부정적 신호 적음 |
| 3위 | **신규 자기자본 발행** | 주가 고평가 신호 → 투자자 부정적 반응 |

→ 자본구조는 최적 목표의 결과가 아닌 **개별 자금 조달 결정의 부산물**

## 핵심 개념 정리

### LOS 25.a 요약

$$\text{WACC} = [W_d \times r_d \times (1-T_c)] + [W_e \times r_e]$$

- 부채 비용 < 자기자본 비용 (채권자가 우선 청구권 보유)
- 이자비용 세금 공제 → 세후 부채 비용 = $r_d \times (1-T_c)$

### LOS 25.b 요약
- 내부 요인: 현금흐름 안정성·예측 가능성 ↑ → 부채 비중 ↑
- 외부 요인: 경기 침체기 → 스프레드 ↑ → 부채 비용 ↑
- 기업 성장 단계: 초창기(자기자본만) → 성장기(보수적 부채) → 성숙기(폭넓은 부채)

### LOS 25.c 요약
- **MM (세금 없음):** 자본구조 무관련 — WACC·기업 가치 불변
- **MM (세금 있음):** 100% 부채 = 최적 (세금 방패 극대)
- **MM II:** $r_e$ = $r_0 + (D/E)(r_0 - r_d)$ — 레버리지 ↑ → $r_e$ ↑, WACC 불변

### LOS 25.d 요약
- **정적 트레이드오프:** 세금 방패 이득 vs 재무적 곤경 비용 → 최적점 존재
- $V_L = V_U + T_c \times D - PV(\text{재무적 곤경 비용})$
- **목표 자본구조:** 시장가치 기반 가중치 사용, 추세·산업 평균으로 추정
- **순서 이론:** 내부 이익 > 부채 > 신규 자기자본 (정보 비대칭 기반)
- **자유 현금흐름 가설:** 부채 ↑ → 경영진 방만 경영 억제 → 대리인 비용 ↓
