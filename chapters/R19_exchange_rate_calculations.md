# Reading 19: Exchange Rate Calculations

## LOS 19.a: 교차환율 계산

### 왜 이걸 알아야 하는가?

서울에서 리스본으로 직항이 없으면 파리를 경유한다. 마찬가지로 원화와 브라질 레알 간 직접 시장이 없으면, 달러를 경유해서 환율을 계산한다. CFA 시험에서 가장 빈출되는 계산 유형이다.

### 핵심 원리: 환율을 분수처럼 다뤄라

MXN/USD = 10.70, USD/AUD = 0.60 일 때, MXN/AUD는?

**1단계:** 원하는 결과를 분수로 쓴다: MXN/AUD = ?

**2단계:** 주어진 것을 배열해서 공통 통화(USD)를 소거:

$$\frac{\text{MXN}}{\cancel{\text{USD}}} \times \frac{\cancel{\text{USD}}}{\text{AUD}} = \frac{\text{MXN}}{\text{AUD}}$$

$$10.70 \times 0.60 = 6.42$$

### 직접 해보자 — 3가지 패턴

**패턴 1: 곱셈 (공통 통화가 분자·분모에 하나씩)**

위 예시가 이 패턴이다.

**패턴 2: 나눗셈 (공통 통화가 같은 위치)**

CHF/USD = 1.7799, NZD/USD = 2.2529 → CHF/NZD = ?

USD가 둘 다 분모에 있다. 나눗셈으로 소거:

$$\frac{\text{CHF}/\cancel{\text{USD}}}{\text{NZD}/\cancel{\text{USD}}} = \frac{\text{CHF}}{\text{NZD}} = \frac{1.7799}{2.2529} = 0.7900$$

**패턴 3: 역수 활용**

AUD/USD = 1.67이 있는데 USD/AUD가 필요하면:

$$\text{USD/AUD} = \frac{1}{1.67} = 0.5988$$

### 연습 — 1 NZD로 시작해서 따라가기

NZD/USD = 1.6000, IDR/USD = 2,400 → IDR/NZD는?

1 NZD로 시작:
1. 1 NZD → 1/1.6 = 0.625 USD
2. 0.625 USD → 0.625 × 2,400 = **1,500 IDR**

∴ IDR/NZD = 1,500

대수적 확인: IDR/USD ÷ NZD/USD = 2,400/1.6 = 1,500 ✓

## LOS 19.b: 선물환율과 금리의 차익거래 관계

### 왜 선물환율 ≠ 현물환율인가?

A 은행 금리 5%, B 은행 금리 3%라고 하자. 아무 비용 없이 B에서 빌려 A에 넣을 수 있다면? 모두가 같은 짓을 해서 차이가 사라진다. 외환 시장에서도 선물환율이 금리 차를 반영하지 않으면 무위험 이익이 가능하고, 차익거래가 그 기회를 없앤다.

### 공식

$$\text{Forward}_{P/B} = \text{Spot}_{P/B} \times \frac{1 + r_{\text{가격통화}}}{1 + r_{\text{기준통화}}}$$

### 숫자를 넣어보자

ABE/DUB spot = 4.5671, ABE 금리 = 5%, DUB 금리 = 3%

$$\text{Forward} = 4.5671 \times \frac{1.05}{1.03} = 4.5671 \times 1.0194 = 4.6558$$

확인: Forward > Spot → ABE(가격통화) 금리가 더 높으므로 DUB(기준통화)가 선물에서 비싸짐(프리미엄) ✓

선물 프리미엄 = 4.6558/4.5671 - 1 = 1.94% ≈ 금리 차이 2%

### 역으로 — 현물 구하기

USD/CHF 1년 선물 = 0.80, USD 금리 = 10%, CHF 금리 = 4%

$$\text{Spot} = \frac{\text{Forward} \times (1 + r_{\text{기준통화}})}{1 + r_{\text{가격통화}}} = 0.80 \times \frac{1.04}{1.10} = 0.7564$$

### 차익거래 — 시장 선물환이 "틀렸을 때"

무차익 선물환 = 4.6558인데 시장 선물환 = 4.6000이라면?

시장 선물이 너무 낮다 = ABE 투자가 유리.

```
1. 1,000 DUB를 3%로 차입
2. 현물 환전: 1,000 × 4.5671 = 4,567.1 ABE
3. ABE에 5% 투자: 4,567.1 × 1.05 = 4,795.45 ABE
4. 선물환으로 환전: 4,795.45 / 4.6000 = 1,042.49 DUB
5. 차입금 상환: 1,000 × 1.03 = 1,030 DUB
6. 순이익 = 1,042.49 - 1,030 = 12.49 DUB (무위험!)
```

시장 반응: ABE 현물 매입 → Spot↑, ABE 선물 매각 → Forward↑ → 4.6558으로 수렴.

### 포인트와 퍼센트 표기

**포인트:** 스팟 호가의 마지막 소수점 단위.

AUD/EUR spot = 0.7313, 1년 선물 = +3.5 포인트
→ 0.7313 + 0.00035 = **0.73165**

**퍼센트:**

AUD/EUR spot = 0.7313, 120일 선물 = -0.062%
→ 0.7313 × (1 - 0.00062) = **0.7308**

**선물 프리미엄/디스카운트** = Forward/Spot - 1. 양수=프리미엄, 음수=디스카운트.

주의: 선물환율은 미래 현물의 좋은 예측치가 **아니다**. 차익거래 방지 조건만 반영한다.

## Key Concepts

**LOS 19.a:**
- 교차환율 = 제3 통화 경유. 분수 배열로 공통 통화 소거
- 곱셈(공통이 분자·분모에), 나눗셈(공통이 같은 위치), 역수 활용

**LOS 19.b:**
- Forward = Spot × (1+r_가격)/(1+r_기준)
- 고금리 통화 = 선물에서 절하 (차익거래 방지)
- 포인트(마지막 소수점 단위), 퍼센트(Spot × (1±%))
- 선물환 ≠ 미래 현물 예측

## Quiz

**Q1.** IDR/USD = 2,400, NZD/USD = 1.6000. NZD/IDR은?

> IDR/NZD = 2,400/1.6 = 1,500. NZD/IDR = 1/1,500 = **0.00067**
>
> **정답: 0.00067**

---

**Q2.** CHF/GBP spot = 1.3050, 180일 선물 = -42.5 포인트. 선물환율은?

> 1포인트 = 0.0001. 42.5포인트 = 0.00425. 1.3050 - 0.00425 = **1.3008**
>
> **정답: 1.3008**

---

**Q3.** NZD/USD spot = 1.4286, 180일 선물 = 1.3889. 이것이 의미하는 바는?

> Forward < Spot → USD(기준통화) 디스카운트 → USD 금리가 NZD보다 **낮음**. (고금리 NZD가 선물에서 절하.)
>
> **정답: 미국 금리가 뉴질랜드보다 낮음**

---

**Q4.** USD 금리 4%, SEK 금리 7%, SEK/USD spot = 9.5238. 1년 선물은?

> Forward = 9.5238 × (1.07/1.04) = 9.5238 × 1.0288 = **9.7985**
>
> SEK(가격) 금리 > USD(기준) 금리 → Forward > Spot ✓
>
> **정답: 9.7985**

---

**Q5.** USD 금리 10%, CHF 금리 4%, USD/CHF 1년 선물 = 0.80. 현물은?

> Spot = 0.80 × (1.04/1.10) = **0.7564**
>
> **정답: 0.7564**
