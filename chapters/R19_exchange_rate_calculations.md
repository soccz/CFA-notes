# Reading 19: Exchange Rate Calculations

> **핵심 질문:** 교차환율은 어떻게 계산하는가? 선물환율과 금리는 어떤 관계인가?

---

## 왜 환율 계산을 배워야 하는가?

실제 외환 거래에서는 두 통화의 직접 시장이 없는 경우가 많습니다. 이럴 때 공통 기준 통화(보통 USD)를 이용한 교차환율 계산이 필요합니다. 또한 선물환율이 금리와 정확히 어떤 관계를 갖는지 이해하면, 그 관계가 깨질 때 발생하는 차익거래 기회를 포착하거나 피할 수 있습니다.

---

## LOS 19.a: 교차환율 계산

### 교차환율이란?

**정의:** 두 통화 간 직접 시장이 없을 때, 공통 제3 통화(주로 USD 또는 EUR)를 이용해 계산하는 환율.

### 교차환율 계산 방법

**핵심 원리:** 표기법이 대수적으로 맞도록 배열하면 됨.

**예시 1 (곱셈):**

$$\text{MXN/USD} = 10.70, \quad \text{USD/AUD} = 0.60$$
$$\text{MXN/AUD} = \text{USD/AUD} \times \text{MXN/USD} = 0.60 \times 10.70 = 6.42$$

USD가 분자·분모에서 소거됨 → 원하는 MXN/AUD 교차환율 획득.

**예시 2 (나눗셈):**

$$\text{CHF/USD} = 1.7799, \quad \text{NZD/USD} = 2.2529$$
$$\text{CHF/NZD} = \frac{\text{CHF/USD}}{\text{NZD/USD}} = \frac{1.7799}{2.2529} = 0.7900$$

USD를 분자·분모에서 소거하면 CHF/NZD 완성.

**인버스 활용:** AUD/USD = 1.67로 주어졌으면 USD/AUD = 1/1.67로 변환 후 사용.

### 예시 3 (교차환율 계산 연습)

> NZD/USD = 1.6000, IDR/USD = 2,400.00 → NZD/IDR는?

1 NZD로 시작:
- 1 NZD → (1/1.6) = 0.625 USD
- 0.625 USD → 0.625 × 2,400 = 1,500 IDR

교차환율: **IDR/NZD = 1,500** 또는 **NZD/IDR = 1/1,500 = 0.00067**

## LOS 19.b: 현물환율-선물환율-금리의 차익거래 관계

### 이자율 평가 (Interest Rate Parity)와 무차익거래 조건

**핵심 아이디어:** 자유롭게 거래되는 통화에서, 선물환율과 현물환율의 차이(%)는 두 국가의 금리 차이와 거의 같아야 함.

**이유:** 이 관계가 성립하지 않으면 **무위험 차익거래(riskless arbitrage)** 가 가능하기 때문.

### 무차익거래 조건 (No-Arbitrage Condition)

$$1 + r_{\text{domestic}} = \text{Spot}_{d/f} \times (1 + r_{\text{foreign}}) \times \frac{1}{\text{Forward}_{d/f}}$$

정리하면:

$$\frac{\text{Forward}_{d/f}}{\text{Spot}_{d/f}} = \frac{1 + r_{\text{price currency}}}{1 + r_{\text{base currency}}}$$

$$\text{Forward}_{d/f} = \text{Spot}_{d/f} \times \frac{1 + r_{\text{price currency}}}{1 + r_{\text{base currency}}}$$

**해석:**
- 가격통화(분자) 금리가 높을수록 → 선물환율 > 현물환율 (기준통화 선물 프리미엄)
- 기준통화(분모) 금리가 높을수록 → 선물환율 < 현물환율 (기준통화 선물 디스카운트)
- **고금리 통화는 선물시장에서 절하됨** (차익거래 방지)

### 선물환율 계산 예시

**예시 1 (1년 선물):**

> ABE/DUB spot = 4.5671, ABE 금리 = 5%, DUB 금리 = 3%

$$\text{Forward}_{A/D} = 4.5671 \times \frac{1.05}{1.03} = 4.6558 \text{ ABE/DUB}$$

확인: 선물이 현물보다 (4.6558/4.5671 - 1) = **1.94%** 높음 ≈ 금리 차이 5% - 3% = 2% (근사적으로 일치)

**예시 2 (90일 선물):**

> 90일 기간 금리 사용: ABE 5%, DUB 3% (모두 90일 금리)

$$\text{Forward}_{A/D} = 4.5671 \times \frac{1 + 0.05}{1 + 0.03} = 4.5671 \times \frac{1.05}{1.03} = 4.5671 \times 1.0194 = 4.6558$$

*주의: 단기 금리는 머니마켓 수익률로 표시됨.*

**예시 3 (역산: 현물환율 구하기):**

> USD/CHF 1년 선물 = 0.80, USD 금리 = 10%, CHF 금리 = 4%

$$\text{Spot} = \text{Forward} \times \frac{1 + r_{\text{CHF}}}{1 + r_{\text{USD}}} = 0.80 \times \frac{1.04}{1.10} = 0.7564$$

### 차익거래 예시

**조건:** ABE/DUB spot = 4.5671, ABE 금리 5%, DUB 금리 3%, 시장 선물환율 = 4.6000 (무차익 4.6558보다 낮음)

→ ABE가 선물에서 과소 절하 → DUB 투자자는 ABE에 투자하면 더 유리

**차익거래 과정:**
1. 1,000 DUB를 3%로 차입
2. 현물 환율로 ABE 매입: 1,000 × 4.5671 = 4,567.1 ABE
3. ABE를 5%로 투자: 4,567.1 × 1.05 = **4,795.45 ABE**
4. 선물환 계약 체결: 4,795.45 ABE → 4,795.45/4.6000 = **1,042.49 DUB**
5. 차입금 상환: 1,000 × 1.03 = 1,030 DUB
6. **순이익 = 1,042.49 - 1,030 = 12.49 DUB** (무위험)

**차익거래자들의 반응:**
- ABE 현물 매입 → ABE/DUB 현물환율 하락
- ABE 선물 매각 → ABE/DUB 선물환율 상승
- 결국 무차익 선물환율로 수렴

### 선물환율 표기: 포인트와 퍼센트

**포인트 (Points):** 스팟 호가의 마지막 소수점 단위.

예: 스팟 2.3481 → 1 포인트 = 0.0001

> AUD/EUR spot = 0.7313, 1년 선물 = +3.5 points
> 선물환율 = 0.7313 + 0.00035 = **0.73165**

**퍼센트 (Percentage):**

> AUD/EUR spot = 0.7313, 120일 선물 = -0.062%
> 선물환율 = 0.7313 × (1 - 0.00062) = **0.7308**

### 선물 프리미엄·디스카운트 해석

$$\text{선물 프리미엄/디스카운트} = \frac{\text{Forward}}{\text{Spot}} - 1$$

> USD/EUR spot = 1.312, 90일 선물 = 1.320
> 유로의 선물 프리미엄 = 1.320/1.312 - 1 = **+0.610%** → 유로 절상 예상

- 양수 → **선물 프리미엄** (기준통화 절상 예상)
- 음수 → **선물 디스카운트** (기준통화 절하 예상)

**중요:** 선물환율은 미래 현물환율의 **좋은 예측치가 아님**. 단지 차익거래를 방지하는 무차익 조건을 반영할 뿐.

## 핵심 개념 정리

### LOS 19.a 요약
- 교차환율 = 제3 통화를 통해 두 통화 간 환율 계산
- 표기법을 대수적으로 배열: 원하는 통화 쌍이 나오도록 곱하거나 나눔

### LOS 19.b 요약
- 무차익거래 선물환율:
$$\text{Forward} = \text{Spot} \times \frac{1 + r_{\text{price currency}}}{1 + r_{\text{base currency}}}$$
- 고금리 통화 = 선물에서 절하 (차익거래 방지)
- 포인트: 스팟의 마지막 소수점 단위 → 더하기/빼기
- 퍼센트: Spot × (1 + %) → 선물환율
- 선물 프리미엄/디스카운트 = Forward/Spot - 1
- 선물환율은 미래 현물환율의 예측치 아님 (무차익 조건만 반영)

## Module Quiz 19.1

**Q1.** IDR/USD = 2,400, NZD/USD = 1.6000. NZD/IDR 교차환율은?

1 NZD → 1/1.6 = 0.625 USD → 0.625 × 2,400 = 1,500 IDR
NZD/IDR = 1/1,500 = **0.00067**

**정답: A**

---

**Q2.** USD/NZD = 0.3500, NZD/SEK = 0.3100. USD/SEK 교차환율은?

$$\text{USD/SEK} = \text{USD/NZD} \times \text{NZD/SEK} = 0.3500 \times 0.3100 = 0.1085$$

NZD 항이 소거됨.

**정답: A (0.1085)**

---

**Q3.** CHF/GBP spot = 1.3050, 180일 선물 = -42.5 포인트. 180일 선물환율은?

$$1.3050 - 0.00425 = 1.30075$$

**정답: B (≈ 1.3008)**

---

**Q4.** NZD/USD spot = 1.4286, 180일 선물 = 1.3889. 이것이 의미하는 바는?

선물에서 NZD/USD가 낮아짐 → USD를 사는 데 NZD가 덜 필요 → **USD가 선물에서 절하, 즉 USD 금리가 낮음**.

선물율 공식: Forward/Spot = (1 + r_USD) / (1 + r_NZD)
1.3889/1.4286 = 0.972 < 1 → r_USD < r_NZD → **미국 금리가 뉴질랜드보다 낮음**

**정답: A**

---

**Q5.** USD/GBP spot = 1.533, 180일 선물 = 1.508. USD의 선물 프리미엄/디스카운트는?

USD를 기준통화로 보려면 GBP/USD로 변환:
$$\text{spot GBP/USD} = 1/1.533 = 0.6523$$
$$\text{forward GBP/USD} = 1/1.508 = 0.6631$$

$$\frac{0.6631}{0.6523} - 1 = +1.66\%$$

→ USD 선물 프리미엄 **+1.66%** (또는 spot/forward - 1 = 1.533/1.508 - 1 = 1.66%)

**정답: B (1.66% 프리미엄)**

---

**Q6.** USD 연 금리 4%, SEK 연 금리 7%, SEK/USD spot = 9.5238. 1년 선물 SEK/USD는?

$$\text{Forward} = 9.5238 \times \frac{1 + 0.07}{1 + 0.04} = 9.5238 \times \frac{1.07}{1.04} = 9.5238 \times 1.0288 = \textbf{9.7985}$$

SEK 금리가 더 높으므로 SEK가 약 3% 절하됨 (더 많은 SEK가 필요).

**정답: B (9.7985)**

---

**Q7.** USD 금리 10%, CHF 금리 4%, USD/CHF 1년 선물 = 0.80. 현물 USD/CHF는?

$$\text{Spot} = \text{Forward} \times \frac{1 + r_{\text{CHF}}}{1 + r_{\text{USD}}} = 0.80 \times \frac{1.04}{1.10} = 0.7564$$

USD 금리가 더 높으므로 현물에서 USD가 더 저렴(더 강한) → 현물 USD/CHF < 선물.

**정답: A (0.7564)**
