# Reading 19: Exchange Rate Calculations

> **핵심 질문:** 교차환율은 어떻게 계산하는가? 선물환율과 금리는 어떤 관계이며, 왜 그런 관계가 성립해야 하는가?

---

## 왜 환율 계산을 배워야 하는가?

### Level 1 (비유): 항공 경유편

서울에서 리스본으로 직항이 없으면, 파리를 경유합니다. 마찬가지로 **두 통화 간 직접 시장이 없으면**, 공통 통화(보통 USD)를 경유하여 환율을 계산합니다. 이것이 교차환율입니다.

### Level 2 (개념): 투자자에게 왜 중요한가

실제 외환 거래에서 모든 통화 쌍에 활성 시장이 있는 것은 아닙니다. 교차환율 계산은 글로벌 포트폴리오의 기본 도구입니다. 또한 선물환율이 금리 차이를 정확히 반영하지 않으면 **무위험 차익거래** 기회가 발생합니다 — 이를 이해하면 위험 관리와 수익 기회 포착에 유리합니다.

### Level 3 (심화): R18과의 연결

R18에서 배운 이자율 평가(IRP)의 개념적 이해를 이 Reading에서 **실제 계산**으로 구체화합니다. 교차환율 + IRP 계산은 CFA 시험에서 가장 빈출되는 계산 문제 유형 중 하나입니다.

---

## LOS 19.a: 교차환율 계산

### 교차환율이란?

**정의:** 두 통화 간 직접 시장이 없을 때, 공통 제3 통화(주로 USD 또는 EUR)를 이용해 계산하는 환율.

### 핵심 원리: 대수적 소거

환율 표기를 **분수**처럼 취급하면 됩니다. 원하는 통화 쌍이 나오도록 곱하거나 나누어 **공통 통화를 소거**합니다.

#### 방법 1: 곱셈 (공통 통화가 분자와 분모에 하나씩)

$$\text{MXN/USD} = 10.70, \quad \text{USD/AUD} = 0.60$$

$$\text{MXN/AUD} = \underbrace{\text{USD/AUD}}_{0.60} \times \underbrace{\text{MXN/USD}}_{10.70} = 6.42$$

USD가 분자(첫 항)·분모(둘째 항)에서 소거 → MXN/AUD 획득.

#### 방법 2: 나눗셈 (공통 통화가 같은 위치)

$$\text{CHF/USD} = 1.7799, \quad \text{NZD/USD} = 2.2529$$

$$\text{CHF/NZD} = \frac{\text{CHF/USD}}{\text{NZD/USD}} = \frac{1.7799}{2.2529} = 0.7900$$

USD가 분모에 공통 → 나눗셈으로 소거.

#### 방법 3: 역수 활용

AUD/USD = 1.67이 주어졌는데 USD/AUD가 필요하면: USD/AUD = 1/1.67 = 0.5988

> **팁:** 교차환율 계산에서 실수하는 가장 큰 이유는 **어떤 통화가 분자이고 어떤 것이 분모인지** 혼동하는 것입니다. 항상 "이 환율은 기준통화 1단위당 가격통화 몇 단위인가?"를 확인하세요.

### 연습 문제 스타일 예시

> NZD/USD = 1.6000, IDR/USD = 2,400.00 → IDR/NZD는?

**단계별 접근 (1 NZD로 시작):**
1. 1 NZD → 1/1.6 = 0.625 USD
2. 0.625 USD → 0.625 × 2,400 = 1,500 IDR

∴ **IDR/NZD = 1,500** (또는 NZD/IDR = 1/1,500 = 0.00067)

**대수적 접근:**
$$\text{IDR/NZD} = \frac{\text{IDR/USD}}{\text{NZD/USD}} = \frac{2400}{1.6} = 1500$$

## LOS 19.b: 현물환율-선물환율-금리의 차익거래 관계

### 왜 선물환율은 현물환율과 다른가? — 무차익거래 논리

#### Level 1 (비유): 금리가 다른 두 은행

A 은행 금리 5%, B 은행 금리 3%. 만약 아무 비용 없이 A에서 빌려 B에 넣을 수 있다면? 모두가 같은 행동을 하므로 금리 차이가 사라집니다. 외환 시장에서 **선물환율이 금리 차이를 반영하는 이유**도 동일합니다 — 반영하지 않으면 무위험 이익이 가능하고, 차익거래가 그 기회를 없앱니다.

#### Level 2: 무차익거래 조건

**논리 체인:**
```
통화 A 차입 (이자율 rA) → 현물환율로 통화 B 환전 → 통화 B 투자 (이자율 rB)
→ 선물환 계약으로 다시 통화 A 환전

이 과정의 수익 = 통화 A 차입 비용이어야 차익거래 불가
```

$$\boxed{\frac{\text{Forward}_{P/B}}{\text{Spot}_{P/B}} = \frac{1 + r_{\text{가격통화}}}{1 + r_{\text{기준통화}}}}$$

$$\text{Forward}_{P/B} = \text{Spot}_{P/B} \times \frac{1 + r_{\text{가격통화}}}{1 + r_{\text{기준통화}}}$$

#### Level 3: 수식 유도

1단위 기준통화를 두 경로로 투자하는 결과가 같아야 합니다:

**경로 1 (기준통화국에서 직접 투자):** $1 \times (1 + r_B) = (1 + r_B)$ 기준통화

**경로 2 (가격통화국으로 환전 후 투자, 선물환으로 환원):**
$1 \times \text{Spot}_{P/B} \times (1 + r_P) \times \frac{1}{\text{Forward}_{P/B}}$ 기준통화

두 경로 동일: $(1 + r_B) = \text{Spot}_{P/B} \times (1 + r_P) / \text{Forward}_{P/B}$

정리: $\text{Forward}_{P/B} = \text{Spot}_{P/B} \times \frac{1 + r_P}{1 + r_B}$ ∎

### 해석 — 외우는 법

| 조건 | 결과 | 직관 |
|-----|------|------|
| 가격통화 금리 > 기준통화 금리 | Forward > Spot | 기준통화 **프리미엄** (선물에서 비싸짐) |
| 가격통화 금리 < 기준통화 금리 | Forward < Spot | 기준통화 **디스카운트** (선물에서 싸짐) |

> **핵심 원칙: 고금리 통화는 선물에서 절하됩니다.** 왜? 고금리 투자의 이점을 선물환율이 상쇄해야 차익거래가 불가능하기 때문입니다.

### 선물환율 계산 예시

**예시 1 (1년 선물):**

> ABE/DUB spot = 4.5671, ABE 금리 = 5%, DUB 금리 = 3%

$$\text{Forward}_{A/D} = 4.5671 \times \frac{1.05}{1.03} = 4.6558$$

확인: 선물 프리미엄 = 4.6558/4.5671 - 1 = **1.94%** ≈ 금리 차이 2% (근사)

ABE(가격통화) 금리 5% > DUB(기준통화) 금리 3% → DUB가 선물에서 비싸짐(프리미엄) ✓

**예시 2 (역산 — 현물환율 구하기):**

> USD/CHF 1년 선물 = 0.80, USD 금리 = 10%, CHF 금리 = 4%

$$\text{Spot} = \text{Forward} \times \frac{1 + r_{\text{CHF}}}{1 + r_{\text{USD}}} = 0.80 \times \frac{1.04}{1.10} = 0.7564$$

### 차익거래 예시 — 단계별

**조건:** ABE/DUB spot = 4.5671, ABE 금리 5%, DUB 금리 3%, 시장 선물환율 = 4.6000 (무차익 4.6558보다 **낮음**)

> 시장 선물환이 "너무 낮다" = ABE가 선물에서 충분히 절하되지 않음 = **ABE 투자가 유리**

**차익거래 과정:**
```
1. 1,000 DUB를 3%로 차입
2. 현물 환율로 ABE 매입: 1,000 × 4.5671 = 4,567.1 ABE
3. ABE를 5%로 투자: 4,567.1 × 1.05 = 4,795.45 ABE
4. 선물환으로 환전: 4,795.45 / 4.6000 = 1,042.49 DUB
5. 차입금 상환: 1,000 × 1.03 = 1,030 DUB
6. 순이익 = 1,042.49 - 1,030 = 12.49 DUB (무위험!)
```

**시장 반응으로 균형 회복:**
- ABE 현물 매입 → Spot ↑
- ABE 선물 매각 → Forward ↑
- 결국 무차익 선물환율(4.6558)로 수렴

### 선물환율 표기: 포인트와 퍼센트

**포인트 (Points):** 스팟 호가의 마지막 소수점 단위.

예: 스팟 2.3481 → 1 포인트 = 0.0001

> AUD/EUR spot = 0.7313, 1년 선물 = +3.5 points
> 선물환율 = 0.7313 + 0.00035 = **0.73165**

**퍼센트 (Percentage):**

> AUD/EUR spot = 0.7313, 120일 선물 = -0.062%
> 선물환율 = 0.7313 × (1 - 0.00062) = **0.7308**

### 선물 프리미엄/디스카운트

$$\text{선물 프리미엄/디스카운트} = \frac{\text{Forward}}{\text{Spot}} - 1$$

- 양수 → **프리미엄** (기준통화가 선물에서 비싸짐 = 절상 반영)
- 음수 → **디스카운트** (기준통화가 선물에서 싸짐 = 절하 반영)

> **중요:** 선물환율은 미래 현물환율의 **좋은 예측치가 아닙니다**. 선물환율은 차익거래 방지 조건만 반영합니다. "선물에서 유로가 프리미엄" ≠ "유로가 실제로 절상될 것"

## 핵심 개념 정리

### LOS 19.a 요약
- 교차환율 = 제3 통화를 통해 두 통화 간 환율 계산
- 핵심: 표기법을 대수적으로 배열하여 공통 통화 소거
- 역수 활용: A/B가 필요한데 B/A만 있으면 1/(B/A)

### LOS 19.b 요약
- 무차익거래 선물환율: Forward = Spot × (1+r_가격)/(1+r_기준)
- **고금리 통화 = 선물에서 절하** (차익거래 방지)
- 포인트: 스팟의 마지막 소수점 단위 가감
- 퍼센트: Spot × (1 ± %)
- 프리미엄/디스카운트 = Forward/Spot - 1
- 선물환율 ≠ 미래 현물 예측치 (무차익 조건만 반영)
- 차익거래: 시장 선물환 ≠ 무차익 선물환 → 차익 발생 → 시장 반응으로 수렴

## Module Quiz 19.1

**Q1.** IDR/USD = 2,400, NZD/USD = 1.6000. NZD/IDR 교차환율은?

IDR/NZD = IDR/USD ÷ NZD/USD = 2,400/1.6 = 1,500
NZD/IDR = 1/1,500 = **0.00067**

**정답: A**

---

**Q2.** USD/NZD = 0.3500, NZD/SEK = 0.3100. USD/SEK 교차환율은?

$$\text{USD/SEK} = \text{USD/NZD} \times \text{NZD/SEK} = 0.3500 \times 0.3100 = 0.1085$$

NZD가 소거됨. **정답: A (0.1085)**

---

**Q3.** CHF/GBP spot = 1.3050, 180일 선물 = -42.5 포인트. 선물환율은?

포인트: 1.3050의 마지막 소수점 = 0.0001. 42.5 포인트 = 0.00425

$$1.3050 - 0.00425 = 1.30075 \approx 1.3008$$

**정답: B (1.3008)**

---

**Q4.** NZD/USD spot = 1.4286, 180일 선물 = 1.3889. 이것이 의미하는 바는?

Forward/Spot = 1.3889/1.4286 = 0.972 < 1

Forward < Spot → USD(기준통화)가 선물에서 디스카운트 → USD 금리가 NZD보다 **낮음** (고금리 통화가 선물에서 절하하므로, NZD가 고금리)

**정답: A (미국 금리가 뉴질랜드보다 낮음)**

---

**Q5.** USD/GBP spot = 1.533, 180일 선물 = 1.508. USD의 선물 프리미엄/디스카운트는?

GBP/USD로 변환: spot = 1/1.533 = 0.6523, forward = 1/1.508 = 0.6631

$$\frac{0.6631}{0.6523} - 1 = +1.66\%$$

**정답: B (USD 1.66% 프리미엄)**

---

**Q6.** USD 연 금리 4%, SEK 연 금리 7%, SEK/USD spot = 9.5238. 1년 선물은?

$$\text{Forward} = 9.5238 \times \frac{1.07}{1.04} = 9.5238 \times 1.0288 = 9.7985$$

SEK(가격통화) 금리 7% > USD(기준통화) 금리 4% → Forward > Spot ✓
(고금리 SEK가 절하 = 더 많은 SEK 필요)

**정답: B (9.7985)**

---

**Q7.** USD 금리 10%, CHF 금리 4%, USD/CHF 1년 선물 = 0.80. 현물은?

$$\text{Spot} = 0.80 \times \frac{1 + 0.04}{1 + 0.10} = 0.80 \times \frac{1.04}{1.10} = 0.7564$$

**정답: A (0.7564)**
