# Reading 37: Financial Analysis Techniques (재무 분석 기법)

> **핵심 질문:** 재무제표 숫자들을 어떻게 해석 가능한 지표로 변환하는가? ROE 뒤에 숨겨진 진실을 어떻게 파악하는가?

---

## 왜 재무 분석 기법을 이해해야 하는가?

초등학생 비유: 시험 점수가 80점이라면 잘한 것인가? 반 평균이 90점이라면 낮은 것이고, 전년도 자신이 60점이었다면 크게 오른 것이다. 재무비율도 마찬가지 — **비교 없이는 의미가 없다**.

이 장에서는 재무제표 데이터를 의사결정에 유용한 형태로 변환하는 분석 도구들을 다룬다. 이 비율들을 단순히 계산하는 것보다, **각 비율이 무엇을 측정하는지, 높거나 낮을 때 무엇을 의미하는지**를 이해하는 것이 핵심이다.

## Module 37.1: 재무비율 소개

## LOS 37.a: 재무 분석 도구와 기법 — 용도와 한계

### 재무 분석 도구의 종류

재무제표 데이터를 분석 가능한 형태로 변환하는 주요 도구:

1. **비율 분석 (Ratio Analysis)**
2. **공통형 분석 (Common-Size Analysis)**
3. **그래프 분석 (Graphical Analysis)**
4. **회귀 분석 (Regression Analysis)**

### 1. 비율 분석 (Ratio Analysis)

비율은 데이터 간의 관계를 표현하는 도구로, 기업 내부 비교 및 기업 간 비교에 유용하다.

**비율 분석의 용도:**
- 미래 이익 및 현금흐름 예측
- 기업의 유연성 평가 (예상치 못한 상황에서도 성장·채무 이행 능력)
- 경영진 성과 평가
- 기업과 산업의 시간적 변화 평가
- 동종 업체와의 비교

**비율 분석의 한계:**
- 단독으로는 유용하지 않음 — 다른 기업 또는 과거 실적과 비교해야 의미 있음
- 서로 다른 회계 처리 방식(특히 미국 vs. 비미국 기업)으로 비교가 어려움
- 복수 산업에 걸친 기업은 비교 가능한 산업 비율 찾기 어려움
- 하나의 비율만으로는 결론 내릴 수 없음 — 모든 비율을 종합적으로 봐야 함
- 목표값 또는 비교값의 범위를 결정하기 어려움

> **실무 포인트:** 비율 분석가들이 동일 비율을 서로 다르게 계산하는 경우가 많다(예: 일부는 총부채 사용, 일부는 이자부 부채만 사용). 중요한 것은 **일관성 있게** 사용하는 것이다.

### 2. 공통형 분석 (Common-Size Analysis)

재무제표를 표준화하여 기업 간, 기간 간 비교를 용이하게 한다.

#### 수직 공통형 분석 (Vertical Common-Size)

$$\text{수직 공통형 재무상태표 비율} = \frac{\text{재무상태표 계정}}{\text{총자산}}$$

$$\text{수직 공통형 손익계산서 비율} = \frac{\text{손익계산서 계정}}{\text{매출액}}$$

### 3. 그래프 분석 (Graphical Analysis)

- **누적 막대 그래프 (Stacked Column/Bar Graph):** 구성 항목들의 연간 변화를 시각적으로 표현
- **꺾은선 그래프 (Line Graph):** 같은 데이터를 추세선으로 표현 — 매입채무 증가와 현금 감소 → 유동성 문제 경고

### 4. 회귀 분석 (Regression Analysis)

변수 간 관계를 식별하고 예측에 활용. 예: GDP와 매출의 관계를 이용한 매출 예측.

## Module 37.2: 재무비율 Part 1

## LOS 37.b: 활동성·유동성·지급능력·수익성 비율 계산 및 해석

### 활동성 비율 (Activity Ratios)

자산 관리 효율성 측정. **자산 이용률 비율** 또는 **운영 효율성 비율**이라고도 한다.

#### 1. 매출채권 회전율 (Receivables Turnover)

$$\text{매출채권 회전율} = \frac{\text{연간 매출액}}{\text{평균 매출채권}}$$

- 높을수록 → 채권 회수를 효과적으로 하거나, 엄격한 신용 조건
- 너무 높으면 → 신용 조건이 지나치게 엄격하여 매출 손실 가능
- 동종 업체 대비 낮으면 → 일부 채권이 회수 불가능하거나 징수 느림

$$\text{매출채권 회수 기간(DSO)} = \frac{365}{\text{매출채권 회전율}}$$

(일 기준 평균 고객 결제 소요 일수)

#### 2. 재고자산 회전율 (Inventory Turnover)

$$\text{재고자산 회전율} = \frac{\text{매출원가(COGS)}}{\text{평균 재고자산}}$$

> **주의:** 분자에 매출액이 아닌 **매출원가(COGS)**를 사용한다.

- 높을수록 → 효과적인 재고 관리; 단, 너무 높으면 재고 부족으로 매출 손실
- 낮을수록 → 일부 재고가 진부화되거나 판매 부진

$$\text{재고 보유 기간(Days of Inventory on Hand)} = \frac{365}{\text{재고자산 회전율}}$$

#### 3. 매입채무 회전율 (Payables Turnover)

$$\text{매입채무 회전율} = \frac{\text{매출원가(COGS)}}{\text{평균 매입채무}}$$

> 일부 분석가는 분자에 **구매액(Purchases)**을 사용. 계산: 구매액 = 기말재고 − 기초재고 + COGS

$$\text{매입채무 지급 기간} = \frac{365}{\text{매입채무 회전율}}$$

- 높을수록 → 공급업체 신용 조건을 충분히 활용하지 않거나, 조기 지급 할인 활용
- 낮을수록 → 단기 유동성 문제 가능성 또는 유리한 조건으로 지급 연장

#### 4. 총자산 회전율 (Total Asset Turnover)

$$\text{총자산 회전율} = \frac{\text{매출액}}{\text{평균 총자산}}$$

- 제조업(자본집약): 회전율 약 1 내외
- 소매업: 회전율 약 10 내외
- 낮을수록 → 자산 과잉 또는 비효율적 활용
- 높을수록 → 자산 과소 또는 노후 자산

#### 5. 고정자산 회전율 (Fixed Asset Turnover)

$$\text{고정자산 회전율} = \frac{\text{매출액}}{\text{평균 순고정자산(Net Fixed Assets)}}$$

> 순고정자산 = 고정자산 취득원가 − 누적감가상각. 최근 취득 자산이 많을수록 회전율 낮음.

#### 6. 운전자본 회전율 (Working Capital Turnover)

$$\text{운전자본 회전율} = \frac{\text{매출액}}{\text{평균 운전자본}}$$

운전자본 = 유동자산 − 유동부채

운전자본이 매우 낮거나 음수인 경우 이 비율의 변동성이 크고 해석이 어렵다.

### 활동성 비율 요약표

| 비율 | 공식 | 높을 때 | 낮을 때 |
|------|------|--------|--------|
| 매출채권 회전율 | 매출액 / 평균매출채권 | 효과적 징수 또는 엄격 신용 | 느린 징수, 손상 채권 가능 |
| DSO | 365 / 매출채권 회전율 | 느린 회수 | 빠른 회수 |
| 재고 회전율 | COGS / 평균재고 | 효과적 관리 또는 재고 부족 | 진부화, 판매 부진 |
| 재고 보유 기간 | 365 / 재고 회전율 | 재고 과잉 | 재고 부족 |
| 매입채무 회전율 | COGS / 평균매입채무 | 조기 지급 | 유동성 문제 또는 유리 조건 |
| 매입채무 지급 기간 | 365 / 매입채무 회전율 | 지급 연장 | 조기 지급 |
| 총자산 회전율 | 매출액 / 평균총자산 | 효율적 활용 | 자산 과잉 또는 비효율 |

### 유동성 비율 (Liquidity Ratios)

단기 채무 상환 능력 측정.

#### 1. 유동비율 (Current Ratio)

$$\text{유동비율} = \frac{\text{유동자산}}{\text{유동부채}}$$

- 1 미만 → 마이너스 운전자본, 유동성 위기 가능성

#### 2. 당좌비율 (Quick Ratio)

$$\text{당좌비율} = \frac{\text{현금} + \text{단기유가증권} + \text{매출채권}}{\text{유동부채}}$$

재고자산 및 기타 비유동성 자산을 제외한 더 엄격한 유동성 지표.

#### 3. 현금비율 (Cash Ratio)

$$\text{현금비율} = \frac{\text{현금} + \text{단기유가증권}}{\text{유동부채}}$$

가장 보수적인 유동성 지표.

#### 4. 방어 기간 비율 (Defensive Interval Ratio)

$$\text{방어 기간 비율} = \frac{\text{현금} + \text{단기유가증권} + \text{매출채권}}{\text{일평균 영업지출}}$$

현재의 유동 자산으로 평균 영업지출을 몇 일 동안 감당할 수 있는지 나타냄.

> 일평균 영업지출: COGS + SG&A + R&D 등 현금 지출. 비현금 항목(감가상각)은 제외.

#### 5. 현금 전환 주기 (Cash Conversion Cycle, CCC)

$$\text{CCC} = \text{매출채권 회수 기간(DSO)} + \text{재고 보유 기간} - \text{매입채무 지급 기간}$$

**의미:** 현금을 재고에 투입한 후, 매출을 통해 다시 현금으로 회수하는 데 걸리는 시간.

- **CCC가 높을수록:** 자본이 영업 프로세스에 오래 묶임 → 단기 자금 조달 필요
- **CCC가 낮을수록 또는 음수이면:** 고객에게 현금을 받기 전에 공급업체에 지급하지 않아도 됨 → 영업이 다른 부분의 자금조달 역할

## Module 37.3: 재무비율 Part 2

### 지급능력 비율 (Solvency Ratios)

장기 채무 상환 능력과 재무 레버리지 측정.

#### 부채 비율 계열 (Balance Sheet Based)

**1. 부채 대 자기자본 비율 (Debt-to-Equity Ratio)**

$$\text{부채 대 자기자본} = \frac{\text{총부채}}{\text{총자기자본}}$$

> 총부채 = 이자부 장단기 부채 (리스는 이자부이지만 CFA 레벨 1에서는 제외하는 것에 주의)

**2. 부채 대 자본 비율 (Debt-to-Capital Ratio)**

$$\text{부채 대 자본} = \frac{\text{총부채}}{\text{총부채} + \text{총자기자본}}$$

자본 구조에서 부채가 차지하는 비율.

**3. 부채 대 자산 비율 (Debt-to-Assets Ratio)**

$$\text{부채 대 자산} = \frac{\text{총부채}}{\text{총자산}}$$

**4. 재무 레버리지 비율 (Financial Leverage Ratio)**

$$\text{재무 레버리지} = \frac{\text{평균 총자산}}{\text{평균 총자기자본}}$$

1에 가까울수록 자기자본으로 자산을 조달; 1 초과 시 부채 활용 증가.

#### 커버리지 비율 계열 (Income Statement Based)

**5. 이자보상비율 (Interest Coverage Ratio)**

$$\text{이자보상비율} = \frac{\text{EBIT}}{\text{이자비용}}$$

낮을수록 → 채무 상환 어려움.

**6. 부채 대 EBITDA (Debt-to-EBITDA)**

$$\text{부채 대 EBITDA} = \frac{\text{총부채}}{\text{EBITDA}}$$

EBITDA는 영업 현금흐름의 근사치 — 현재 총부채를 상환하는 데 몇 년이 걸리는지.

**7. 고정비용 커버리지 비율 (Fixed Charge Coverage Ratio)**

$$\text{고정비용 커버리지} = \frac{\text{EBIT} + \text{리스비용}}{\text{이자비용} + \text{리스비용}}$$

리스 자산 비중이 높은 기업(항공사 등)에게 특히 유용.

### 수익성 비율 (Profitability Ratios)

전반적 수익 창출 능력 측정.

#### 마진 비율

**1. 매출총이익률 (Gross Profit Margin)**

$$\text{매출총이익률} = \frac{\text{매출총이익}}{\text{매출액}}$$

**2. 영업이익률 (Operating Profit Margin)**

$$\text{영업이익률} = \frac{\text{영업이익(EBIT)}}{\text{매출액}}$$

**3. 세전이익률 (Pretax Margin)**

$$\text{세전이익률} = \frac{\text{EBT}}{\text{매출액}}$$

**4. 순이익률 (Net Profit Margin)**

$$\text{순이익률} = \frac{\text{순이익}}{\text{매출액}}$$

> 계속 사업 순이익 기준 사용 권장 (중단 사업 항목 제외 — 미래에 영향 없음).

#### 수익률 비율

**5. 총자산이익률 (Return on Assets, ROA)**

$$\text{ROA} = \frac{\text{순이익}}{\text{평균 총자산}}$$

단, 이자비용은 순이익에서 차감되어 있지만 총자산에는 부채 자금도 포함되어 있어 약간 오해의 소지. 이를 보정한 식:

$$\text{ROA (조정)} = \frac{\text{순이익} + \text{이자비용} \times (1 - \text{세율})}{\text{평균 총자산}}$$

영업 ROA:

$$\text{영업 ROA} = \frac{\text{EBIT}}{\text{평균 총자산}}$$

**6. 투하자본이익률 (Return on Invested Capital, ROIC)**

$$\text{ROIC} = \frac{\text{세후 영업이익}}{\text{평균 장기 자본}}$$

장기 자본 = 장기부채 + 우선주 + 보통주 (운전자본 제외).

**7. 자기자본이익률 (Return on Equity, ROE)**

$$\text{ROE} = \frac{\text{순이익}}{\text{평균 총자기자본}}$$

**8. 보통주자본이익률 (Return on Common Equity)**

$$\text{보통주자본이익률} = \frac{\text{순이익} - \text{우선주 배당}}{\text{평균 보통주자본}}$$

---

> **예제: 비율 계산 — Sedgwick Company**

> **풀이:**
>
> Sedgwick Company (현재 연도 / 전년도):
> - 유동자산: $620 / $580
> - 유동부채: $325 / $275
> - 총자산: $2,060 / $1,940
> - 매출액: $4,000
> - 순이익: $200
> - 이자부 부채: 장기부채 $610 + 단기부채 $160 + 유동성 장기부채 $55 = $825
> - 보통주자본: $1,020 / $880
>
> **유동비율:**
> $$\text{유동비율} = \frac{620}{325} = 1.9$$
>
> **총자산 회전율:**
> $$\text{총자산 회전율} = \frac{4,000}{(2,060 + 1,940)/2} = \frac{4,000}{2,000} = 2.0\text{ 회}$$
>
> **순이익률:**
> $$\text{순이익률} = \frac{200}{4,000} = 5.0\%$$
>
> **보통주자본이익률:**
> $$\text{보통주자본이익률} = \frac{200}{(1,020 + 880)/2} = \frac{200}{950} = 21.1\%$$
>
> **부채 대 자기자본:**
> $$\text{D/E} = \frac{825}{1,020} = 80.9\%$$

## Module 37.4: DuPont 분석

## LOS 37.d: ROE의 DuPont 분해

### DuPont 분석이란?

ROE를 구성 요소로 분해하여 레버리지, 수익 마진, 자산 회전율이 주주 수익에 미치는 영향을 파악하는 방법.

> **비유:** ROE라는 결과물의 '레시피'를 분해하여, 어떤 재료가 ROE를 높이거나 낮추는지 파악한다.

### 3단계 DuPont (Original DuPont)

**출발:**

$$\text{ROE} = \frac{\text{순이익}}{\text{평균 자기자본}}$$

**분해 Step 1:** 분자·분모에 평균 총자산을 곱하고 정리:

$$\text{ROE} = \frac{\text{순이익}}{\text{평균 총자산}} \times \frac{\text{평균 총자산}}{\text{평균 자기자본}} = \text{ROA} \times \text{재무 레버리지}$$

**분해 Step 2:** ROA를 추가로 분해:

$$\text{ROE} = \frac{\text{순이익}}{\text{매출액}} \times \frac{\text{매출액}}{\text{평균 총자산}} \times \frac{\text{평균 총자산}}{\text{평균 자기자본}}$$

$$\boxed{\text{ROE} = \text{순이익률} \times \text{총자산 회전율} \times \text{재무 레버리지}}$$

**해석:**
- ROE가 낮다면: 순이익률 낮음 **또는** 자산 효율성 낮음 **또는** 레버리지 부족
- ROE가 높다면: 높은 레버리지의 결과일 수도 있어 위험 분석 필요

> **예제: 원래 DuPont 분해 — Staret, Inc.**

> | | 20X3 | 20X4 | 20X5 |
> |--|------|------|------|
> | 순이익 | 21.5 | 22.3 | 21.9 |
> | 매출액 | 305 | 350 | 410 |
> | 평균 자기자본 | 119 | 124 | 126 |
> | 평균 총자산 | 230 | 290 | 350 |
>
> **ROE 계산:**
> - 20X3: 21.5/119 = **18.1%**
> - 20X4: 22.3/124 = **18.0%**
> - 20X5: 21.9/126 = **17.4%**
>
> **DuPont 분해:**
>
> | | 순이익률 | 총자산 회전율 | 재무 레버리지 |
> |--|---------|------------|------------|
> | 20X3 | 7.0% | 1.33 | 1.93 |
> | 20X4 | 6.4% | 1.21 | 2.34 |
> | 20X5 | 5.3% | 1.17 | 2.78 |
>
> **분석:** ROE는 거의 일정하게 유지되었으나, 순이익률과 자산 회전율이 모두 하락. 이를 레버리지 증가가 상쇄함. 분석가는 마진 하락 원인(가격 압력 또는 비용 증가)을 조사해야 하며, 증가한 부채로 인한 위험 상승에 주목해야 함.

### 5단계 확장 DuPont (Extended DuPont)

순이익률을 3개 요소로 추가 분해:

$$\text{ROE} = \frac{\text{순이익}}{\text{EBT}} \times \frac{\text{EBT}}{\text{EBIT}} \times \frac{\text{EBIT}}{\text{매출액}} \times \frac{\text{매출액}}{\text{평균 총자산}} \times \frac{\text{평균 총자산}}{\text{평균 자기자본}}$$

$$\boxed{\text{ROE} = \text{세금 부담} \times \text{이자 부담} \times \text{EBIT 마진} \times \text{자산 회전율} \times \text{재무 레버리지}}$$

| 구성 요소 | 공식 | 의미 |
|---------|------|------|
| **세금 부담 (Tax Burden)** | 순이익 / EBT | = (1 − 실효세율). 높을수록 낮은 세율 |
| **이자 부담 (Interest Burden)** | EBT / EBIT | 이자비용이 클수록 낮아짐 |
| **EBIT 마진** | EBIT / 매출액 | 영업 수익성 |
| **자산 회전율** | 매출액 / 평균 총자산 | 자산 효율성 |
| **재무 레버리지** | 평균 총자산 / 평균 자기자본 | 레버리지 |

> **주의:** 레버리지가 높아질수록 이자 부담이 높아져(비율 감소), 레버리지의 긍정적 효과를 상쇄할 수 있다. 세금은 항상 ROE를 낮추는 방향으로 작용한다.

> **예제: 확장 DuPont 분석 — Company A vs. Company B**

> | | Company A | Company B |
> |--|---------|---------|
> | 매출액 | $500 | $900 |
> | EBIT | 35 | 100 |
> | 이자비용 | 5 | 0 |
> | EBT | 30 | 100 |
> | 세금 | 10 | 40 |
> | 순이익 | 20 | 60 |
> | 평균 총자산 | 250 | 300 |
> | 평균 자기자본 | 150 | 250 |
>
> **ROE:**
> - Company A: 20/150 = **13.3%**
> - Company B: 60/250 = **24.0%**
>
> **5단계 분해:**
>
> | 요소 | Company A | Company B |
> |------|---------|---------|
> | EBIT 마진 | 35/500 = 7.0% | 100/900 = 11.1% |
> | 총자산 회전율 | 500/250 = 2.0 | 900/300 = 3.0 |
> | 이자 부담 | 30/35 = 85.7% | 100/100 = 100% |
> | 재무 레버리지 | 250/150 = 1.67 | 300/250 = 1.20 |
> | 세금 부담 | 20/30 = 66.7% | 60/100 = 60.0% |
>
> **결론:** Company B의 ROE가 높은 이유는 더 높은 EBIT 마진과 자산 회전율 때문이다. Company B는 레버리지가 낮고 이자 부담이 없으며, 세금 부담은 더 높지만 영업 효율성이 훨씬 우수하다.

## Module 37.5: 산업별 특수 비율

## LOS 37.e: 산업별 재무 분석에 사용되는 특수 비율

비율의 중요성은 산업마다 다르다. 일부 산업에서는 다른 산업에는 관련이 없는 특수 비율을 계산한다.

### 산업별 주요 특수 비율

| 산업 | 주요 지표 | 의미 |
|------|---------|------|
| **서비스·컨설팅** | 직원 1인당 순이익, 직원 1인당 매출 | 인적 자원 활용 효율 |
| **음식점·소매업** | 동일 점포 매출 성장률 (Same-Store Sales Growth) | 신규 점포 효과 제거 후 기존 점포 성과 |
| **소매업** | 매장 면적당 매출 (Sales per Square Foot) | 매장 공간 효율 |
| **호텔업** | 평균 객실 단가 (Average Daily Rate, ADR), 객실 점유율 (Occupancy Rate) | 가격 경쟁력, 수요 |
| **구독 서비스 (스트리밍 등)** | 사용자 1인당 평균 수익 (Average Revenue per User, ARPU) | 고객 가치 |

### 금융 기관 관련 비율

| 비율 | 설명 |
|------|------|
| **자본 적정성 (Capital Adequacy)** | 운영·재무 리스크 대비 자기자본 비율. 규제 기관이 모니터링. VaR(Value at Risk): 특정 확률로 초과하지 않을 최대 손실 추정치 |
| **지급 준비율 요건** | 은행의 중앙은행 지준에 대한 특정 부채 비율 최소 기준 |
| **유동자산 요건 (Liquid Asset Requirement)** | 은행의 유동자산 대 특정 부채 비율 |
| **순이자 마진 (Net Interest Margin)** | 이자 수익 / 이자 수익 자산. 금융 기관의 핵심 수익성 지표 |

### 사업 위험 측정 (Business Risk)

변동 계수(Coefficient of Variation, CV)는 규모 조정된 변동성 지표:

$$\text{CV}_\text{매출} = \frac{\text{매출의 표준편차}}{\text{평균 매출}}$$

$$\text{CV}_\text{영업이익} = \frac{\text{영업이익의 표준편차}}{\text{평균 영업이익}}$$

$$\text{CV}_\text{순이익} = \frac{\text{순이익의 표준편차}}{\text{평균 순이익}}$$

기업 규모가 다르더라도 위험의 상대적 크기를 비교 가능.

## LOS 37.f: 비율 분석을 이용한 이익 예측 (Earnings Forecasting)

비율 분석은 미래 재무제표 추정치(pro forma financial statements)를 만드는 데 활용된다.

**기본 프로세스:**
1. 다음 기간 매출액 추정
2. 공통형 손익계산서의 비율(COGS/매출, SG&A/매출 등)을 적용하여 비용 추정
3. 영업이익률을 적용하여 영업이익 추정

**불확실성 분석 방법:**

| 방법 | 설명 |
|------|------|
| **민감도 분석 (Sensitivity Analysis)** | "만약 매출이 5%가 아닌 3% 증가하면?" 형태의 What-if 분석 |
| **시나리오 분석 (Scenario Analysis)** | 특정 가정 세트(시나리오)에 기반한 결과 범위 분석 |
| **시뮬레이션 (Simulation)** | 핵심 변수에 확률 분포를 부여하고 컴퓨터로 반복 추출하여 결과 분포 생성 |

### Module Quiz 37.4, 37.5

**Q1.** 전통적 DuPont 공식으로 표현한 ROE는?
A. (순이익률)(이자 요소)(지급능력 비율)
B. (순이익률)(총자산 회전율)(세금 유보율)
C. (순이익률)(총자산 회전율)(재무 레버리지 승수)

**풀이:** 3단계 DuPont: ROE = 순이익률 × 총자산 회전율 × 재무 레버리지.
**정답: C**

---

**Q2.** 다음 중 ROE를 가장 정확하게 표현하지 않는 식은?
A. (순이익률)(자기자본 회전율)
B. (순이익률)(총자산 회전율)(자산/자기자본)
C. (ROA)(이자 부담)(세금 유보율)

**풀이:** (ROA)(이자 부담)(세금 유보율)은 DuPont 모형이 아니다.
**정답: C**

**Q3.** 변동 계수(CV)에 대한 설명 중 가장 부정확한 것은?
A. 손익계산서 변동 계수는 통화 단위당 변동성을 나타냄
B. 변동 계수는 평균값을 표준편차로 나누어 계산
C. 변동 계수는 규모 조정된 변동성 지표

**풀이:** 변동 계수는 표준편차를 평균으로 나눈 값이다(평균을 표준편차로 나누는 것이 아님).
**정답: B**

**Q4.** 향후 3년간 이익을 예측하는 분석가가 가장 하지 않을 것은?
A. 예측 기간 동안 주요 재무비율이 유지된다고 가정
B. 공통형 재무제표를 사용하여 비용을 순이익의 비율로 추정

## 핵심 개념 정리

### LOS 37.a 요약
- 재무 분석 도구: 비율 분석, 공통형 분석, 그래프 분석, 회귀 분석
- 수직 공통형: 손익계산서 = 매출액 대비 %, 재무상태표 = 총자산 대비 %
- 수평 공통형: 기준 연도 = 1.0으로 표준화
- 비율의 한계: 단독으로 무의미, 비교 대상 필요, 산업 차이, 단일 비율 부족

### LOS 37.b 요약

**활동성 비율:**

| 비율 | 공식 |
|------|------|
| 매출채권 회전율 | 매출액 / 평균 매출채권 |
| DSO | 365 / 매출채권 회전율 |
| 재고 회전율 | COGS / 평균 재고 |
| 재고 보유 기간 | 365 / 재고 회전율 |
| 매입채무 회전율 | COGS / 평균 매입채무 |
| 매입채무 지급 기간 | 365 / 매입채무 회전율 |
| 총자산 회전율 | 매출액 / 평균 총자산 |
| 고정자산 회전율 | 매출액 / 평균 순고정자산 |
| 운전자본 회전율 | 매출액 / 평균 운전자본 |

**유동성 비율:**

| 비율 | 공식 |
|------|------|
| 유동비율 | 유동자산 / 유동부채 |
| 당좌비율 | (현금+단기유가증권+매출채권) / 유동부채 |
| 현금비율 | (현금+단기유가증권) / 유동부채 |
| 방어 기간 비율 | (현금+단기유가증권+매출채권) / 일평균 지출 |
| CCC | DSO + 재고 보유 기간 − 매입채무 지급 기간 |

**지급능력 비율:**

| 비율 | 공식 |
|------|------|
| 부채 대 자기자본 | 총부채 / 총자기자본 |
| 부채 대 자본 | 총부채 / (총부채 + 총자기자본) |
| 부채 대 자산 | 총부채 / 총자산 |
| 재무 레버리지 | 평균 총자산 / 평균 총자기자본 |
| 이자보상비율 | EBIT / 이자비용 |
| 부채 대 EBITDA | 총부채 / EBITDA |
| 고정비용 커버리지 | (EBIT+리스비용) / (이자비용+리스비용) |

**수익성 비율:**

| 비율 | 공식 |
|------|------|
| 매출총이익률 | 매출총이익 / 매출액 |
| 영업이익률 | EBIT / 매출액 |
| 세전이익률 | EBT / 매출액 |
| 순이익률 | 순이익 / 매출액 |
| ROA | 순이익 / 평균 총자산 |
| ROE | 순이익 / 평균 총자기자본 |
| 보통주자본이익률 | (순이익-우선주 배당) / 평균 보통주자본 |

### LOS 37.c 요약
- 여러 비율을 종합적으로 분석해야 올바른 결론 도출 가능
- ROE 상승이 레버리지 증가에 의한 것인지, 마진 또는 효율성 개선에 의한 것인지 구별 중요

### LOS 37.d 요약

$$\text{3단계 DuPont: ROE} = \text{순이익률} \times \text{총자산 회전율} \times \text{재무 레버리지}$$

$$\text{5단계 DuPont: ROE} = \text{세금 부담} \times \text{이자 부담} \times \text{EBIT 마진} \times \text{자산 회전율} \times \text{레버리지}$$

### LOS 37.e 요약
- 산업별 특수 비율: 소매(동일 점포 매출, 면적당 매출), 호텔(ADR, 점유율), 구독(ARPU), 금융(자본 적정성, 순이자 마진)
- 사업 위험 측정: CV = 표준편차 / 평균 (규모 조정 변동성)

### LOS 37.f 요약
- 비율 분석 + 매출 성장 가정으로 pro forma 재무제표 작성
- 불확실성 분석: 민감도 분석, 시나리오 분석, 시뮬레이션
