## LOS 4.a: 기댓값, 분산, 표준편차

### 왜 기댓값이 필요한가?

내년 주가가 얼마일까요? 확실한 답은 없습니다. 대신 여러 시나리오와 각각의 확률을 생각해봅시다:
- 경기 호황이면 → 주가 20% 상승 (확률 30%)
- 경기 보통이면 → 주가 12% 상승 (확률 50%)
- 경기 침체면 → 주가 5% 상승 (확률 20%)

"평균적으로 기대되는 수익률은?" 각 결과에 확률을 곱해서 더하면 됩니다.

동전으로 생각하면 더 직관적입니다. 동전을 100번 던져 앞면(1점), 뒷면(0점)이라면: 총점 ≈ 50점, 1번당 평균 = 0.5 = P(앞면)×1 + P(뒷면)×0. **기댓값은 장기적으로 1번 시행할 때 기대되는 평균 결과**입니다.

$$E(X) = \sum_{i=1}^{n} P(x_i) \cdot x_i$$

**예제: Ron's Stores EPS**

| 확률 | EPS |
|------|-----|
| 10%  | £1.80 |
| 20%  | £1.60 |
| 40%  | £1.20 |
| 30%  | £1.00 |

$$E[\text{EPS}] = 0.10(1.80) + 0.20(1.60) + 0.40(1.20) + 0.30(1.00) = £1.28$$

### 분산과 표준편차 (확률 모형)

확률 모형에서의 분산은 기댓값으로부터의 확률 가중 제곱 편차입니다:

$$\text{Var}(X) = \sum P(x_i) \cdot [x_i - E(X)]^2, \quad \sigma(X) = \sqrt{\text{Var}(X)}$$

R03에서 표본분산은 분모 n-1이었는데, 확률 모형은 왜 n-1 보정이 없을까요? 확률 모형은 관측 데이터가 아니라 **미래의 가능한 결과와 확률을 정의한 것**이므로 추정이 아니라 정의입니다.

**예제: Stock A**

| 경제 상황 | 확률 | 수익률 |
|-----------|------|--------|
| 호황 | 30% | 20% |
| 보통 | 50% | 12% |
| 침체 | 20% | 5% |

$E(R_A) = 0.30(0.20) + 0.50(0.12) + 0.20(0.05) = 13\%$

$\text{Var} = 0.30(0.07)^2 + 0.50(-0.01)^2 + 0.20(-0.08)^2 = 0.00280$

$\sigma = \sqrt{0.00280} = 5.29\%$

## LOS 4.b: 확률 나무 (Probability Tree)

### 왜 확률 나무가 필요한가?

보드게임에서 주사위를 두 번 던져 결과에 따라 다른 경로를 갑니다. 가능한 모든 경로를 나무 모양으로 그리면 어떤 결과가 나올 수 있는지, 각 결과의 확률이 한눈에 보입니다.

확률 나무의 각 경로 확률은 곱셈 규칙에서 나옵니다: $P(A \cap C) = P(A) \times P(C|A)$. 모든 최종 결과의 확률 합은 반드시 1입니다.

**예제: EPS 확률 나무**

```
                         ┌→ 좋은 실적 (P=0.60): EPS=$1.80  → 0.30×0.60 = 0.18
    ┌→ 좋은 경제 (0.30) ──┤
    │                    └→ 나쁜 실적 (P=0.70): EPS=$1.70  → 0.30×0.70 = 0.42
────┤
    │                    ┌→ 좋은 실적 (P=0.60): EPS=$1.30  → 0.40×0.60 = 0.24
    └→ 나쁜 경제 (0.40) ──┤
                          └→ 나쁜 실적 (P=0.40): EPS=$1.00  → 0.40×0.40 = 0.16
```

$$E[\text{EPS}] = 0.18(1.80) + 0.42(1.70) + 0.24(1.30) + 0.16(1.00) = \$1.51$$

확률 합: 0.18 + 0.42 + 0.24 + 0.16 = 1.00 ✓

### 조건부 기댓값

새로운 정보가 도착하면 기댓값을 수정합니다. 예를 들어 관세 부과 결정이 나왔다면:

$$E(X) = P(\text{관세}) \cdot E(X|\text{관세}) + P(\text{비관세}) \cdot E(X|\text{비관세})$$

전체 기댓값 = 조건부 기댓값들의 확률 가중 평균. 이것이 **전체 기댓값 법칙**입니다.

## LOS 4.c: 베이즈 공식 (Bayes' Formula)

### 왜 베이즈 공식이 필요한가?

일기예보가 "내일 비 올 확률 30%"라고 합니다. 아침에 일어나보니 하늘이 잔뜩 흐립니다. 비 올 확률은 이제 30%보다 높아져야 합니다. **새로운 정보를 보고 원래 예측을 수정하는 것** — 이것이 베이즈 공식의 핵심입니다.

우리는 보통 원인 → 결과 방향으로 확률을 생각합니다: "비가 오면 땅이 젖을 확률 = 95%". 하지만 투자에서는 역방향이 필요할 때가 많습니다: "땅이 젖어 있는데, 비가 왔을 확률은?"

### 유도

결합확률의 두 가지 표현: $P(B|A) \cdot P(A) = P(A|B) \cdot P(B)$

양변을 $P(B)$로 나누면:

$$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$

하늘에서 떨어진 공식이 아니라, 조건부 확률의 정의를 두 방향으로 쓰고 연립한 것입니다.

| 항 | 이름 | 의미 |
|----|------|------|
| $P(A)$ | 사전 확률 (Prior) | 새 정보 이전의 A 확률 |
| $P(B\|A)$ | 우도 (Likelihood) | A가 사실일 때 B가 관찰될 확률 |
| $P(B)$ | 증거 확률 | B의 비조건부 확률 |
| $P(A\|B)$ | 사후 확률 (Posterior) | B를 관찰한 후 업데이트된 확률 |

### 예제

경기 호황 확률 60%(호황 시 상승 70%, 하락 30%), 부진 확률 40%(부진 시 상승 20%, 하락 80%). 주가가 상승했을 때, 경기가 호황이었을 확률은?

```
    ┌→ 호황 (0.60) → 상승: 0.42 / 하락: 0.18
────┤
    └→ 부진 (0.40) → 상승: 0.08 / 하락: 0.32
```

$$P(\text{호황}|\text{상승}) = \frac{0.42}{0.42 + 0.08} = \frac{0.42}{0.50} = 84\%$$

사전 확률 60%가 사후 확률 84%로 상승했습니다. 호황일 때 상승 확률(70%)이 부진일 때(20%)보다 높으므로, "상승"이라는 증거가 호황 가설을 강하게 지지합니다.

**만약 호황/부진 모두 상승 확률이 50%라면?** 사후 확률 = 사전 확률 = 60%. 정보가 가설을 구별하는 능력이 없으면 확률이 업데이트되지 않습니다.

## Module Quiz 4.1

**Q1.** 조건부 확률과 P(Y=1)=0.3, P(Y=2)=0.7이 주어졌을 때 X의 기댓값은?

| $x_i$ | $P(x_i \| Y=1)$ | $P(x_i \| Y=2)$ |
|--------|-----------------|-----------------|
| 0      | 0.2             | 0.1             |
| 5      | 0.4             | 0.8             |
| 10     | 0.4             | 0.1             |

$E(X|Y=1) = 0+2+4 = 6$, $E(X|Y=2) = 0+4+1 = 5$
$E(X) = 0.3(6) + 0.7(5) = 1.8 + 3.5 = 5.3$

**정답: B. 5.3**

**Q2.** EPS > $2 확률 40%. EPS>$2이면 상향 70%, EPS≤$2이면 상향 20%. 상향 관찰 시 EPS>$2일 확률은?

```
    ┌→ EPS > $2 (0.40) → 상향: 0.28
────┤
    └→ EPS ≤ $2 (0.60) → 상향: 0.12
```

$P(\text{EPS}>\$2 | \text{상향}) = 0.28/(0.28+0.12) = 70\%$

**정답: C. 70%**

## 핵심 개념 정리 (Key Concepts)

### LOS 4.a
$$E(X) = \sum P(x_i) x_i, \quad \text{Var}(X) = \sum P(x_i)[x_i - E(X)]^2$$
확률 모형은 n-1 보정 불필요

### LOS 4.b
확률 나무: 경로 확률 = 각 가지 확률의 곱. 모든 최종 경로 확률 합 = 1.
전체 기댓값 = 조건부 기댓값의 확률 가중 평균

### LOS 4.c
$$P(A|C) = \frac{P(C|A) \cdot P(A)}{P(C)}$$
새 증거가 가설을 구별하는 능력이 있어야 사후 확률이 업데이트됨
