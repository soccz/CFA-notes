# Reading 4: 확률 나무와 조건부 기댓값 (Probability Trees and Conditional Expectations)


## LOS 4.a: 기댓값, 분산, 표준편차

### 왜 기댓값이 필요한가?

내년 주가가 얼마일까요? 확실한 답은 없습니다. 대신 여러 시나리오와 각각의 확률을 생각해봅시다:
- 경기 호황이면 → 주가 20% 상승 (확률 30%)
- 경기 보통이면 → 주가 12% 상승 (확률 50%)
- 경기 침체면 → 주가 5% 상승 (확률 20%)

"평균적으로 기대되는 수익률은?" 각 결과에 확률을 곱해서 더하면 됩니다.

동전으로 생각하면 더 직관적입니다. 동전을 100번 던져 앞면(1점), 뒷면(0점)이라면: 총점 ≈ 50점, 1번당 평균 = 0.5 = P(앞면)×1 + P(뒷면)×0. **기댓값은 장기적으로 1번 시행할 때 기대되는 평균 결과**입니다.

$$E(X) = \sum_{i=1}^{n} P(x_i) \cdot x_i$$

투자에서 이게 왜 중요한가? 포트폴리오를 구성할 때 각 자산의 기대 수익률을 알아야 합니다. 기대 수익률이 없으면 "이 자산을 살 가치가 있는가?"라는 가장 기본적인 질문에 답할 수 없습니다.

**예제 1: Ron's Stores EPS**

| 확률 | EPS |
|------|-----|
| 10%  | £1.80 |
| 20%  | £1.60 |
| 40%  | £1.20 |
| 30%  | £1.00 |

$$E[\text{EPS}] = 0.10(1.80) + 0.20(1.60) + 0.40(1.20) + 0.30(1.00) = £1.28$$

해석: Ron's Stores가 이 시나리오를 무한히 반복할 수 있다면, 평균 EPS는 £1.28에 수렴합니다. 현재 주가가 £12라면 기대 P/E = 12/1.28 = 9.4배입니다.

**만약 확률 분포가 바뀌면?** 호황 확률(EPS=£1.80)이 10%에서 30%로 올라가고, 불황(EPS=£1.00)이 30%에서 10%로 내려가면:

$$E[\text{EPS}] = 0.30(1.80) + 0.20(1.60) + 0.40(1.20) + 0.10(1.00) = £1.44$$

기댓값이 £1.28에서 £1.44로 상승합니다. 같은 결과값이라도 확률 배분이 달라지면 기댓값이 변합니다. 이것이 경기 전망이 주가에 영향을 미치는 수학적 근거입니다.

**예제 2: 주사위 기댓값**

공정한 주사위 1개를 던져 나온 눈만큼 만 원을 받는 게임입니다.

$$E(X) = \frac{1}{6}(1+2+3+4+5+6) = 3.5\text{만 원}$$

참가비가 3만 원이라면? 기대 이득 = 3.5 - 3 = 0.5만 원 > 0이므로 참가하는 것이 유리합니다. 참가비 4만 원이라면? 기대 이득 = -0.5만 원이므로 불리합니다. 카지노 게임 대부분은 기대 이득이 음수로 설계되어 있습니다.

### 분산과 표준편차 (확률 모형)

기댓값만으로는 충분하지 않습니다. 두 투자가 기대 수익률 10%로 같아도 하나는 확실히 10%이고, 다른 하나는 -30%~+50% 사이에서 변동합니다. **분산은 결과가 기댓값 주위로 얼마나 흩어지는지를 측정합니다.**

확률 모형에서의 분산은 기댓값으로부터의 확률 가중 제곱 편차입니다:

$$\text{Var}(X) = \sum P(x_i) \cdot [x_i - E(X)]^2, \quad \sigma(X) = \sqrt{\text{Var}(X)}$$

R03에서 표본분산은 분모 n-1이었는데, 확률 모형은 왜 n-1 보정이 없을까요? 확률 모형은 관측 데이터가 아니라 **미래의 가능한 결과와 확률을 정의한 것**이므로 추정이 아니라 정의입니다. 표본분산의 n-1은 표본으로 모집단을 "추정"할 때 필요한 보정(자유도 손실)이지만, 확률 모형은 모집단 자체를 정의하므로 보정이 불필요합니다.

**예제 3: Stock A**

| 경제 상황 | 확률 | 수익률 |
|-----------|------|--------|
| 호황 | 30% | 20% |
| 보통 | 50% | 12% |
| 침체 | 20% | 5% |

$E(R_A) = 0.30(0.20) + 0.50(0.12) + 0.20(0.05) = 13\%$

$\text{Var} = 0.30(0.20-0.13)^2 + 0.50(0.12-0.13)^2 + 0.20(0.05-0.13)^2$
$= 0.30(0.0049) + 0.50(0.0001) + 0.20(0.0064) = 0.00147 + 0.00005 + 0.00128 = 0.00280$

$\sigma = \sqrt{0.00280} = 5.29\%$

해석: 기대 수익률 13%에서 위아래로 대략 5.29% 범위에 수익률이 집중됩니다.

**예제 4: Stock B (비교)**

| 경제 상황 | 확률 | 수익률 |
|-----------|------|--------|
| 호황 | 30% | 35% |
| 보통 | 50% | 8% |
| 침체 | 20% | -10% |

$E(R_B) = 0.30(0.35) + 0.50(0.08) + 0.20(-0.10) = 12.5\%$

$\text{Var} = 0.30(0.225)^2 + 0.50(-0.045)^2 + 0.20(-0.225)^2$
$= 0.30(0.0506) + 0.50(0.0020) + 0.20(0.0506) = 0.01519 + 0.00101 + 0.01013 = 0.02633$

$\sigma_B = \sqrt{0.02633} = 16.23\%$

Stock A: 기대 수익률 13%, σ = 5.29%. Stock B: 기대 수익률 12.5%, σ = 16.23%.

**Stock A가 수익률도 높고 위험도 낮습니다.** 이런 경우 Stock A가 Stock B를 "지배(dominate)"한다고 합니다. 분산이 없으면 이 판단을 할 수 없습니다.

**만약 Stock B의 기대 수익률이 20%라면?** 이제 수익률은 B가 높고 위험도 B가 높습니다. 지배 관계가 사라지고, 투자자의 위험 선호도에 따라 선택이 달라집니다. 이것이 "위험-수익 트레이드오프"의 시작입니다.

### 공분산과 상관계수 (확률 모형)

두 자산이 "같이 움직이는 정도"를 측정합니다. 포트폴리오를 구성할 때 핵심적인 정보입니다.

$$\text{Cov}(X,Y) = \sum P(s_i) \cdot [X(s_i) - E(X)] \cdot [Y(s_i) - E(Y)]$$

$$\rho_{XY} = \frac{\text{Cov}(X,Y)}{\sigma_X \cdot \sigma_Y}, \quad -1 \le \rho \le +1$$

왜 공분산 대신 상관계수를 쓸까요? 공분산은 단위가 "%²"처럼 직관적이지 않습니다. 상관계수는 -1~+1로 표준화되어 "얼마나 같이 움직이는지" 즉시 파악 가능합니다. +1이면 완벽히 같은 방향, -1이면 완벽히 반대, 0이면 선형 관계 없음.

**예제 5: 두 주식의 공분산**

| 시나리오 | 확률 | $R_A$ | $R_B$ |
|----------|------|-------|-------|
| 호황 | 0.30 | 20% | 30% |
| 보통 | 0.50 | 12% | 10% |
| 침체 | 0.20 | 5% | 0% |

$E(R_A) = 13\%$, $E(R_B) = 14\%$

$\text{Cov} = 0.30(0.07)(0.16) + 0.50(-0.01)(-0.04) + 0.20(-0.08)(-0.14)$
$= 0.00336 + 0.00020 + 0.00224 = 0.0058$

양수 → 호황일 때 둘 다 오르고, 침체일 때 둘 다 떨어지는 경향. 분산투자 효과가 제한적입니다.


## LOS 4.b: 확률 나무 (Probability Tree)

### 왜 확률 나무가 필요한가?

현실의 의사결정은 단계적입니다. 먼저 경기 상황이 결정되고, 그다음 기업 실적이 결정됩니다. 보드게임에서 주사위를 두 번 던져 결과에 따라 다른 경로를 갑니다. 가능한 모든 경로를 나무 모양으로 그리면 어떤 결과가 나올 수 있는지, 각 결과의 확률이 한눈에 보입니다.

투자에서 이것이 필수적인 이유: "경기 호황이면 이 회사 실적이 좋을 확률 70%, 나쁠 확률 30%지만, 경기 침체면 좋을 확률 20%, 나쁠 확률 80%"처럼 **조건에 따라 확률이 달라지는 상황**을 체계적으로 분석할 수 있습니다.

확률 나무의 각 경로 확률은 곱셈 규칙에서 나옵니다: $P(A \cap C) = P(A) \times P(C|A)$. 모든 최종 결과의 확률 합은 반드시 1입니다. 1이 아니면 어딘가 계산이 틀린 것이므로 **검증 도구**로도 유용합니다.

**예제 1: EPS 확률 나무**

```
                         ┌→ 좋은 실적 (P=0.60): EPS=1.80  → 0.30×0.60 = 0.18
    ┌→ 좋은 경제 (0.30) ──┤
    │                    └→ 나쁜 실적 (P=0.40): EPS=1.70  → 0.30×0.40 = 0.12
────┤
    │                    ┌→ 좋은 실적 (P=0.60): EPS=1.30  → 0.70×0.60 = 0.42
    └→ 나쁜 경제 (0.70) ──┤
                          └→ 나쁜 실적 (P=0.40): EPS=1.00  → 0.70×0.40 = 0.28
```

$$E[\text{EPS}] = 0.18(1.80) + 0.12(1.70) + 0.42(1.30) + 0.28(1.00) = 0.324 + 0.204 + 0.546 + 0.280 = \text{\$}1.354$$

확률 합: 0.18 + 0.12 + 0.42 + 0.28 = 1.00 ✓

**만약 경제 전망이 바뀌면?** 좋은 경제 확률이 0.30에서 0.60으로 올라가면:

$$E[\text{EPS}] = 0.36(1.80) + 0.24(1.70) + 0.24(1.30) + 0.16(1.00) = 0.648 + 0.408 + 0.312 + 0.160 = \text{\$}1.528$$

기대 EPS가 1.354에서 1.528로 상승합니다. 경기 전망 변화가 기업 가치 평가에 직접 반영되는 것입니다.

**예제 2: 신약 개발 의사결정 나무**

제약회사가 신약 개발 투자를 고려합니다:

```
                              ┌→ 대박 (P=0.30): 수익 500M → 0.60×0.30 = 0.18
    ┌→ 임상 성공 (0.60) ──────┤
    │                         ├→ 보통 (P=0.50): 수익 200M → 0.60×0.50 = 0.30
    │                         └→ 부진 (P=0.20): 수익 50M  → 0.60×0.20 = 0.12
────┤
    └→ 임상 실패 (0.40) ──────→ 수익 0M                    → 0.40
```

$E[\text{수익}] = 0.18(500) + 0.30(200) + 0.12(50) + 0.40(0) = 90 + 60 + 6 + 0 = \text{\$}156M$

개발 비용이 120M이라면 기대 순수익 = 36M > 0이므로 투자할 가치가 있습니다. 개발 비용이 200M이라면? 기대 순수익 = -44M이므로 투자하지 않는 것이 합리적입니다.

### 조건부 기댓값 (Conditional Expectation)

새로운 정보가 도착하면 기댓값을 수정합니다. 예를 들어 관세 부과 결정이 나왔다면:

$$E(X) = P(\text{관세}) \cdot E(X|\text{관세}) + P(\text{비관세}) \cdot E(X|\text{비관세})$$

전체 기댓값 = 조건부 기댓값들의 확률 가중 평균. 이것이 **전체 기댓값 법칙(Law of Total Expectation)**입니다.

**왜 이것이 투자에서 중요한가?** 매일 새로운 뉴스가 나옵니다. "금리 인상 결정" → 경기 침체 확률 상승 → 주식 기대 수익률 하향 조정. 조건부 기댓값은 **새로운 정보를 체계적으로 가격에 반영하는 도구**입니다.

**예제 3: 관세 시나리오**

- P(관세 부과) = 40%, 관세 부과 시 E(수익률) = 5%
- P(관세 미부과) = 60%, 관세 미부과 시 E(수익률) = 15%

$$E(R) = 0.40(5\%) + 0.60(15\%) = 2\% + 9\% = 11\%$$

관세 부과가 확정되면? E(R) = 5%로 하향 조정. 관세 미부과가 확정되면? E(R) = 15%로 상향 조정. **불확실성이 해소될 때마다 기댓값이 업데이트됩니다.**

만약 관세 부과 확률이 40%에서 70%로 올라가면(아직 확정은 아님):

$$E(R) = 0.70(5\%) + 0.30(15\%) = 3.5\% + 4.5\% = 8\%$$

기대 수익률이 11%에서 8%로 하락합니다. 확률만 변해도 기대 수익률이 움직입니다. 이것이 "루머에 의한 주가 변동"의 수학적 근거입니다.


## LOS 4.c: 베이즈 공식 (Bayes' Formula)

### 왜 베이즈 공식이 필요한가?

일기예보가 "내일 비 올 확률 30%"라고 합니다. 아침에 일어나보니 하늘이 잔뜩 흐립니다. 비 올 확률은 이제 30%보다 높아져야 합니다. **새로운 정보를 보고 원래 예측을 수정하는 것** — 이것이 베이즈 공식의 핵심입니다.

우리는 보통 원인 → 결과 방향으로 확률을 생각합니다: "비가 오면 땅이 젖을 확률 = 95%". 하지만 투자에서는 역방향이 필요할 때가 많습니다: "땅이 젖어 있는데, 비가 왔을 확률은?"

실제 투자 상황: 어떤 기업의 주가가 갑자기 20% 떨어졌습니다(관찰된 사건). 원인이 뭘까요? (a) 분기 실적 부진, (b) 업종 전체 하락, (c) 시장 공황. 각 원인의 사전 확률이 있고, "20% 하락"이라는 증거를 보고 각 원인의 확률을 업데이트하는 것이 베이즈 추론입니다.

### 유도

결합확률의 두 가지 표현: $P(B|A) \cdot P(A) = P(A|B) \cdot P(B)$

양변을 $P(B)$로 나누면:

$$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$

하늘에서 떨어진 공식이 아니라, 조건부 확률의 정의를 두 방향으로 쓰고 연립한 것입니다.

| 항 | 이름 | 의미 | 투자 예시 |
|----|------|------|----------|
| $P(A)$ | 사전 확률 (Prior) | 새 정보 이전의 A 확률 | "이 기업이 부도날 확률 2%" |
| $P(B\|A)$ | 우도 (Likelihood) | A가 사실일 때 B가 관찰될 확률 | "부도 기업 중 80%는 부채비율 300% 초과" |
| $P(B)$ | 증거 확률 | B의 비조건부 확률 | "전체 기업 중 부채비율 300% 초과 비율 = 10%" |
| $P(A\|B)$ | 사후 확률 (Posterior) | B를 관찰한 후 업데이트된 확률 | "부채비율 300% 초과인 기업이 부도날 확률" |

위 예시에 숫자를 대입하면:

$$P(\text{부도}|\text{고부채}) = \frac{0.80 \times 0.02}{0.10} = \frac{0.016}{0.10} = 16\%$$

사전 확률 2%가 사후 확률 16%로 8배 상승했습니다. "부채비율 300% 초과"라는 증거가 부도 가설을 강하게 지지하기 때문입니다.

### 예제 1: 경기 호황 추론

경기 호황 확률 60%(호황 시 상승 70%, 하락 30%), 부진 확률 40%(부진 시 상승 20%, 하락 80%). 주가가 상승했을 때, 경기가 호황이었을 확률은?

```
    ┌→ 호황 (0.60) → 상승: 0.42 / 하락: 0.18
────┤
    └→ 부진 (0.40) → 상승: 0.08 / 하락: 0.32
```

$$P(\text{호황}|\text{상승}) = \frac{0.42}{0.42 + 0.08} = \frac{0.42}{0.50} = 84\%$$

사전 확률 60%가 사후 확률 84%로 상승했습니다. 호황일 때 상승 확률(70%)이 부진일 때(20%)보다 높으므로, "상승"이라는 증거가 호황 가설을 강하게 지지합니다.

**만약 호황/부진 모두 상승 확률이 50%라면?**

$$P(\text{호황}|\text{상승}) = \frac{0.60 \times 0.50}{0.60 \times 0.50 + 0.40 \times 0.50} = \frac{0.30}{0.50} = 60\%$$

사후 확률 = 사전 확률 = 60%. 정보가 가설을 구별하는 능력이 없으면 확률이 업데이트되지 않습니다. 이것이 **정보의 가치**를 판단하는 기준입니다: 우도가 가설에 따라 다를 때만 정보가 유용합니다.

### 예제 2: 펀드매니저 실력 판단

- 사전 확률: 실력 있는 매니저 확률 = 20% (대부분은 시장을 못 이김)
- 실력 있는 매니저가 시장을 이길 확률 = 70%
- 실력 없는 매니저가 시장을 이길 확률 = 40% (운으로도 이길 수 있음)

이 매니저가 올해 시장을 이겼습니다. 실력이 있을 확률은?

$$P(\text{실력}|\text{시장 초과}) = \frac{0.70 \times 0.20}{0.70 \times 0.20 + 0.40 \times 0.80} = \frac{0.14}{0.14 + 0.32} = \frac{0.14}{0.46} = 30.4\%$$

1년 시장 초과 성과만으로는 실력 있는 매니저일 확률이 20%에서 30.4%로만 올라갑니다. 여전히 70%는 운일 수 있습니다. 이것이 "1년 성과로 매니저를 판단하지 마라"의 수학적 근거입니다.

**만약 3년 연속 시장을 이겼다면?** (독립 가정)

- 실력 있는 매니저가 3년 연속 이길 확률: 0.70^3 = 0.343$
- 실력 없는 매니저가 3년 연속 이길 확률: 0.40^3 = 0.064$

$$P(\text{실력}|3\text{년 초과}) = \frac{0.343 \times 0.20}{0.343 \times 0.20 + 0.064 \times 0.80} = \frac{0.0686}{0.0686 + 0.0512} = \frac{0.0686}{0.1198} = 57.3\%$$

3년 연속이면 57.3%로 올라갑니다. 증거가 누적될수록 사후 확률이 점점 사전 확률에서 벗어납니다. 이것이 트랙 레코드의 길이가 중요한 이유입니다.

### 예제 3: 매출 상향 조정

EPS > 2 확률 40%. EPS>2이면 상향 70%, EPS≤2이면 상향 20%. 상향 관찰 시 EPS>2일 확률은?

```
    ┌→ EPS > 2 (0.40) → 상향: 0.28
────┤
    └→ EPS ≤ 2 (0.60) → 상향: 0.12
```

$$P(\text{EPS}>\text{\$}2 | \text{상향}) = \frac{0.28}{0.28+0.12} = \frac{0.28}{0.40} = 70\%$$

사전 확률 40%가 사후 확률 70%로 대폭 상승. 애널리스트의 상향 조정은 높은 EPS의 강한 신호입니다.


## Module Quiz 4.1

**Q1.** 조건부 확률과 P(Y=1)=0.3, P(Y=2)=0.7이 주어졌을 때 X의 기댓값은?

| $x_i$ | $P(x_i \| Y=1)$ | $P(x_i \| Y=2)$ |
|--------|-----------------|-----------------|
| 0      | 0.2             | 0.1             |
| 5      | 0.4             | 0.8             |
| 10     | 0.4             | 0.1             |

$E(X|Y=1) = 0(0.2) + 5(0.4) + 10(0.4) = 0+2+4 = 6$
$E(X|Y=2) = 0(0.1) + 5(0.8) + 10(0.1) = 0+4+1 = 5$

전체 기댓값 법칙 적용:
$E(X) = P(Y=1) \cdot E(X|Y=1) + P(Y=2) \cdot E(X|Y=2) = 0.3(6) + 0.7(5) = 1.8 + 3.5 = 5.3$

**정답: B. 5.3**

**Q2.** EPS > 2 확률 40%. EPS>2이면 상향 70%, EPS≤2이면 상향 20%. 상향 관찰 시 EPS>2일 확률은?

```
    ┌→ EPS > 2 (0.40) → 상향: 0.28
────┤
    └→ EPS ≤ 2 (0.60) → 상향: 0.12
```

$P(\text{EPS}>\text{\$}2 | \text{상향}) = 0.28/(0.28+0.12) = 70\%$

**정답: C. 70%**


## 핵심 개념 정리 (Key Concepts)

### LOS 4.a
$$E(X) = \sum P(x_i) x_i, \quad \text{Var}(X) = \sum P(x_i)[x_i - E(X)]^2$$
확률 모형은 n-1 보정 불필요 (모집단 자체를 정의하므로)
기댓값은 확률 분포가 바뀌면 함께 변함 — 경기 전망 변화가 주가에 영향을 미치는 수학적 근거

### LOS 4.b
확률 나무: 경로 확률 = 각 가지 확률의 곱. 모든 최종 경로 확률 합 = 1.
전체 기댓값 = 조건부 기댓값의 확률 가중 평균 (전체 기댓값 법칙)
새로운 정보가 도착하면 조건부 기댓값으로 업데이트

### LOS 4.c
$$P(A|C) = \frac{P(C|A) \cdot P(A)}{P(C)}$$
새 증거가 가설을 구별하는 능력이 있어야 사후 확률이 업데이트됨
우도가 가설에 따라 같으면 → 정보 가치 없음 → 사후 = 사전
증거가 누적될수록 사후 확률의 업데이트 폭이 커짐 (트랙 레코드의 가치)
