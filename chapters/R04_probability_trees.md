# Reading 4: Probability Trees and Conditional Expectations

> **핵심 질문:** 불확실한 미래에서 어떻게 합리적인 기댓값을 계산하고, 새로운 정보가 들어왔을 때 확률을 어떻게 업데이트하는가?

---

## 왜 이 개념이 필요한가?

투자에서 미래는 불확실합니다. "내년 주가가 얼마일까?"라는 질문에 하나의 확실한 답은 없습니다. 대신 우리는 **여러 시나리오와 각각의 확률**을 생각합니다.

예를 들어:
- 경기 호황이면 → 주가 20% 상승 (확률 30%)
- 경기 보통이면 → 주가 12% 상승 (확률 50%)
- 경기 침체면 → 주가 5% 상승 (확률 20%)

이런 상황에서 "평균적으로 기대되는 수익률은?" 을 계산하는 것이 **기댓값(Expected Value)** 입니다.

그리고 새로운 정보(예: "정부가 관세를 부과했다")가 들어오면, 기존 확률을 어떻게 수정해야 할까요? 이것이 **베이즈 공식(Bayes' Formula)** 의 역할입니다.

---

## LOS 4.a: 기댓값, 분산, 표준편차

### 기댓값 (Expected Value)

**기댓값이란?** 확률 가중 평균(probability-weighted average)입니다. 각 결과에 해당 확률을 곱해서 더한 값입니다.

$$E(X) = \sum_{i=1}^{n} P(x_i) \cdot x_i = P(x_1)x_1 + P(x_2)x_2 + \cdots + P(x_n)x_n$$

**왜 이런 공식이 나왔는가?**

동전을 100번 던진다고 하면:
- 앞면(1점): 50번 기대
- 뒷면(0점): 50번 기대

100번 던진 총 점수 = 50 × 1 + 50 × 0 = 50점
1번당 평균 = 50/100 = 0.5점 = P(앞면) × 1 + P(뒷면) × 0

이것이 기댓값의 본질입니다. 장기적으로 1번 시행할 때 기대되는 평균 결과입니다.

---

### 예제: EPS 기댓값 계산

Ron's Stores의 주당순이익(EPS) 확률 분포:

| 확률 | EPS |
|------|-----|
| 10%  | £1.80 |
| 20%  | £1.60 |
| 40%  | £1.20 |
| 30%  | £1.00 |

$$E[\text{EPS}] = 0.10(1.80) + 0.20(1.60) + 0.40(1.20) + 0.30(1.00)$$
$$= 0.18 + 0.32 + 0.48 + 0.30 = £1.28$$

---

### 분산과 표준편차 (확률 모형에서)

확률 모형에서의 분산은 기댓값으로부터의 **확률 가중 제곱 편차**입니다:

$$\text{Var}(X) = \sum_{i=1}^{n} P(x_i) \cdot [x_i - E(X)]^2$$

표준편차는 분산의 양의 제곱근:
$$\sigma(X) = \sqrt{\text{Var}(X)}$$

**왜 분모에 n-1을 쓰지 않는가?**

표본 데이터(과거 관측치)에서는 모집단 평균을 추정할 때 편향이 생겨 n-1로 보정합니다. 하지만 **확률 모형**은 관측치가 없습니다. 미래의 가능한 결과와 각각의 확률을 정의한 것이므로, 확률 가중치로 전체 분포를 직접 표현합니다. 따라서 n-1 보정이 필요 없습니다.

---

### 예제: 확률 모형에서의 분산, 표준편차

Stock A의 수익률 확률 분포:

| 경제 상황 | 확률 | 수익률 $R_A$ | $P \times R_A$ | $R_A - E(R_A)$ | $[R_A - E(R_A)]^2$ | $P \times [R_A - E(R_A)]^2$ |
|-----------|------|-------------|----------------|-----------------|---------------------|------------------------------|
| 호황 (Boom) | 30%  | 20% | 0.06 | +7% | 0.0049 | 0.00147 |
| 보통 (Normal) | 50%  | 12% | 0.06 | -1% | 0.0001 | 0.00005 |
| 침체 (Slow) | 20%  | 5%  | 0.01 | -8% | 0.0064 | 0.00128 |
| **합계** | 100% | | **0.13** | | | **0.00280** |

**계산 과정:**

1. 기댓값:
$$E(R_A) = 0.30(0.20) + 0.50(0.12) + 0.20(0.05) = 0.06 + 0.06 + 0.01 = 0.13 = 13\%$$

2. 분산:
$$\text{Var}(R_A) = 0.30(0.20-0.13)^2 + 0.50(0.12-0.13)^2 + 0.20(0.05-0.13)^2$$
$$= 0.30(0.0049) + 0.50(0.0001) + 0.20(0.0064) = 0.00280$$

3. 표준편차:
$$\sigma(R_A) = \sqrt{0.00280} = 0.0529 = 5.29\%$$

## LOS 4.b: 확률 나무 (Probability Tree)

### 확률 나무란?

**확률 나무**는 여러 단계의 사건을 시각적으로 표현한 도구입니다. 각 가지(branch)는 하나의 결과를 나타내고, 가지의 확률을 곱하면 그 경로의 결합 확률을 얻습니다.

```
                              ┌→ 결과 AC: P(A) × P(C|A)
          ┌→ A: P(A) ────────┤
          │                  └→ 결과 AD: P(A) × P(D|A)
시작 ──────┤
          │                  ┌→ 결과 BC: P(B) × P(C|B)
          └→ B: P(B) ────────┤
                              └→ 결과 BD: P(B) × P(D|B)
```

- **모든 최종 결과의 확률 합 = 1**
- 기댓값 = 각 최종 결과 × 해당 확률의 합

### 예제: EPS 확률 나무

경제 상황(좋음/나쁨)과 회사 실적(좋음/나쁨) 2단계:

```
                         ┌→ 좋은 실적 (P=0.60): EPS=$1.80  → 결합확률 = 0.30×0.60 = 0.18
    ┌→ 좋은 경제 (0.30) ──┤
    │                    └→ 나쁜 실적 (P=0.70): EPS=$1.70  → 결합확률 = 0.30×0.70 = 0.42
────┤
    │                    ┌→ 좋은 실적 (P=0.60): EPS=$1.30  → 결합확률 = 0.40×0.60 = 0.24
    └→ 나쁜 경제 (0.40) ──┤
                          └→ 나쁜 실적 (P=0.40): EPS=$1.00  → 결합확률 = 0.40×0.40 = 0.16
```

**기대 EPS 계산:**
$$E[\text{EPS}] = 0.18(1.80) + 0.42(1.70) + 0.24(1.30) + 0.16(1.00)$$
$$= 0.324 + 0.714 + 0.312 + 0.16 = \$1.51$$

확률 합 확인: 0.18 + 0.42 + 0.24 + 0.16 = 1.00 ✓

---

### 조건부 기댓값 (Conditional Expected Value)

**조건부 기댓값**이란, 다른 사건의 결과가 알려졌을 때 계산되는 기댓값입니다.

투자에서의 활용: 새로운 정보(예: 정부의 관세 부과 결정)가 입수되면, 주식의 기대 수익률을 **조건부로 수정**합니다.

- 관세 부과 **시**: $E[R | \text{관세 있음}]$ = 더 높은 수익률 기대
- 관세 **미부과 시**: $E[R | \text{관세 없음}]$ = 낮은 수익률 기대

이처럼 새 정보가 예측을 정교하게 만드는 데 사용됩니다.

## LOS 4.c: 베이즈 공식 (Bayes' Formula)

### 왜 베이즈 공식이 필요한가?

우리는 보통 원인 → 결과 방향으로 확률을 생각합니다:
- "비가 오면(원인), 땅이 젖을(결과) 확률 = 95%"

그런데 투자에서는 **역방향**이 필요할 때가 많습니다:
- "땅이 젖어 있는데(결과 관찰), 비가 왔을(원인) 확률은?"

즉, **사후 확률(posterior probability)** — 새 증거를 본 후 원인의 확률을 업데이트합니다. 이것이 베이즈 공식입니다.

---

### 베이즈 공식 유도

곱셈 규칙에서 시작합니다:

$$P(AB) = P(B|A) \cdot P(A)$$
$$P(AB) = P(A|B) \cdot P(B)$$

두 식의 좌변이 같으므로:
$$P(B|A) \cdot P(A) = P(A|B) \cdot P(B)$$

양변을 $P(B)$로 나누면:

$$\boxed{P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}}$$

**각 항의 의미:**
| 항 | 이름 | 의미 |
|----|------|------|
| $P(A)$ | 사전 확률 (Prior) | 새 정보 이전의 A 확률 |
| $P(B\|A)$ | 우도 (Likelihood) | A가 사실일 때 B가 관찰될 확률 |
| $P(B)$ | 증거 확률 (Evidence) | B가 관찰될 전체(비조건부) 확률 |
| $P(A\|B)$ | 사후 확률 (Posterior) | B를 관찰한 후 A의 업데이트된 확률 |

**분모 $P(B)$ 계산** (전체 확률 법칙):
$$P(B) = P(B|A) \cdot P(A) + P(B|A^C) \cdot P(A^C)$$

---

### 베이즈 공식 요약 (직관적 형태)

$$\text{업데이트된 확률} = \frac{\text{해당 사건 조건에서의 새 정보 확률} \times \text{사전 확률}}{\text{새 정보의 비조건부 확률}}$$

---

### 예제: 베이즈 공식 적용

**상황:**
- 경기 호황 확률: P(호황) = 60%
  - 호황 + 주가 상승: P(상승|호황) = 70%
  - 호황 + 주가 하락: P(하락|호황) = 30%
- 경기 부진 확률: P(부진) = 40%
  - 부진 + 주가 상승: P(상승|부진) = 20%
  - 부진 + 주가 하락: P(하락|부진) = 80%

**질문:** 주가가 상승했다는 것을 관찰했을 때, 경기가 호황이었을 확률은?

**확률 나무 작성:**

```
                           ┌→ 상승: P = 0.60 × 0.70 = 0.42
    ┌→ 호황 (0.60) ────────┤
    │                      └→ 하락: P = 0.60 × 0.30 = 0.18
────┤
    │                      ┌→ 상승: P = 0.40 × 0.20 = 0.08
    └→ 부진 (0.40) ────────┤
                           └→ 하락: P = 0.40 × 0.80 = 0.32
```

**베이즈 공식 적용:**

$$P(\text{호황}|\text{상승}) = \frac{P(\text{상승}|\text{호황}) \times P(\text{호황})}{P(\text{상승})}$$

분모 계산 (상승의 비조건부 확률):
$$P(\text{상승}) = P(\text{상승}|\text{호황}) \times P(\text{호황}) + P(\text{상승}|\text{부진}) \times P(\text{부진})$$
$$= 0.42 + 0.08 = 0.50$$

최종 계산:
$$P(\text{호황}|\text{상승}) = \frac{0.42}{0.50} = 0.84 = 84\%$$

**해석:** 주가가 상승했다는 정보를 얻기 전에는 호황 확률이 60%였지만, 주가 상승을 관찰한 후에는 84%로 업데이트됩니다.

## 핵심 개념 정리 (Key Concepts)

### LOS 4.a 요약

**기댓값:**
$$E(X) = \sum P(x_i) x_i$$

**분산 (확률 모형):**
$$\text{Var}(X) = \sum P(x_i)[x_i - E(X)]^2$$

**표준편차:**
$$\sigma(X) = \sqrt{\text{Var}(X)}$$

> 표본 통계량과 달리 확률 모형은 n-1 보정 불필요 — 관측 데이터가 아닌 완전한 확률 분포로 정의되기 때문

### LOS 4.b 요약

**확률 나무:**
```
           ┌→ AC: P(A) × P(C|A)
A: P(A) ───┤
           └→ AD: P(A) × P(D|A)

           ┌→ BC: P(B) × P(C|B)
B: P(B) ───┤
           └→ BD: P(B) × P(D|B)
```

조건부 기댓값은 새 정보 도착 시 기대 수익/이익/배당을 정밀 수정하는 데 활용됩니다.

### LOS 4.c 요약

**베이즈 공식:**
$$P(A|C) = \frac{P(AC)}{P(AC) + P(BC)}$$

확률 나무를 활용:
- 주어진 결과 C가 나타난 경로들의 확률 합이 분모
- 관심 사건 A를 거쳐 C가 나타난 경로 확률이 분자

## Module Quiz 4.1

**Q1.** 다음 조건부 확률과 비조건부 확률 P(Y=1)=0.3, P(Y=2)=0.7이 주어졌을 때, X의 기댓값은?

| $x_i$ | $P(x_i \| Y=1)$ | $P(x_i \| Y=2)$ |
|--------|-----------------|-----------------|
| 0      | 0.2             | 0.1             |
| 5      | 0.4             | 0.8             |
| 10     | 0.4             | 0.1             |

A. 5.0  B. 5.3  C. 5.7

**풀이:**

1단계: Y=1일 때 E(X):
$$E(X|Y=1) = 0.2(0) + 0.4(5) + 0.4(10) = 0 + 2 + 4 = 6$$

2단계: Y=2일 때 E(X):
$$E(X|Y=2) = 0.1(0) + 0.8(5) + 0.1(10) = 0 + 4 + 1 = 5$$

3단계: 전체 기댓값 (전체 확률 법칙):
$$E(X) = P(Y=1) \cdot E(X|Y=1) + P(Y=2) \cdot E(X|Y=2)$$
$$= 0.3(6) + 0.7(5) = 1.8 + 3.5 = 5.3$$

**정답: B. 5.3**

---

**Q2.** Davies Company가 EPS > $2를 달성할 확률은 40%입니다. EPS > $2이면 신용등급 상향 확률 70%, EPS ≤ $2이면 신용등급 상향 확률 20%입니다. 신용등급이 실제로 상향된 것을 확인했을 때, EPS > $2일 확률은?

A. 50%  B. 60%  C. 70%

**풀이 (베이즈 공식):**

확률 나무:
```
                          ┌→ 상향: 0.40 × 0.70 = 0.28
    ┌→ EPS > $2 (0.40) ───┤
    │                     └→ 비상향: 0.40 × 0.30 = 0.12
────┤
    │                     ┌→ 상향: 0.60 × 0.20 = 0.12
    └→ EPS ≤ $2 (0.60) ───┤
                          └→ 비상향: 0.60 × 0.80 = 0.48
```

신용등급 상향의 비조건부 확률:
$$P(\text{상향}) = 0.28 + 0.12 = 0.40$$

베이즈 공식:
$$P(\text{EPS} > \$2 | \text{상향}) = \frac{0.28}{0.28 + 0.12} = \frac{0.28}{0.40} = 0.70 = 70\%$$

**정답: C. 70%**

## 실전 응용: 투자 분석에서의 활용

### 1. 시나리오 분석 (Scenario Analysis)
애널리스트는 확률 나무를 활용해 다음과 같이 분석합니다:
- 매크로 경제 시나리오 (호황/보통/침체)
- 각 시나리오별 기업 실적 예측
- 각 경로별 결합 확률 계산
- 기대 주가/EPS/배당 계산

### 2. 실시간 확률 업데이트
새로운 정보 도착 시 베이즈 공식으로 즉각 확률 수정:
- Fed 금리 결정 발표 → 주가 반응 확률 업데이트
- 기업 실적 발표 → 파산 확률 업데이트
- 정치적 이벤트 → 환율 방향 확률 업데이트

### 3. 주요 포인트
- 기댓값 = 평균적 결과 (장기 평균)
- 분산/표준편차 = 불확실성의 크기
- 확률 나무 = 복잡한 조건부 관계를 시각화
- 베이즈 공식 = 증거 기반 확률 업데이트
