# Reading 46: Equity Valuation — Concepts and Basic Tools (주식 가치평가)

---

## 주식의 "진짜 가치"는 얼마인가?

주식 투자의 본질은 싸게 사서 비싸게 파는 것이다. 그런데 "싸다"와 "비싸다"를 판단하려면 주식의 **내재가치**를 알아야 한다.

- 시장가격 = 마트 가격표
- 내재가치 = 그 물건의 진짜 가치

마트 가격표가 틀렸을 때(시장이 비효율적일 때) 진짜 가치를 아는 투자자가 수익을 낸다.

이 Reading은 R43(과거·현재) → R44(산업) → R45(예측) → **R46(가치평가)**로 이어지는 Equity 분석의 정점이다.

## LOS 46.a: 과대·적정·과소평가 판단

$$\text{내재가치} > \text{시장가격} \Rightarrow \text{과소평가 → 매수}$$
$$\text{내재가치} < \text{시장가격} \Rightarrow \text{과대평가 → 매도}$$

### 포지션을 취하기 전 5가지 체크리스트

1. **가격 차이의 크기** — 차이가 클수록 포지션 취할 가능성↑
2. **모델 신뢰도** — 사용한 모델이 적합한가?
3. **입력값 신뢰도** — 성장률 0.5% 차이로 결론이 뒤집히면 확신이 있어야 함
4. **시장가격의 신뢰성** — 커버 애널리스트가 많은 종목일수록 시장가격이 정확
5. **수렴 기대** — 시장가격이 내재가치로 수렴할 것이라는 확신

## LOS 46.b: 세 가지 가치평가 모델

| 모델 | 핵심 | 대표 |
|------|------|------|
| **현재가치(DCF)** | 미래 현금흐름의 현재가치 | DDM, FCFE |
| **승수(Multiplier)** | 시장 배수 비교 | P/E, P/B, EV/EBITDA |
| **자산 기반** | 순자산가치 | 장부가 조정 |

## LOS 46.c: 배당, 주식분할, 자사주 매입

### 현금배당

| 구분 | 설명 |
|------|------|
| **정기 배당** | 분기별 지급, 안정적 |
| **특별 배당** | 일회성 (호황기) |

### 주식배당과 주식분할

피자 8조각을 10조각으로 자르면 피자 크기는 그대로지만 조각당 크기가 줄어든다.

- 20% 주식배당 = 보유 주식의 20% 추가 지급
- 3-for-1 주식분할 = 1주 → 3주, 주가 1/3

**50% 주식배당과 3-for-2 주식분할은 같은 효과.** 둘 다 주식 수 50% 증가, 주가 동일 하락, 주주 부 불변.

### 자사주 매입

기업이 시장에서 자사 주식을 되사는 것. 현금 배당의 대안이다. 동기: 주가 지지, 스톡옵션 희석 방지, 배당보다 자본이득세가 낮은 국가에서 세금 우위.

## LOS 46.d: 배당 지급 연대표

```
선언일          배당락일          기준일          지급일
(Declaration)   (Ex-dividend)    (Record)        (Payment)
```

| 날짜 | 핵심 |
|------|------|
| **선언일** | 이사회가 배당 승인 |
| **배당락일** | 이날 이후 매수하면 배당 못 받음 |
| **기준일** | 주주명부 확정일 |
| **지급일** | 실제 돈이 입금되는 날 |

배당락일 **이전**에 사야 배당을 받는다. 배당락일 당일, 주가는 배당금만큼 하락한다.

예: 배당락일 전날 $25, 배당금 $1 → 배당락일 약 $24로 시작.

## LOS 46.e: DDM과 FCFE 모델

### 배당할인모델(DDM)의 논리

주식의 내재가치 = 미래 배당의 현재가치 합계

$$V_0 = \sum_{t=1}^{\infty} \frac{D_t}{(1+k_e)^t}$$

### 1년 보유 DDM

$$V_0 = \frac{D_1}{(1+k_e)} + \frac{P_1}{(1+k_e)}$$

**예시:** 작년 배당 $1, 내년 5% 성장, 연말 주가 $13.45, 요구수익률 13.2%.

$D_1 = 1.00 × 1.05 = \$1.05$

$V_0 = \frac{1.05}{1.132} + \frac{13.45}{1.132} = 0.93 + 11.88 = \$12.81$

### 복수 기간 DDM

**예시:** 최근 배당 $1.50, 성장률 8%, 요구수익률 12%, 3년 후 주가 $51.

$D_1 = 1.62,\ D_2 = 1.75,\ D_3 = 1.89$

$V_0 = \frac{1.62}{1.12} + \frac{1.75}{1.12^2} + \frac{1.89}{1.12^3} + \frac{51.00}{1.12^3} = 4.19 + 36.30 = \$40.49$

### FCFE 모델

배당을 안 하는 기업에도 적용 가능. FCFE = 주주에게 귀속 가능한 현금.

$$FCFE = CFO - FCInv + \text{순차입}$$

CFO에서 자산 투자를 빼고, 새로 빌린 돈을 더한다.

## LOS 46.g: 우선주 가치평가

우선주 = 고정 배당을 영구히 지급 → 영구연금

$$V_p = \frac{D_p}{k_p}$$

예: 액면 $100, 배당 $5, 요구수익률 8%.

$V_p = 5/0.08 = \$62.50$

---

## LOS 46.h: 고든 성장 모델 (Gordon Growth Model)

배당이 일정한 성장률로 영구히 성장할 때:

$$\boxed{V_0 = \frac{D_1}{k_e - g_c} = \frac{D_0(1+g_c)}{k_e - g_c}}$$

**조건:** $k_e > g_c$이어야 한다.

**시험 팁:**
- "just paid" / "recently paid" → $D_0$
- "will pay" / "expected to pay" → $D_1$
- "forever" / "indefinitely" → 고든 모델 사용

### 숫자로 확인

작년 배당 $1.50, 성장률 8%, 요구수익률 12%.

$D_1 = 1.50 × 1.08 = \$1.62$

$V_0 = \frac{1.62}{0.12 - 0.08} = \frac{1.62}{0.04} = \$40.50$

### 성장의 가치는 얼마인가?

성장률 0%이면: $V_0 = 1.50/0.12 = \$12.50$

성장이 기여하는 가치: $40.50 - 12.50 = \$28.00$. 현재 가치 $40.50 중 $28은 미래 성장에 대한 기대다.

### 민감도

$(k_e - g_c)$ 분모가 작아질수록 가치가 극단적으로 민감해진다. 성장률 추정값을 0.5%만 바꿔도 가치가 크게 달라지므로 다양한 조합으로 범위를 계산해야 한다.

### 배당 성장률 추정

**방법 3가지:**
1. 과거 배당 성장률
2. 업계 중앙값
3. 지속가능 성장률:

$$g = (1 - \text{배당성향}) \times ROE$$

배당을 적게 하고(이익 중 많은 부분을 재투자) ROE가 높으면 성장률이 높다.

## LOS 46.i: 다단계 DDM

현실에서 기업은 "영구히 일정 성장"하지 않는다. 초기에 빠르게 성장하다가 나중에 안정화된다.

**2단계 DDM:**
1. 초기 고성장 기간: 각 연도 배당을 개별 할인
2. 안정 성장 진입 후: 고든 모델로 **터미널 가치** 계산
3. 둘을 합산

**예시:** 올해 배당 $1.72, 처음 3년 12% 성장, 이후 영구 4% 성장, 요구수익률 11%.

$D_1 = 1.93,\ D_2 = 2.16,\ D_3 = 2.42$

터미널 가치(3년 말): $V_3 = \frac{D_4}{k_e - g} = \frac{2.42 \times 1.04}{0.11 - 0.04} = \frac{2.52}{0.07} = \$35.97$

$V_0 = \frac{1.93}{1.11} + \frac{2.16}{1.11^2} + \frac{2.42 + 35.97}{1.11^3}$

$= 1.74 + 1.75 + 28.08 = \$31.57$

## LOS 46.j: 승수 모델 (P/E)

### P/E 비율의 두 가지

| | 후행(Trailing) | 선행(Leading) |
|--|---------------|--------------|
| 공식 | 주가/지난 12개월 EPS | 주가/향후 12개월 예상 EPS |
| 특징 | 확정 데이터 기반 | 예측 기반 |

### P/E의 근본 가치와 결정 요인

고든 성장 모델에서:

$$\frac{P_0}{E_1} = \frac{D_1/E_1}{k_e - g} = \frac{\text{배당성향}}{k_e - g}$$

P/E를 높이는 요인:
- 배당성향↑ (더 많은 배당)
- 성장률↑
- 요구수익률↓ (위험↓)

### 비교 가치평가

동종 기업의 평균 P/E를 적용:

$$\text{내재가치} = \text{벤치마크 P/E} \times \text{해당 기업 EPS}$$

**주의:** 비경상 항목(일회성 이익/손실)이 EPS를 왜곡하면 P/E도 왜곡된다. 정상화된 EPS를 사용해야 한다.

### EV/EBITDA

$$EV = \text{주식 시가총액} + \text{부채 시장가치} - \text{현금}$$

EV/EBITDA는 자본 구조의 영향을 제거하므로 부채 수준이 다른 기업 간 비교에 유용하다.

---

## LOS 46.k: 자산 기반 모델

총자산(공정가치) - 부채 - 우선주 = 보통주 가치

장부가를 공정가치로 조정해야 한다. 부동산, 자원 기업처럼 자산이 가치의 핵심인 경우에 적합. 기술기업처럼 무형자산이 핵심인 경우에는 부적합.

## Key Concepts

### LOS 46.a~b
- 내재가치 > 시장가격 → 과소평가(매수)
- DCF, 승수, 자산 기반 세 가지 모델

### LOS 46.c~d
- 주식배당/분할: 총 가치 불변, 주당 가치 감소
- 배당락일 이전 매수해야 배당 수령

### LOS 46.e, g~h
- DDM: $V_0 = \sum D_t/(1+k_e)^t$
- 우선주: $V_p = D_p/k_p$
- 고든: $V_0 = D_1/(k_e - g_c)$, 조건 $k_e > g_c$
- $g = (1 - \text{배당성향}) \times ROE$

### LOS 46.i~k
- 다단계 DDM: 고성장기 개별 할인 + 안정기 고든 모델
- P/E = 배당성향/(ke-g). EV/EBITDA = 자본 구조 무관 비교
- 자산 기반: 공정가치 조정 후 순자산
