# Reading 46: Equity Valuation — Concepts and Basic Tools (주식 가치평가)


## 주식의 "진짜 가치"는 얼마인가?

마트에서 사과 한 봉지에 5,000원이라 적혀 있다. 그런데 옆 마트에서는 3,000원이다. 어느 가격이 "진짜"인가? 사과의 품질을 직접 확인해보면 알 수 있다.

주식도 마찬가지다. 삼성전자가 70,000원에 거래된다. 이 가격이 "싼" 것인지 "비싼" 것인지 판단하려면, 이 주식의 **내재가치**를 추정해야 한다.

- 시장가격 = 마트 가격표
- 내재가치 = 그 물건의 진짜 가치

마트 가격표가 틀렸을 때(시장이 비효율적일 때) 진짜 가치를 아는 투자자가 수익을 낸다.


## LOS 46.a: 과대·적정·과소평가 판단

$$\text{내재가치} > \text{시장가격} \Rightarrow \text{과소평가 → 매수}$$
$$\text{내재가치} < \text{시장가격} \Rightarrow \text{과대평가 → 매도}$$

숫자로 확인하자. 내가 분석한 A사의 내재가치가 50,000원인데 시장가격이 40,000원이면? 10,000원(20%) 과소평가 → 매수 신호다.

하지만 "50,000원이 맞다"는 확신이 얼마나 있는가? 여기서 5가지 체크리스트가 등장한다:

1. **가격 차이의 크기** — 5%보다 30% 차이가 더 확신을 준다
2. **모델 신뢰도** — 사용한 모델이 이 기업에 적합한가?
3. **입력값 신뢰도** — 성장률을 0.5% 바꿨더니 결론이 뒤집히면 확신이 약하다
4. **시장가격의 신뢰성** — 커버 애널리스트가 30명인 종목은 시장가격이 정확할 확률이 높다
5. **수렴 기대** — 시장가격이 내재가치로 수렴할 것이라는 확신


## LOS 46.b: 세 가지 가치평가 모델

| 모델 | 핵심 | 대표 |
|------|------|------|
| **현재가치(DCF)** | 미래 현금흐름의 현재가치 | DDM, FCFE |
| **승수(Multiplier)** | 시장 배수 비교 | P/E, P/B, EV/EBITDA |
| **자산 기반** | 순자산가치 | 장부가 조정 |

아파트 가치를 평가할 때: DCF = "앞으로 받을 월세의 현재가치", 승수 = "옆 동 아파트가 10억이니 이것도 비슷하겠다", 자산 기반 = "토지값 + 건물값 - 담보대출".


## LOS 46.c: 배당, 주식분할, 자사주 매입

### 현금배당

| 구분 | 설명 |
|------|------|
| **정기 배당** | 분기별 지급, 안정적 |
| **특별 배당** | 일회성 (호황기에 "보너스") |

### 주식배당과 주식분할

피자 8조각을 10조각으로 자르면 피자 크기는 그대로지만 조각당 크기가 줄어든다.

- 20% 주식배당 = 100주 보유자에게 20주 추가 지급. 총 120주, 주가는 100/120 = 83.3%로 조정
- 3-for-1 주식분할 = 1주 → 3주, 주가 1/3

**50% 주식배당과 3-for-2 주식분할은 같은 효과.** 둘 다 주식 수 50% 증가, 주가 동일 하락, 주주 부 불변.

### 자사주 매입

기업이 시장에서 자사 주식을 되사는 것. 왜? 배당 대신 주주에게 현금을 돌려주되, 주당 순이익(EPS)을 높이는 효과도 있다. 발행주식수가 줄어들므로:

기존: 순이익 100억 ÷ 1억주 = EPS 100원
매입 후: 순이익 100억 ÷ 9,000만주 = EPS 111원


## LOS 46.d: 배당 지급 연대표

```
선언일          배당락일          기준일          지급일
(Declaration)   (Ex-dividend)    (Record)        (Payment)
```

| 날짜 | 핵심 |
|------|------|
| **선언일** | 이사회가 배당 승인 |
| **배당락일** | 이날 이후 매수하면 배당 못 받음 |
| **기준일** | 주주명부 확정일 |
| **지급일** | 실제 돈이 입금되는 날 |

배당락일 **이전**에 사야 배당을 받는다. 배당락일 당일, 주가는 배당금만큼 하락한다.

숫자로 확인: 배당락일 전날 25, 배당금 1 → 배당락일 약 24로 시작. 왜? 1은 이미 기존 주주에게 귀속되었으니 새 매수자는 1 빠진 가격에 사는 것이 공정하다.


## LOS 46.e: DDM과 FCFE 모델

### 배당할인모델(DDM)의 논리

주식의 가치는 결국 **내가 받을 현금**의 현재가치다. 배당이 주주가 받는 유일한 현금이라면:

$$V_0 = \sum_{t=1}^{\infty} \frac{D_t}{(1+k_e)^t}$$

### 1년 보유 DDM — 가장 단순한 경우

올해 주식을 사서, 배당 한 번 받고, 연말에 판다.

$$V_0 = \frac{D_1}{(1+k_e)} + \frac{P_1}{(1+k_e)}$$

숫자를 넣자. 작년 배당 1, 내년 5% 성장, 연말 주가 13.45, 요구수익률 13.2%.

$D_1 = 1.00 × 1.05 = 1.05$

$V_0 = \frac{1.05}{1.132} + \frac{13.45}{1.132} = 0.93 + 11.88 = 12.81$

시장가격이 11이면? 내재가치 12.81 > 시장가격 11 → 과소평가 → 매수.

### 복수 기간 DDM

3년 보유하고 판다면, 3년간 배당 + 3년 후 매도가를 모두 현재가치로 할인한다.

최근 배당 1.50, 성장률 8%, 요구수익률 12%, 3년 후 주가 51.

$D_1 = 1.62,\ D_2 = 1.75,\ D_3 = 1.89$

$V_0 = \frac{1.62}{1.12} + \frac{1.75}{1.12^2} + \frac{1.89}{1.12^3} + \frac{51.00}{1.12^3} = 4.19 + 36.30 = 40.49$

### FCFE 모델

아마존처럼 배당을 안 하는 기업은 DDM을 쓸 수 없다. 대신 FCFE(주주에게 귀속 가능한 현금)를 사용한다.

$$FCFE = CFO - FCInv + \text{순차입}$$

영업현금흐름에서 설비 투자를 빼고, 새로 빌린 돈을 더한다. "배당을 **줄 수 있는** 최대 금액"이다.


## LOS 46.f: 가치평가 모델별 장단점

세 가지 모델 중 어떤 것을 쓸지 선택하려면 각각의 강점과 약점을 알아야 한다.

### 1. 할인현금흐름(DCF) 모델

| 장점 | 단점 |
|------|------|
| 현재가치 이론에 기반 — 재무학 이론적으로 가장 건전 | 입력값을 추정해야 함 |
| 애널리스트 커뮤니티에서 폭넓게 수용 | 입력값에 매우 민감 |
| FCFE 모델: 배당 미지급 기업에도 사용 가능 | 고든 모델: k-g 분모에 극도로 민감 |
| 고든 모델: 안정적·성숙·비순환 기업에 유용 | 고든 모델: k > g 필수, k와 g가 상수여야 함 |
| 다단계 모델: 비정률 성장 기업에 사용 가능 | 고든 모델: 배당을 지급해야 함 |

### 2. 주가 승수(Price Multiples) 모델

| 장점 | 단점 |
|------|------|
| 주식 수익률 예측에 유용하다는 증거 | 펀더멘탈 P/E는 입력값에 민감 |
| 애널리스트에게 널리 사용되고 쉽게 계산 | 기업 간 비교 불가능할 수 있음 (특히 국제적) |
| 시계열·횡단면 비교 가능 | 경기순환 기업의 배수는 경기 상황에 왜곡 |
| EV/EBITDA: 자본 구조 무관 비교, 음의 이익 시 유용 | 비교법과 펀더멘탈법의 결론이 상반될 수 있음 |
| | 음의 분모 → 무의미한 비율 (특히 P/E) |

### 3. 자산 기반(Asset-Based) 모델

| 장점 | 단점 |
|------|------|
| 최소 가치(floor value) 제공 가능 | 시장가치 확보 어려움, 장부가와 대부분 다름 |
| 유형 단기자산, 시장가 확인 가능 자산, 청산 시 가장 신뢰 | 무형자산 비중 높은 기업에 부정확 |
| 공정가치 보고 기업 증가 → 유용성 확대 | 초인플레이션 기간에 자산 가치 추정 곤란 |


## LOS 46.g: 우선주 가치평가

우선주 = 고정 배당을 영구히 지급. 만기가 없다. 이것은 **영구연금**이다.

$$V_p = \frac{D_p}{k_p}$$

숫자를 넣자. 액면 100, 연간 배당 5, 투자자 요구수익률 8%.

$V_p = 5/0.08 = 62.50$

왜 액면(100)보다 훨씬 낮은가? 시장이 8%를 요구하는데 배당률은 5%뿐이다. 이 부족분을 가격 할인으로 보상해야 한다.

요구수익률을 바꿔보자: 5%라면 $V_p = 5/0.05 = 100$(액면가). 3%라면 $V_p = 5/0.03 = 166.67$. 요구수익률이 낮아질수록 가치가 급등한다.


## LOS 46.h: 고든 성장 모델 (Gordon Growth Model)

배당이 매년 일정 비율로 영구히 성장하면, 무한급수를 깔끔한 공식으로 정리할 수 있다:

$$\boxed{V_0 = \frac{D_1}{k_e - g_c} = \frac{D_0(1+g_c)}{k_e - g_c}}$$

**조건:** $k_e > g_c$이어야 한다. 성장률이 요구수익률보다 높으면 분모가 음수 → 가치가 무한대 → 말이 안 된다.

**시험 팁:**
- "just paid" / "recently paid" → $D_0$ (과거 배당, 1+g를 곱해야)
- "will pay" / "expected to pay" → $D_1$ (바로 사용)
- "forever" / "indefinitely" → 고든 모델

### 숫자를 직접 넣어보자

작년 배당 1.50, 성장률 8%, 요구수익률 12%.

$D_1 = 1.50 × 1.08 = 1.62$

$V_0 = \frac{1.62}{0.12 - 0.08} = \frac{1.62}{0.04} = 40.50$

### 성장의 가치는 얼마인가?

성장률이 0%이면 어떻게 될까?

$V_0 = 1.50/0.12 = 12.50$

성장이 기여하는 가치: 40.50 - 12.50 = 28.00$. 현재 가치 40.50 중 28(69%)은 순전히 미래 성장에 대한 기대다. 성장이 실현되지 않으면 가치가 70% 가까이 증발한다.

### 변수를 바꿔보자 — 분모의 위력

$(k_e - g_c)$ 분모가 작아질수록 가치가 극단적으로 민감해진다.

| $k_e$ | $g_c$ | $k_e - g_c$ | $V_0$ |
|--------|--------|-------------|-------|
| 12% | 8% | 4% | 40.50 |
| 12% | 9% | 3% | 54.00 |
| 12% | 10% | 2% | 81.00 |
| 12% | 11% | 1% | 162.00 |

성장률을 8%에서 11%로 3%p만 올렸을 뿐인데 가치가 4배가 된다! 그래서 다양한 조합으로 범위를 계산하는 민감도 분석이 필수다.

### 배당 성장률 추정 — 3가지 방법

1. 과거 배당 성장률
2. 업계 중앙값
3. **지속가능 성장률:**

$$g = (1 - \text{배당성향}) \times ROE$$

배당을 적게 하고(이익 중 많은 부분을 재투자) ROE가 높으면 성장률이 높다. 배당성향 40%, ROE 15%이면: g = 0.60 × 15% = **9%**.


## LOS 46.i: 다단계 DDM

현실에서 기업은 "영구히 일정 성장"하지 않는다. 스타트업은 처음에 50% 성장하다가, 경쟁이 심해지면 5%로 안정화된다.

**2단계 DDM:**
1. 초기 고성장 기간: 각 연도 배당을 개별 할인
2. 안정 성장 진입 후: 고든 모델로 **터미널 가치** 계산
3. 둘을 합산

숫자를 넣자. 올해 배당 1.72, 처음 3년 12% 성장, 이후 영구 4% 성장, 요구수익률 11%.

$D_1 = 1.93,\ D_2 = 2.16,\ D_3 = 2.42$

터미널 가치(3년 말): $V_3 = \frac{D_4}{k_e - g} = \frac{2.42 \times 1.04}{0.11 - 0.04} = \frac{2.52}{0.07} = 35.97$

$V_0 = \frac{1.93}{1.11} + \frac{2.16}{1.11^2} + \frac{2.42 + 35.97}{1.11^3}$

$= 1.74 + 1.75 + 28.08 = 31.57$

전체 가치 31.57 중 터미널 가치의 현재가치가 28.08 → 약 89%! 대부분의 가치가 3년 이후에서 온다. 그래서 장기 성장률(4%) 추정이 결과를 좌우한다.


## LOS 46.j: 승수 모델 (P/E)

### P/E 비율 — 한 주당 이익의 몇 배를 지불하는가?

50짜리 주식의 EPS가 5이면 P/E = 10. "이 기업 이익의 10배를 지불한다"는 뜻이다.

| | 후행(Trailing) | 선행(Leading) |
|--|---------------|--------------|
| 공식 | 주가/지난 12개월 EPS | 주가/향후 12개월 예상 EPS |
| 특징 | 확정 데이터 | 예측 기반 |

### P/E의 결정 요인 — 고든 모델에서 유도

$$\frac{P_0}{E_1} = \frac{D_1/E_1}{k_e - g} = \frac{\text{배당성향}}{k_e - g}$$

P/E를 높이는 요인:
- 배당성향↑ (더 많은 배당 → 주주에게 후한 기업)
- 성장률↑ (더 밝은 미래)
- 요구수익률↓ (위험이 낮은 기업)

### 비교 가치평가

같은 업종 평균 P/E가 15인데 A사 EPS가 3이면:

$$\text{내재가치} = 15 \times 3 = 45$$

시장가격이 35라면 과소평가. **주의:** 일회성 이익이 EPS를 부풀렸다면 P/E가 왜곡된다. 정상화된 EPS를 써야 한다.

### EV/EBITDA — 자본 구조를 무시하는 비교

$$EV = \text{주식 시가총액} + \text{부채 시장가치} - \text{현금}$$

A사와 B사가 같은 사업인데 A는 무차입, B는 부채가 많다. P/E로 비교하면 이자비용 차이 때문에 왜곡된다. EV/EBITDA는 이자 전 이익을 사용하므로 자본 구조의 영향을 제거한다.


## LOS 46.k, 46.m: 자산 기반 모델

$$\text{자기자본 가치} = \text{자산의 시장(공정)가치} - \text{부채의 시장(공정)가치}$$

장부가는 거의 항상 시장가와 다르다. 가능한 접근법: 감가상각 후 가치, 인플레이션 조정 감가상각 가치, 추정 대체원가 등.

### 숫자로 확인하자

Williams Optical의 재무제표:
- 현금 10,000, 매출채권 20,000, 재고 50,000, 순고정자산 120,000 (장부가)
- 매입채무 5,000, 단기차입금 30,000, 장기차입금 45,000
- 발행주식수 2,000주, 고정자산 시장가 = 장부가의 120%

자산 시장가: 10,000 + 20,000 + 50,000 + 120,000 × 1.20 = **224,000**
부채 시장가: 5,000 + 30,000 + 45,000 = **80,000**
조정 자기자본: 224,000 - 80,000 = **144,000**
주당 가치: 144,000 / 2,000 = **72**

### 적합한 경우와 부적합한 경우

| 적합 | 부적합 |
|------|--------|
| 유형 단기자산 위주 기업 | 무형자산 비중 높은 기업 (구글, 소프트웨어) |
| 시장가치 확인 쉬운 자산 (금융, 자원 기업) | 초인플레이션 국가 — 자산 가치 추정 곤란 |
| 청산 중인 기업 | 현재 경영진의 역량·고객 관계 등 미반영 |

무형자산이 중요한 기업에서는 자산 기반 가치를 **최소 가치(floor value)**로 보고, DCF 등 미래지향적 모델로 보완해야 한다.


## 체감 설명 보강: 주식 가치평가의 핵심 직관들

### 왜 "터미널 가치"가 전체 가치의 80–90%를 차지하는가?

**일상 상황:** 나무를 심었다. 처음 5년간 열매가 10개, 15개, 20개, 25개, 30개 열린다. 이후 매년 30개씩 영구히 열린다. 처음 5년 열매 합계 = 100개. 이후 영구 열매 = 연 30개 × ∞. 당연히 영구 부분이 압도적이다.

**숫자로 직접 확인:**

기업 Z: 처음 5년 배당 2, 2.2, 2.4, 2.6, 2.8. 이후 3% 영구 성장. 요구수익률 10%.

- 5년간 배당의 PV = 2/1.10 + 2.2/1.21 + ... ≈ **8.8**
- 터미널 가치: $V_5 = \frac{2.8 \times 1.03}{0.10 - 0.03} = 41.20$. PV = 41.20/1.10^5 = **25.58**
- **총 가치 = 8.8 + 25.58 = 34.38**

터미널 가치 비중 = 25.58 / 34.38 = **74%**. 예측 가능한 5년의 배당은 전체의 26%에 불과하다.

**변수를 바꾸면?** 영구 성장률을 3% → 5%로 올리면?
- $V_5 = 2.8 \times 1.05 / 0.05 = 58.80$. PV = 58.80/1.10^5 = **36.52**
- 총 가치 = 8.8 + 36.52 = **45.32**
- 터미널 가치 비중 = **81%**

성장률 2%p 차이로 총 가치가 **32% 증가.** 영구 성장률 추정이 가치평가에서 가장 민감한 변수다. 이것이 "장기 성장률 가정에 매우 신중해야 한다"는 교훈이다.


### 왜 "P/E 비율"을 그냥 비교하면 위험한가?

**일상 상황:** "이 아파트는 월세 대비 가격이 300배(P/R = 300)야. 저 아파트는 200배야. 저게 더 싸네!" — 이렇게 단순 비교하면 안 된다. 강남 아파트는 지가 상승 기대가 있고, 외곽 아파트는 그렇지 않다. **성장 기대가 다르면 배수도 달라야 한다.**

**숫자로 직접 확인:**

| 기업 | EPS | 주가 | P/E | 성장률 | PEG |
|------|-----|------|-----|-------|-----|
| A (성숙 식품) | 5 | 75 | 15 | 3% | 5.0 |
| B (성장 IT) | 3 | 90 | 30 | 25% | 1.2 |

P/E만 보면 A(15배)가 B(30배)보다 "싸다." 하지만 PEG(P/E ÷ 성장률)를 보면 B(1.2)가 A(5.0)보다 훨씬 합리적이다. **성장률 대비 얼마를 지불하는가**가 진짜 비교 기준이다.

**변수를 바꾸면?** B의 성장률이 25% → 10%로 하락하면? PEG = 30/10 = 3.0. 갑자기 비싸진다. 성장주의 가격은 **성장률 기대에 극도로 민감하다.** 2022년 나스닥 하락이 바로 이 메커니즘 — 금리 인상 → 성장률 기대 하락 → P/E 수축 → 주가 폭락.


### 왜 "FCFE"가 배당보다 더 정확한 가치 척도인가?

**일상 상황:** 부모님이 월급 500만원을 버는데, 용돈을 10만원만 준다. 부모님의 가치가 용돈 10만원의 현재가치인가? 아니다. **줄 수 있는 능력**(500만원 - 고정비)이 진짜 가치다.

기업도 마찬가지다. 아마존은 배당을 **0** 준다. DDM으로 계산하면 가치 = 0. 하지만 아마존의 시가총액은 **2조 달러**다. 왜? 배당은 안 하지만 FCFE(주주에게 줄 수 있는 현금)는 연 수백억 달러이기 때문이다.

**숫자로 비교:**

| | 기업 C (배당 중심) | 기업 D (아마존형) |
|--|-----------------|----------------|
| 순이익 | 100억 | 100억 |
| 배당 | 60억 | 0 |
| 자사주 매입 | 0 | 40억 |
| 재투자 | 40억 | 60억 |
| FCFE | 60억 | 40억 |

DDM 가치(C) = 60억/0.08 = **750억**
DDM 가치(D) = 0/0.08 = **0** ← 말도 안 됨!
FCFE 가치(D) = 40억/0.08 = **500억** ← 합리적

**변수를 바꾸면?** 기업 D가 갑자기 배당을 시작하면(FCFE 40억 전부 배당)? DDM 가치 = 40억/0.08 = 500억. FCFE 가치와 동일해진다. **배당 정책이 바뀌어도 기업의 본질적 현금 창출 능력은 같다.** FCFE가 배당보다 근본적인 이유다.


### 왜 "자산 기반 모델"이 무형자산 시대에 한계가 있는가?

**일상 상황:** 유튜버의 가치를 평가한다. 카메라 장비 200만원 + 컴퓨터 300만원 = 자산 500만원. 하지만 이 유튜버의 연수익은 3억원이다. 자산 500만원이 이 사람의 진짜 가치인가? 아니다. **구독자, 콘텐츠, 브랜드**라는 무형자산이 99%의 가치를 만든다.

**숫자로 체감 — S&P 500의 유·무형자산 비중 변화:**

| 연도 | 유형자산 비중 | 무형자산 비중 |
|------|-----------|-----------|
| 1975 | 83% | 17% |
| 2000 | 32% | 68% |
| 2024 | **10%** | **90%** |

2024년 기준 S&P 500 기업 가치의 **90%가 무형자산**(브랜드, 특허, 네트워크 효과, 데이터)이다. 자산 기반 모델로 이 기업들을 평가하면 가치의 90%를 놓친다.

**변수를 바꾸면?** 청산(파산) 중인 제조업체라면? 공장, 토지, 장비의 시장가가 존재하므로 자산 기반 모델이 가장 적합하다. 금융업(은행)도 자산(대출 포트폴리오)의 공정가치가 명확하므로 자산 기반 모델이 유용하다. **산업 특성에 따라 최적 모델이 달라진다.**

**Q5.** 정률 성장 모델(constant growth model)의 필수 조건은?

**Q7.** 다음 해 배당 2, 연말 주가 40. 요구수익률 15%. 주식 가치는?

A. 33.54.  B. 36.52.  C. 43.95.

**풀이:** V₀ = (2 + 40)/1.15 = 42/1.15 = 36.52.
**정답: B**


**Q8.** 배당 성장률 5%, 최근 배당 1, 요구수익률 10%. 내재 가치는?

A. 20.00.  B. 21.00.  C. 22.05.

**풀이:** V₀ = D₁/(k − g) = 1 × 1.05/(0.10 − 0.05) = 1.05/0.05 = 21.00.
**정답: B**


**Q9.** 1년말 배당 1.25, 2년말 배당 1.56. 이후 5% 정률 성장. kₑ = 11%. 주식 가치는?

A. 22.30.  B. 23.42.  C. 24.55.

**풀이:** D₃ = 1.56 × 1.05 = 1.638. P₂ = D₃/(kₑ − g) = 1.638/0.06 = 27.30. V₀ = 1.25/1.11 + (1.56 + 27.30)/(1.11)² = 1.126 + 28.86/1.2321 = 1.126 + 23.42 = 24.55.
**정답: C**


**Q10.** Brown사: 2년간 배당 성장률 25%, 이후 6% 정률 성장. 할인율 10%, 최근 배당 1. 다단계 DDM 가치는?

A. 31.25.  B. 33.54.  C. 36.65.

## Key Concepts

### LOS 46.a~b
- 내재가치 > 시장가격 → 과소평가(매수)
- DCF, 승수, 자산 기반 세 가지 모델

### LOS 46.c~d
- 주식배당/분할: 총 가치 불변, 주당 가치 감소
- 배당락일 이전 매수해야 배당 수령

### LOS 46.e, g~h
- DDM: $V_0 = \sum D_t/(1+k_e)^t$
- 우선주: $V_p = D_p/k_p$
- 고든: $V_0 = D_1/(k_e - g_c)$, 조건 $k_e > g_c$
- $g = (1 - \text{배당성향}) \times ROE$

### LOS 46.f
- DCF: 이론적으로 건전, 입력값 민감. FCFE는 배당 미지급 기업에 유용
- 승수: 계산 쉽고 널리 사용, 기업 간 비교 어려울 수 있음
- 자산 기반: 최소 가치 제공, 무형자산 비중 높으면 부정확

### LOS 46.i~k
- 다단계 DDM: 고성장기 개별 할인 + 안정기 고든 모델
- P/E = 배당성향/(ke-g). EV/EBITDA = 자본 구조 무관 비교

### LOS 46.m
- 자산 기반 모델: 자산 공정가치 − 부채 공정가치 = 자기자본 가치
- 유형 자산·시장가 확인 가능 자산·청산 시 가장 신뢰
- 무형자산 비중 높은 기업에는 부적합, floor value로 활용
